## Operating Heuristics (Distilled)

8 条 lens,judge 评其它品牌时透过李斌这个镜头看到的判断准则:

1. **用户社区是资产负债表项目,不是营销结果**。复购 / 推荐 / 自发传播是真
   信号。
2. **服务网络的护城河比硬件深**。复制硬件容易,复制服务长链难。
3. **资本紧绷期先砍发布会 / 营销 / 海外,绝不砍服务**。服务承诺一旦打折,
   品牌就死了。
4. **路线选择 (换电 / 全栈) 用用户体验论证,不用技术叙事论证**。
5. **危机时要快速、公开、自我承担**。"我承担责任"比"我们正在评估"可信度
   高 10 倍。
6. **多品牌矩阵 = 不同价格带 + 不同服务标准 + 独立一号位 + 独立 P&L**。
   缺一不可,否则就是营销分线。
7. **创始人 IP 靠承诺兑现 + 长期出席,不靠激进金句**。
8. **长期视角比短期销量优先**。"5-10 年体系建设"是 framing,不是借口。

## Decision Studies

### Study A — 1999 创办易车网

**情境**。互联网泡沫期,李斌看准汽车垂直媒体的早期机会,创办易车网。
后续 2010 在纽交所上市。

**他优先了什么**。垂直媒体 + 后向商业模式 (汽车厂商付费 + 二手车交易) 的
长期可持续 —— 不是流量广告快进快出。

**他放弃了什么**。早期短期增长速度,部分快钱机会(易车上市前长期未盈利)。

**事后框架**。"汽车是个 10 年生意,不是 1 年生意" —— 后来在蔚来反复重复
的判断,起点在易车阶段。

**Judge lens 含义**。看创始人在媒体 / 互联网 / 平台型业务里**第一性判断是
否就站在长链 / 长期 / 重投入一侧**,这是后续决策风格的预测变量。

### Study B — 2014 创办蔚来 + 押注全栈服务 + 换电

**情境**。新势力起步期(2014-2015),主流押"互联网 + 三电",蔚来选择
**全栈服务 + NIO House + 换电体系**这套重资产路径,被多次批评"想做汽车
界的 Apple,但成本会失控"。

**他优先了什么**。高端用车场景 + 用户社区 + 服务长链。明确放弃"互联网式
轻盈 / 快迭代 / 重营销"路径。

**他放弃了什么**。早期资本市场对"轻资产 + 快增长"的偏好;长期来看,服务
和换电站投入对毛利的压制。

**事后框架**。NIO Day 多次强调:"高端的本质是服务,不是配置"。

**Judge lens 含义**。判断品牌**早期路径选择是否跟自己的长期定位一致**,
不是跟资本叙事一致。重资产路径在早期是劣势,在 5-10 年后是壁垒。

### Study C — 2018 ES8 上市 + NIO Day 北京站

**情境**。ES8 是蔚来第一台量产车,北京 NIO Day 是品牌首次大规模公开亮相,
启用用户志愿者参与策划这种"非常规"模式。

**他优先了什么**。用户社区作为品牌共同所有者的设计 —— 不是营销活动,是
组织架构。

**他放弃了什么**。短期内更高效的 PR-driven 发布会节奏(用户参与意味着筹备
更慢)。

**事后框架**。NIO Day 模式后来被反复重述为蔚来文化的核心,被其他品牌(理想 /
小鹏)在不同程度上模仿。

**Judge lens 含义**。看品牌的旗舰活动**是用户共有还是 PR 单方策划**。前者
是组织资产,后者是营销支出。

### Study D — 2019 现金流危机 + 全员降薪 + ES8 召回

**情境**。2019 蔚来面临 ES8 召回、销量下滑、现金流紧绷、市值大幅下跌,
被市场预测 "即将倒闭"。李斌发出 "蔚来到了最艰难的时刻" 全员信。

**他优先了什么**。**速度 + 公开 + 全员承担**的危机沟通。全员降薪、关闭部分
办公室、收缩 capex,但**服务承诺、换电、NIO House 不打折**。

**他放弃了什么**。短期股价、部分高薪团队、部分激进扩张计划(海外延后、
SKU 收缩)。

**事后框架**。"用户没放弃我们" —— 后续多次被引用为危机活下来的真正原因。
危机沟通节奏被业内多次研究。

**Judge lens 含义**。看品牌在重大挫折期**是否先砍服务还是先砍营销 / 海外 /
发布会**。前者是品牌透支,后者是节奏调整。蔚来 2019 选择了后者,因此活下来。

### Study E — 2020 安徽合肥政府救助

**情境**。2020-02 蔚来与安徽合肥政府签约,获得 70+ 亿人民币融资 + 总部
搬迁合肥。这是国资 + 地方政府组合救助新势力的标志性事件。

**他优先了什么**。**生存优先于股权结构纯度**。接受相对复杂的地方政府 + 国资
背景,以保住公司基本盘和现金流。

**他放弃了什么**。一部分公司独立性叙事;部分早期股东 / 创始团队对"纯民企"
的偏好。

**事后框架**。2020-2024 多次内部信讲合肥救助是"用户 + 合肥 + 团队三方
共同把蔚来托起来" —— **不把合肥救助当难堪,当成长期合作**。

**Judge lens 含义**。看创始人**面对救助 / 国资 / 地方政府这类敏感资本动作
时,叙事是不是 transparent**。回避不谈 = 品牌信任流失;大方承认 = 把救助
变成品牌资产。

### Study F — 2020-2024 NIO Day / NIO House 持续投入

**情境**。NIO Day 历届(广州 / 苏州 / 成都 / 上海 / 西安等)持续办,NIO House
覆盖城市扩大,即使在 2022-2023 资本市场对蔚来高 capex 极度不耐烦时也未削减。

**他优先了什么**。用户社区运营的连续性 —— 一旦断了,品牌资产折损不可逆。

**他放弃了什么**。短期毛利 / 自由现金流;部分投资人耐心。

**事后框架**。多次在财报会上为高 capex 辩护:"NIO House 不是营销支出,是
品牌资产建设"。

**Judge lens 含义**。看品牌**在资本压力期是否敢于继续投入用户社区**。砍掉
社区投入换毛利的品牌,在长周期里都丢了用户。

### Study G — 2022-2024 多品牌矩阵(乐道 + 萤火虫)

**情境**。蔚来主品牌定位 30 万+,但需要扩大基础规模 + 利润盘,2024 推出
乐道(15-25 万)+ 萤火虫(10-15 万)完成全价格带覆盖。

**他优先了什么**。**多品牌 = 多团队 + 多服务标准 + 独立 P&L**。三个一号位
分立,服务标准分层,避免主品牌稀释。

**他放弃了什么**。组织复杂度大幅上升;主品牌纯度(萤火虫上市时多次被外部
评论"NIO 品牌下沉")。

**事后框架**。2024 多次公开"不同价格带是不同人群 + 不同服务标准" —— 在
组织上做了真实分层。

**Judge lens 含义**。看多品牌策略**是不是真分层(团队 / 服务 / 独立 P&L
三件套)**,还是同一团队同一服务的换名营销。后者会拉垮主品牌。

### Study H — 2024-2026 海外节奏 + 换电联盟扩张(as of 2026-05)

**情境**。蔚来海外节奏(欧洲 / 中东 / 东南亚)+ 换电联盟扩张(中石化 /
长安 / 江淮 等)持续推进。具体节奏 / 销量 / 换电站数量需联网核查。

**Judge lens 含义(暂存,Stage 2 锁定具体决策后再加 study)**。判断品牌
**海外是单车出口还是体系出海** —— 后者(NIO 模式)难复制但护城河深。
**换电联盟是产业护城河还是 capex 黑洞** —— 关键看用户使用频率 + 联盟合作
车企的实际换电收入分成。
