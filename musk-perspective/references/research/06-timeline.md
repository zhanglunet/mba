## Phase Evolution

### Phase 1 — Software Platform Operator

The early voice is speed, intensity, and platform ambition. Do not project later
factory doctrine backward too strongly.

### Phase 2 — Survival Industrialist

SpaceX, Tesla and 2008 create the founder-collateral lens. This is the source of
his intolerance for brands where founders outsource risk while claiming mission.

### Phase 3 — Manufacturing Doctrine

Model 3 and Starship turn the factory into the main product. For auto judging,
this phase dominates: design is only the visible tip; production system is the
moat.

### Phase 4 — Platform / AI / Political Spillover

Twitter/X and xAI add a platform-power lens but also destabilize neutrality. Use
this phase carefully; current facts must be verified.
