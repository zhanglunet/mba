# Dimension 3 — Distribution & Channels

**Brand under audit:** 智坊 atelier (前身 ai 坊；过渡机制 harness farm)
**CBC = 宽带资本 China Broadband Capital，田溯宁 2006 创办**
**As-of date:** 2026-05-09
**Researcher:** dimension_3 sub-agent

---

## 0. Pre-conclusion (per task brief)

三个候选品牌名（ai 坊 / 智坊 atelier / harness farm）的公开 footprint 已被 dimension_1、dimension_2 确认接近零。本路调研**不再重复搜这三个名字**，而是建立一个 channel baseline：CBC 真实在用的曝光路径 + 同类对手孵化器的标准 channel mix → 推断如果智坊 atelier 真的发布会走哪条路。

### 1.2 田溯宁个人 IP 借势（Founder-led earned media）

田溯宁是 CBC 最强的曝光资产，他的演讲场合就是 CBC 投后的事实分发管道：

- **亚布力中国企业家论坛**：田溯宁兼任理事，CBC 是参会企业。2025 年第 25 届年会与 2024 年第 24 届年会都举办，但**搜索没有返回田溯宁 2024-2025 在亚布力的具体演讲文字记录** [亚布力百科](https://baike.baidu.com/item/%E4%BA%9A%E5%B8%83%E5%8A%9B%E4%B8%AD%E5%9B%BD%E4%BC%81%E4%B8%9A%E5%AE%B6%E8%AE%BA%E5%9D%9B/20814741)
- **世界互联网大会乌镇峰会**：2014 年首届田溯宁演讲，提出"4 大力量汇聚 = 产业互联网"。2024、2025 年峰会都举办，但**未找到田溯宁 2024/2025 的演讲记录** [CAC 报道 2014](https://www.cac.gov.cn/2014-11/19/c_1114046365.htm)
- **GSMA 2025 Post-MWC 思享汇**（2025-04-10）：田溯宁以亚信科技董事长身份出席，论 AI 时代运营商转型 [亚信科技报道](https://www.asiainfo.com/zh_hk/content_4834.html)
- **北大国发院**：这是 CBC 最重要的"earned + owned 共建"通道。**侯宏 = 北大国发院助理教授 + 承泽企业家项目学术主任**；田溯宁与侯宏 2025-09-28 联合署名 op-ed《智能体互联网——被"人工智能+"行动忽视的关键未来》在新浪财经发表，11-13 再次刊发 [新浪财经 9-28 版](https://finance.sina.com.cn/cj/2025-09-28/doc-infrzcmt9028783.shtml), [11-13 版](https://finance.sina.com.cn/cj/2025-11-13/doc-infxhkcu5220933.shtml), [侯宏 PKU 主页](https://nsd.pku.edu.cn/szdw/qzjs/h/516313.htm)
- **小宇宙播客**：侯宏单独做客一期专门解读"智能体互联网" [Vol.38 对话侯宏博士](https://www.xiaoyuzhoufm.com/episode/688acaf88e06fe8de78c4177)
- 民生银行独董、中国信息化百人会执行理事、中国企业家俱乐部理事——都是闭门高管型场合，不是 to-developer 通道

### 1.3 第三方媒体（Earned media）

- **新浪财经**：op-ed 主要落地媒体（智库精选栏目）

### 1.4 历史孵化案例的曝光路径（reference pattern）

- **2010 北京云基地**、**雪浪云**、**亚信 AI 路线**：搜索"宽带资本 + 发布会"无独立 PR 流；这些项目的曝光主要靠**被投公司自己的渠道 + 田溯宁演讲带流**，CBC 母品牌不做产品发布会

## 3. If 智坊 atelier launched today, channel mix would likely be... *(inference)*

基于 §1 + §2 的模式匹配：

1. **首发主场地**：北大国发院承泽企业家项目内部活动 — *(high-confidence inference)*：侯宏的双重身份（学者 + op-ed 共同作者）已经把 CBC 的 AI 叙事钉在国发院；任何"坊 / atelier"形态的孵化器最自然的首秀是承泽企业家圆桌会或国发院 EMBA 论坛。
2. **概念背书路径**：田溯宁 + 侯宏联合署名长文，主投新浪财经"智库精选"栏目 → 关注者通过小宇宙播客二次传播。这正是 2025 年"智能体互联网"叙事走的路 — *(verified pattern)*。
3. **行业大会借势**：2026 下半年的世界互联网大会（乌镇）+ 中关村论坛 + 亚布力夏季峰会 — 田溯宁会以"智能体互联网 2.0 / 智能体孵化平台"题目登场 — *(inference based on his 2014/2015 Wuzhen + Yabuli 历史模式)*。
4. **媒体合作**：财新 / 财经杂志（顶层 op-ed）+ 36 氪 PitchHub（机构数据收录）+ 雷峰网 / DOIT（科技垂媒）— *(inference based on §1.3 历史轨迹)*。
5. **不会出现**的渠道：B 站直播 Demo Day、抖音视频号、Twitter/X、Discord 社群、developer hackathon — CBC 的 DNA 是"运营商级 + 投行级"叙事，不是开发者社区运营，与奇绩 / 真格的下沉打法不同。
6. **harness farm 这层不会单独 PR**：作为"过渡机制 / 内部代号"，按 CBC 历史习惯（云基地、雪浪云）这类工具品牌通常不做独立发布，只在主品牌叙事里附带提一句。

## 4. Paid / Owned / Earned 占比推断 *(inference)*

| 类别 | 占比推断 | 依据 |
|---|---|---|
| **Paid（付费投放）** | **<5%** | 没有任何搜索证据显示 CBC 历史买过广告 / 信息流 / KOL 软广。20 亿美元 AUM 的 PE 客户群是 LP 与企业家，不是 C 端，付费投放 ROI 极低。 |
| **Owned（自有）** | **~25-30%** | 官网 + 子基金机构页 + 田溯宁个人发声占比的"自有阵地"。但 owned 在 CBC 这里**结构性偏弱**：没有强势公众号 / newsletter / 视频号矩阵，与红杉中国的 owned 强度相差至少一个数量级。 |
| **Earned（赚来）** | **~65-70%** | CBC 的核心打法。机制：①"造词 → 学术化"（产业互联网→智能体互联网，每 10 年一个大词）；②"借场子说"（亚布力、乌镇、国发院、GSMA）；③"借学者背书"（侯宏、北大国发院智库栏目）；④"借被投讲故事"（亚信、世纪互联、雪浪云）。这是非常典型的"Davos-style 中国版"打法。 |

## 5. Confidence + 原因

| 项 | Confidence | 原因 |
|---|---|---|
| §1 CBC 真实渠道清单 | **High** | 官网、LinkedIn、36 氪、新浪财经、亚信官网、北大 NSD 都有交叉验证 |
| §2 对手 baseline | **High** | 奇绩、真格、红杉中国三家都有公开活动报道与官方文章为证 |
| §3 智坊 atelier 渠道推断 | **Medium** | 是 pattern-matching 的 inference，**没有**任何关于"智坊 atelier"的发布证据；只有"如果它要发，CBC 历史习惯会怎么做"这一推论 |
| §4 Paid / Owned / Earned 占比 | **Medium-Low** | 数字是 directional inference，CBC 没有公开的 marketing budget 数据；但"earned 远大于 paid"的方向 high-confidence。 |
| 全局风险 | — | dimension_1、2 都已确认这三个品牌名公开 footprint ≈0，本路也未发现任何反证。如果 CBC 已私下启动智坊 atelier，按其历史 pattern 它**不会**先做大众发布，而是先在承泽企业家圈层 + 田溯宁 op-ed 里铺概念 → 因此"footprint 接近零"本身就是和 CBC 一贯渠道哲学一致的，不能反推"项目不存在"。 |
