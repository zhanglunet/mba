## 0. 关键修正（先于一切其它结论）

**任务源给出的三段式叙事——"ai坊 → harness farm → 智坊 atelier"——在 2026-05-09 时点的公开互联网上不成立。** 但 7 路调研中浮上来一个**真实且更重要**的事实：

- **"AI坊"（首两字母大写英文 AI）真实存在**，不是 "ai 坊"（小写）。它的官方写法是 **"宽带资本·AI坊"**，由 Wuying 在搜狗微信 SERP 与小宇宙 D Talk 节目页双重锁定 [W1][W2]。任务源给的"ai坊"小写写法在搜索引擎里不同 token，会吃 SEO 亏——本身就是任务源属于二手转述的硬证据。
- **"harness farm"** 中英文世界 0 命中。"harness" 真实出处是 HashiCorp 联创 Mitchell Hashimoto 2026-02-05 博文《My AI Adoption Journey》[D1-1]，与 CBC 完全无关。中国 Harness 赛道头部交易（李笛 / 明日新程，Q1 2026）由创新工场 + Atypical + 奇绩创坛主导，**CBC 不在公开 cap table** [D2-1][D6-1]。
- **"智坊 atelier"** 作为 CBC 子品牌 0 命中。"智坊"二字的中文搜索命名空间被海能达 (002583.SZ) 的 **"精工智坊"**（智能制造概念，与 CBC 无关）占据 [W6]。

**最自然的解释：** 三段式叙事大概率是任务发起方拿到的一份未经核实的二手描述（更可能由某 LLM 编织），用 MBA 流水线来反向验真。**Lead 与 5 评委必须区别评估两个不同对象**：
- **真实存在的 AI坊**（CBC 当前在跑的低声量 AI 早期孵化平台）— 可正面打分
- **三段式品牌叙事**（声称把 AI坊 升级为 智坊 atelier）— 公开层面不存在，应作为"叙事尚未落地"对待

## 1. Executive synthesis — 影响力是怎么构建的

1. **CBC 母品牌的影响力机制是"造词 + 借场子 + 借学者"，而非产品发布。** 30 年历史里造过两个核心词："产业互联网"（2014 田溯宁 + 丁健《财经》联合署名）与"智能体互联网"（2025 田 + 北大国发院侯宏新浪财经智库栏目双发）[D3-1][D1-2]。CBC 一贯不做 demo day，不做开发者社区，子品牌（云基地、雪浪云）很少独立 PR。"footprint=0"与 CBC 渠道哲学是一致的，不能反推"项目不存在"[D3-2]。

2. **真实的 AI 载体不是名字，是一个紧耦合的算力—孵化—园区闭环**。Wuying 抓到的硬证据：CBC（资本）↔ **西云算力**（中卫数据中心，田溯宁创立）↔ **丹摩智算**（西云算力的算力品牌）↔ **智优沃**（中关村东升科技园孵化器）↔ **AI坊**（孵化业务）—— 全部由田溯宁本人投资或创立 [W3]。AI坊给早期 AI 创业者**直接发竞争力工资**（OpenAI Startup Fund / a16z Speedrun 路数），这一项 spec 比"智坊 atelier"这个名字重要十倍。

3. **CBC 的影响力杠杆主要靠田溯宁个人 IP，但这个 IP 活在错频道。** 他的 AI 表达 70% 一手 / 30% 二手，落地在"智库 × 运营商 × 老牌财经"（北大国发院、亚布力、GSMA、新浪财经智库、中国移动研究院）；在"创投 × 创业 × AI 应用"频道（拾象、海外独角兽、Founder Park、机器之心、即刻）**自然提及为零** [D7-1][D7-2]。同期红杉中国 xbench、启明智谱、金秋基金的对外渗透是 CBC 的一个数量级以上。**当 2026 年讨论"VC 做 AI 孵化器"时，CBC 不在 top of mind**。

4. **三段名字本身命名学是断裂的，更像 LLM brand exercise 而非真实路线图。** "坊/atelier"是手工/高定隐喻线（汉宋作坊 + 巴黎 Bottega），"harness farm"是工业挽具/工程脚手架隐喻线，三个名字传递三种打架的人格 [D5-1]。"坊"在中国 VC/科技品牌赛道是稀有空赛道（5000 家创业公司里仅浮上青果工坊、紫牛工坊等非头部案例），用了显眼但代价是显得"小作坊"——与 VC 喜欢的规模叙事反向。"智坊"二字命名空间已被海能达精工智坊抢占 SEO，对 CBC 不利。

5. **真实竞争对手不是奇绩 / 真格 / 红杉。** CBC 缺一个陆奇式技术布道型 lieutenant、缺 cohort 运营基因、缺 demo day 校园流量入口 [D4-1]。如果 CBC 真要走 cohort 制孵化器路径，与奇绩正面打需要从零建团队。**真正与 CBC 同频的对手是协议层布道者**：李开复（AI-First 元年）、邬贺铨（智能体即服务）、Cisco（IoA 白皮书）、中国移动 ACN [D6-2]。CBC 应该在协议布道路径上加倍下注，而不是切到 cohort 赛道。

## 2. The Leverage Map — 哪些维度做真活、哪些是装饰

| 维度 | 真实杠杆 | 杠杆质量 | 备注 |
|---|---|---|---|
| 1. 创始人 / 母品牌 | **田溯宁 30 年技术叙事 backbone（亚信→网通→云基地→产业互联网→智能体互联网）** | **强**（结构性资产，不可复制） | 但 IP 落点是"老钱论坛宾客型"，不是 demo day 主持型 |
| 2. 产品 / 定位 | AI坊 真实运营（早期发工资模式 + 算力 / 园区闭环），但**"智坊 atelier"作为升级版本未落地** | 中（AI坊 真但低声量；智坊 atelier 是空名） | 三段式叙事是装饰 |
| 3. 分发 / 渠道 | **北大国发院承泽企业家圈层 + 田溯宁 op-ed + 借大会场子** = earned 65-70%；owned 25-30%；paid<5% | 中（earned 强，owned 弱） | 这是真实在做的事，但天花板是政策圈而非创业圈 |
| 4. 社群 / PR | portfolio CEO 军团（亚信系 / 达观陈运文 / 小 i / 雪浪云）+ 侯宏学术通道 | 中-强（已落地资产） | 缺 lieutenant；缺 demo day 校友光环 |
| 5. 视觉 / 命名 | "坊"占空赛道 + "智坊→atelier"隐喻对位准确 | **弱**（命名学聪明但内核断裂） | "harness farm"是装饰（多半是 LLM 草稿）；"智坊"被海能达占名 |
| 6. 竞争站位 | **协议层布道（智能体互联网）= CBC 唯一可独占的位** | **强（条件成立时）** | 仅当 CBC 不去强行做 cohort 制孵化时此位才稳 |
| 7. 接受度 / 情绪 | 学院 / 运营商 / 老牌财经体系内中性偏正；创业圈静默 | **弱** | 静默就是最重要的负面信号 |

**核心判断**：
- **真做事的杠杆**：母品牌 IP（D1）+ 协议层叙事（D3 / D6）+ 算力—园区闭环（W3）+ AI坊 工资模式（W3）。
- **装饰性的杠杆**："智坊 atelier" 命名（D5）+ "harness farm" 中转概念（D5）+ 三段式品牌升级故事。
- **被低估的杠杆**：AI坊 直接给创业者发工资，这一条在中国 VC 孵化器里激进且差异化，但任何调研都没有把它作为头条。

## 3. The Fragile Edges — 影响力依赖什么单点

1. **依赖田溯宁一人讲故事。** 他 1963 年生，63 岁。整套 CBC AI 叙事（智能体互联网 + ANP 协议）几乎全部由他 + 侯宏一篇文章承担。一旦田溯宁缺席（健康、退休、注意力转移），整套叙事会瞬间失效。CBC 没有"田溯宁 2.0"。
2. **依赖一篇 op-ed 撑起整年叙事。** 2025-09-28 的《智能体互联网》新浪财经文章 + 11-13 重发是 CBC 整 2025 年最高声量 AI 动作。一篇文章的 lifespan 是 6-12 个月——2026 中需要下一篇，否则失声。
3. **依赖侯宏一个学者通道。** 北大国发院承泽企业家研修项目学术主任的双重身份是 CBC 学术背书的唯一管道。侯宏跳槽 / 研究方向转移 = CBC AI 叙事失去学术加冕。
4. **依赖 AI坊 不被外界发现是低声量。** 现在 AI坊 footprint 薄但不为零（搜狗微信 + 小宇宙 + 智优沃公众号），可看作"内部口碑期"。一旦有头部媒体对照写"AI坊 vs 奇绩 vs MiraclePlus"对照稿，目前 CBC 缺 demo day 校友 / 缺数据 / 缺品牌符号，会很难看。
5. **依赖"harness farm"不被英文社区注意到。** 如果 CBC 真在某场合用了这个词，2026-02 之后 Hashimoto 已经把"harness"占为 agent 工程评估词；海外 X / Hacker News 大概率会问"为什么 CBC 蹭 Hashimoto 概念"——是命名层对外攻击向量。
6. **依赖海能达不积极行使"精工智坊"商标 / SEO 控制权。** 真要用"智坊"做主品牌，需要先解决一个智能制造领域已存在 13 年的同名占位。

## 4. Cross-dimension Contradictions — 维度之间的不一致

1. **D2 与 W3 矛盾的解开**：D2 用小写"ai 坊"搜索得 0 命中；W3 用大写"AI坊"搜索拿到硬证据。**任务源的小写写法 (ai 坊) 错了**，正确写法是 AI 坊。这印证 D1/D2 的"footprint=0"是搜索 token 错配造成的部分假象——AI坊 footprint 是"低，但不是零"，仅"智坊 atelier"和"harness farm"是真零。
2. **D1 + D3 矛盾的解开**：D1 说 footprint 接近零是命题失败信号；D3 说 footprint 接近零与 CBC 一贯哲学一致不能反推项目不存在。**两者都对，但适用对象不同**：对"智坊 atelier / harness farm"而言 D1 对（因为完全无内部品牌活动）；对"AI坊"而言 D3 对（CBC 的 default 状态就是低声量）。
3. **D5 与 D6 张力**：D5 说"坊+atelier"双名结构如果真成立可以装下"高定式小批量孵化"定位；D6 说 CBC 缺 cohort 运营基因、跨度风险大。两者一致：**纯靠命名美学但没有运营动作的路径会在 12 个月内失去新闻周期**（D6 列为"脆弱差异化路径"）。
4. **D4 与 D7 关于田溯宁 IP 价值的张力**：D4 说田 portfolio 军团 + 学者通道 + 老钱企业家圈是硬护城河；D7 说创投/创业 KOL 圈零提及他。两者一致：田溯宁 IP **在政策叙事维度强、在创业带货维度弱**——所以孵化器路径决定了哪一面发力。

## 5. 给 5 评委的"两个对象"打分指南

为避免评委混淆，本审计**显式区分两个评分对象**：

### 对象 A — **真实在跑的"宽带资本·AI坊"**（小声量但已落地）
- 已知特征：早期 AI 孵化、给创业者发工资、合伙人有"Jason"、与丹摩智算/西云算力/智优沃/共绩科技 联营、官方写法 AI坊（大写 AI）
- 应被打分的是：发工资模式的差异化、算力—园区闭环的结构性优势、AI坊 在 CBC 体系内的真实位次

### 对象 B — **声称的三段式叙事"AI坊 → harness farm → 智坊 atelier"升级故事**
- 已知特征：在公开互联网完全不存在 / 命名学断裂 / 中转概念无英文圈共识 / 主品牌名命名空间被精工智坊抢占
- 应被打分的是：这套叙事如果真要 launch 的可信度、命名经济学、与 CBC 一贯打法的吻合度

**两个对象的总分会差异很大**——评委如果只看到对象 B 会打地板分；如果只看到对象 A 会高估；本指南让评委在 5 个维度上分别考虑两个对象的合理评分（评委可在自己的 in-character 风格下选择平衡或拆分打分）。
