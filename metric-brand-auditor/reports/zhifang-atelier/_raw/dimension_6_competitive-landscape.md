# Dimension 6 — Competitive Landscape

**Brand under audit:** "智坊 atelier"（前身"ai 坊"，过渡机制"harness farm"），CBC 宽带资本（田溯宁）旗下假定 AI 孵化器品牌
**Cutoff:** 2026-05-09
**Method:** 7 WebSearch + 3 WebFetch（一手优先：奇绩官网、奇绩 2025 春季路演纪要、田溯宁/侯宏《智能体互联网》原文、真格官网/Z Events、火山引擎、AWS 加速器、Atypical 官网）
**Pre-confirmed by sibling agents:** "智坊 atelier / ai 坊 / harness farm" 三个品牌名公开 footprint ≈ 0；CBC 真实 2025 公开叙事是 "智能体互联网"；harness 概念真实出处是 HashiCorp 联创 Hashimoto 2026-02 博文；CBC 不在李笛/明日新程公开 cap table。

## Findings — 真竞争景观

- **奇绩创坛是这条赛道事实上的"基准线"。** 截至 2025 春，已完成 11 期创业营，累计加速 480+ 公司、AI 项目占绝对主流；2025 春季路演日（2025-06-10，北京中关村国际创新中心）一届就放出 57 个项目，AI Agent 方向 28 家、认知智能 18 家、具身 16 家、AI4S 4 家——几乎全 AI [一手：MiraclePlus 官网；二手：智源社区 BAAI 路演纪要]。投资条款 30 万美元换 7%，无对赌、无回购，是中国本土"YC-style"的事实标准。任何新孵化器要进场，都必须先解释"我和奇绩有什么不一样"。
- **真格不是孵化器，是"奖学金 + 黑客松 + 联合开营"组合拳。** 真Summer 奖学金 20-50 万 RMB / 8-12 周；真驿站 13 期、近 400 站友；2025-10-29 与微软孵化器 25-26 年度联合开营（上海，全球巡回硅谷/香港/新加坡）；2025-06 深圳"Level Up！AI 游戏黑客松" [一手：zhenfund.com/News]。投出 Manus、Genspark、Pollo AI、未来不远（2026-01 完成 2 亿天使）等明星 Agent/具身项目。本质是"早期天使 + 社群运营"，不是 cohort 制。
- **红杉中国（HongShan）走的是"AI Day 闭门会 + xbench 评测体系 + 200+ CEO 网络"路线，不孵化。** 第三届 HongShan AI Day 2025（主题"AI Agents: From Copilot to Colleague"）聚集 200+ 成员公司 CEO；合伙人公元发布 xbench 试图重定义 Agent 评测 [一手：hongshan.com 官网文章]。这是"已投后社群"，不是开放招募。
- **创新工场是"塔尖孵化"模式，规模小但赌注大。** 公开披露的 AI 工程院塔尖孵化共 7 家（创新奇智、澜舟、呈元、零一万物等）。2025 主轴是李开复"AI-First 应用爆发年"叙事 + 零一万物全面拥抱 DeepSeek 的战略调整 [一手：cls.cn、新华网]。同时它是李笛/明日新程（中国 Harness 赛道头部）的天使方——这正好是 CBC "harness" 关键词被误关联的源头。
- **字节 Coze 不是孵化器，是开发者平台 + 生态。** 2025-07-26 Coze Studio + Coze Loop 开源（Apache 2.0），48 小时 9000+ Star [一手：GitHub；二手：CSDN/36kr]。火山引擎 V-START 加速器另开一条线，已孵化 50+ 家。Coze 抢的是 Agent 开发者心智，不是早期股权，但它正在变成"上游"——任何中国 Agent 孵化器都要回答"我们和 Coze 是什么关系"。
- **大模型六小龙基本不做对外孵化器。** 智谱（2026-01-07 港股上市）、MiniMax（2026-01-09 港股上市）、阶跃星辰（2026-01 拿下 50 亿 B+ 创纪录）目前都在自身商业化与上市冲刺阶段；公开搜索未找到三家任何一家有"对外孵化器"公开品牌 [一手：财经/36kr]。零一万物战略转向后也已收缩孵化布局。
- **联想之星 + 联想创投是"创业 CEO 特训班 + 早期天使"的另一条事实路径。** 联想之星管理 7 支共 ~35 亿 RMB 早期基金、累计 300+ 项目；2025 Q2 具身智能最活跃机构 TOP10 包含联想创投、锦秋、五源、高瓴创投等 [二手：第一新声榜单、知乎]。这一组传统 VC 没有 cohort 制孵化品牌——这是 CBC 想做"智坊 atelier"时事实上的"同档位空白点"。
- **Atypical Ventures 与 CBC 的真实关系：被外界误关联，实则赛道相邻、cap table 不重叠。** Atypical 邵帅 2025 公开观点是"通用 Agent 红海，看好垂直 Agent + Agent 重塑 SaaS" [二手：智源社区]。Atypical 是李笛/明日新程公开 cap table 中的另一家关键 VC，但 CBC 不在；两家在 Agent/Harness 叙事上有共振，但没有公开协同。
- **CBC 自身 2025 真实叙事 = "智能体互联网" 政策评论 + 田溯宁 MWC 演讲，不是孵化器。** 田溯宁 + 侯宏 2025-09-28《新浪财经》专栏《智能体互联网——被"人工智能+"行动忽视的关键未来》是 CBC 全年最高声量动作；田溯宁 2025-04-10 巴塞罗那 MWC GSMA 思享汇提"数据大发现 / 通信网络进化为感知网络" [一手：finance.sina.com.cn、asiainfo.com]。**全文未提孵化器、未提 atelier / 智坊 / ai 坊 / harness farm**——直接 WebFetch 验证。
- **Gap：政策叙事赛道挤满了"概念布道者"，CBC 在叙事层面的真实对手反而不是孵化器，而是中国移动研究院《ACN 白皮书》、Cisco《Internet of Agents White Paper》、刘军等《ACPs 协作协议体系》论文等"标准/协议"玩家** [二手：telecomsci.com、白皮书原文]。

## CBC 真实站位 — 协议层布道 vs 孵化器运营

公开证据指向**协议层布道**，且只到"政策评论员 + 行业演讲"层级，**没有任何孵化器实操动作**：

- 田溯宁 2025 双主战场：(a) 国际侧 MWC GSMA 思享汇推"数据大发现 / 感知网络 / AGI"；(b) 国内侧《新浪财经》专栏推"智能体互联网应被纳入'人工智能+'政策"。
- 这两个动作的共同点：**都是叙事层 / 概念命名层动作**，不是 cohort 招募、不是 Demo Day、不是基金 LP 公告。
- 与 CBC 一贯打法吻合：2006 起"基金 + 基地"模式（北京云基地）、产业互联网概念命名者——田溯宁的 IP 资产是"概念命名 + 产业网络 + 政策影响力"，不是"陪跑 cohort 100 小时 office hours"。
- 协议布道路径上的真实对手是中国移动研究院 ACN 白皮书、Cisco IoA 白皮书、邬贺铨"智能体即服务"判断、ACPs 协作协议体系学术圈——**全是央企/学院派**，CBC 是少数民营资本玩家。
- 孵化器运营路径上 CBC **没有任何公开人员、cohort、办公空间、投资条款披露**——如果"智坊 atelier"真要落地为运营型孵化器，等于 CBC 跨界到完全陌生的运营赛道，与奇绩、真格、火山引擎正面竞争。

**结论：基于现有证据，CBC 真实身位 ≈ 协议层布道者 + 老牌产业互联网基金；"智坊 atelier"如果存在，更可能是政策叙事 / 智库 / 沙龙级别品牌，而非奇绩式 cohort。**

## "vs" map — 智坊 atelier 真正在和谁抢什么

- **抢"AI 时代的产业互联网命名权" → vs 李开复（AI-First 应用元年）、邬贺铨（智能体即服务）、Cisco（Internet of Agents 白皮书）、中国移动（ACN）。** 这是 CBC 实际正在打的仗。
- **抢"互联网老兵 IP × 协议层叙事"心智 → vs 李开复、张亚勤、王小川、王坚。** 田溯宁的差异化是 1990s 亚信/网通血统 + 产业互联网命名前科。
- **如果真要做 cohort → vs 奇绩 + 真格 + 火山 V-START。** 这场仗里 CBC 没有任何公开筹码：没有陆奇式技术合伙人、没有真格式 800+ 校友网络、没有火山式平台流量。
- **抢命名美学（atelier / 智坊 / harness farm 这种"非工程"词汇）→ 公开赛道暂时无对手。** 中国 AI 孵化器普遍用"创坛/训练营/加速器/X Day"这类工业词汇；如果 CBC 真用 atelier/智坊，会是叙事层独占——但同时也意味着没有受众参考系。

## 差异化路径（按可行性排序）

1. **协议层 + 智库定位（最可行）**：以"智能体互联网"为锚，做白皮书 + 政策建议 + 高层级闭门沙龙。对手是央企研究院，CBC 的民营 + 国际化（MWC）身位有差异化空间。命名上"atelier"作为高端沙龙品牌合理。
2. **田溯宁个人 IP × 协议层叙事 IP 化（次可行）**：把田溯宁包装成"中国 Internet of Agents 命名者"，复刻他 1990s 命名"产业互联网"的成功。这是 IP 资产最大化路径，不需要建运营团队。
3. **CBC 后台资源 × 选择性下注几家协议层创业（中等可行）**：不办 cohort，但用 LP/老股东身份选择性进入 1-2 家"Agent 协议/Agent OS"创业（路径上是阶跃星辰那种 infra 公司，不是 Manus 那种应用）。这条路径与 CBC 历史打法（投熟人 / 投赛道枢纽）一致。
4. **命名美学路径（脆弱）**：纯靠"atelier / 智坊"语言美学 + 田溯宁 IP，没有运营动作。短期声量可制造，但 12 个月内会因为没有 cohort、没有 Demo Day、没有项目 portfolio 而失去新闻周期。
5. **正面挑战奇绩（不可行）**：CBC 缺技术合伙人、缺 cohort 运营基因、缺 office hours 文化，跨度过大。

## Contradictions / gaps

- **关键空白：** "智坊 atelier" 三个品牌名公开 footprint ≈ 0（前两路调研已确认），所以本维度只能基于"如果它真上线 / 它最可能定位为什么"来反推竞争对手。
- **harness 语义错位：** CBC 公开材料无 "harness" 一词；该词 2026-02 由 HashiCorp 的 Hashimoto 用作 agent 工程实践语义；中国"明日新程 / 李笛 / Harness 赛道"由创新工场 + Atypical + 奇绩主导，CBC 不在 cap table。这意味着如果 CBC 真用了"harness farm"，要么是内部代号被外界误传，要么是叙事借词——存在被读成"借用 HashiCorp 概念"的风险。
- **CBC 跨度风险：** 从协议层布道到 cohort 运营是两套完全不同的能力栈。CBC 历史上从未运营过 cohort 制加速器（2010 年的"基金+基地"模式更接近园区/产业园，不是 YC-style cohort）。
- **未验证：** 奇绩 2025 秋季 / 2026 春季路演具体数据未在公开搜索中检索到完整版；红杉中国是否有"Atelier"分支无任何证据，原始 Prompt 中的"红杉 Atelier"假设未得到验证（搜索结果反证）。
- **田溯宁动态时效性：** 调研截止 2026-05；2026-05 后 CBC 是否真有"智坊 atelier"上线动作不可知。

## Confidence

**Overall: Medium (0.55)**

**高置信（0.8+）部分：**
- 奇绩、真格、红杉、字节 Coze、创新工场、阶跃/智谱/MiniMax 的 2024-2026 公开动作和规模数据（一手网站 + 主流财经媒体交叉验证）。
- CBC 2025 真实公开叙事是"智能体互联网"，且田溯宁/侯宏文章原文 WebFetch 验证未提孵化器（直接证据）。

**低置信（0.4-）部分：**
- "智坊 atelier" 真实定位为何（无任何公开材料，全部基于合理推断）。
- CBC 是否真在筹备孵化器（无任何招募、办公空间、人员公告）。
- 各对手"借用智坊 atelier 语言"的可能性（推测，非证据）。

**置信度受限的根本原因：审计对象是"接近零 footprint"的品牌假设，本维度的价值在于校准 CBC 真实站位 + 真竞争对手矩阵，而非裁判一个尚未公开的品牌。**
