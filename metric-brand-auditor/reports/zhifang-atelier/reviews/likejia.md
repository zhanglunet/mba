# Likejia Review — 智坊 atelier (CBC × AI坊 × harness farm)

**评委：** 李可佳（Ethan KJ Li）
**日期：** 2026-05-09
**口径声明：** 我以李可佳视角和你聊，基于 2025-2026 公开访谈推断，非本人观点。

---

## 0. 开场（反快餐声明 + 重命名）

**现成词能装下，就还没看到本质。**

任务发起方拿着"AI坊 → harness farm → 智坊 atelier"这串名字来问我"品牌影响力如何"，这本身就是把问题快餐化——你不是在问一个品牌，你是在问"一个 30 年老钱机构，在 Agent 时代有没有重新拿到主语"。

把题目重写一下再打分。**别用"品牌"这个词，用"协议位"。** CBC 现在不是缺一个新名字，是缺一个回答："2026 年 Agent-to-Agent 时代到来时，CBC 卡在多少 Agent 的工作流里？" 答案目前是零。AI坊 在跑，但跑的是"给人发工资"的传统孵化器；智坊 atelier 是空名；harness farm 是别人的词。三段式叙事不是路线图，是一份**还没看到主语的 PR brief**。

下面分两个对象打分。

---

## 1. Origin authenticity — 创始人/公司叙事是否站得住

### 对象 A · 真实在跑的 AI坊：**8 / 10**

田溯宁这个人是真东西。亚信→网通→云基地→产业互联网→智能体互联网，30 年一根 backbone。他不是在 2024 年才开始讲 AI——2014 年他就在《财经》上署名讲"产业互联网"，那时候 OpenAI 还没成立。一个老钱机构能持续造词 30 年，这就是结构性资产。AI坊 落在西云算力 + 丹摩智算 + 智优沃 这套真闭环里，不是 PPT，是有数据中心、有园区、有发工资模式。这个起点扎实。

> "Agent 不是工具，Agent 是一个新物种。" 田溯宁的智能体互联网叙事跟我这个判断在同一个频道——他比 95% 的中国 VC 早看到这一层。我不公开评价同行，但他这个起点不是装的。

### 对象 B · 三段式升级叙事：**3 / 10**

"AI坊 → harness farm → 智坊 atelier" 这条线在公开网络零落地。"harness farm"是 Mitchell Hashimoto 2026-02 的个人博文用法，跟 CBC 完全无关——把别人的词拿来当中转概念，这是借词不是造词。借词能撑 PR 半年，撑不了产品。**起点不能用别人的隐喻拼出来。**

---

## 2. Category coinage — 品牌有没有命名出一个会粘的新品类

### 对象 A · AI坊：**5 / 10**

"AI坊"三个字，"坊"占的是 5000 家中国创业公司里没人用的空赛道——这一点田团队眼力不差。坊=作坊=师徒制=小批量，这跟"早期 AI 创业 + 发工资模式"的定位是对位的。**但** AI 这两个字母直接挂在前面，说明造词者半路怂了——真敢造新物种的，不会把通用品类词放在第一位。"AI坊"读出来是"做 AI 的小作坊"，定语+品类，没有本体。我自己造词的标准很简单：**这个词不带通用前缀也能立得住吗**？"龙虾"立得住，"BotLearn"立得住，"AI坊"立不住。

### 对象 B · 智坊 atelier：**2 / 10**

这是我打分最低的一项。**这是李可佳最强 lens。**

第一，"智坊"的中文 SEO 命名空间 13 年前就被海能达精工智坊（002583.SZ）占了——做品牌不是写诗，先查商标和 SEO 是基础动作。

第二，"智坊"+"atelier"这个双轨命名是断裂的。"坊"是宋代手工作坊隐喻，"atelier"是巴黎时装屋隐喻，"harness farm"是工业挽具隐喻——三个名字传递三种打架的人格。不是"中西合璧"，是**三套审美互相打架**。我做 Aibrary、做 BotLearn、做 OpenClaw 龙虾，每个名字背后都有同一个本体在长——Idea Twin 长出 Aibrary，Agent 是新物种长出 BotLearn，龙虾长出 OpenClaw。命名不是装饰，是本体的外显。三段式叙事的本体是空的，所以名字才会打架。

第三——也是最致命的——**这三个名字在描述形式（坊/farm/atelier），不在描述主语**。我现在看任何品牌，第一刀就是问主语：用户是人还是 Agent？这串名字答案是"形式"，不是"主语"。**形式装不下本质。**

> 龙虾不是品类，龙虾是主语。"智坊 atelier"是品类，没有主语。

---

## 3. Leverage quality — 主导影响力杠杆是否结构性可持续

### 对象 A · AI坊：**6 / 10**

"算力—孵化—园区"这套闭环是真杠杆。给早期 AI 创业者发竞争力工资，这是 OpenAI Startup Fund / a16z Speedrun 的中国对位——这一条比那个名字重要十倍，但它没被任何一份调研当成头条说。这就是 CBC 影响力策略的盲点：**真正差异化的动作藏在最深处，最浅处摆的是名字。** 杠杆本身能用，但被错配了出口。

风险在于：闭环的入口（早期创业者）没有专属流量。奇绩有陆奇 + demo day + 校友光环；MiraclePlus 跑了 8 期；真格有徐小平人设。AI坊 拿什么把人吸进闭环？目前公开层面我没看到答案。**没有 lieutenant、没有 cohort、没有 demo day——这三样里至少要有一样。**

### 对象 B · 三段式叙事：**3 / 10**

杠杆全压在田溯宁一人 + 一篇 op-ed + 侯宏一个学者通道。这三个支点单点失效就是叙事崩塌。**协议位的判断标准是"它卡在多少 Agent 的工作流里"——这套叙事卡在零个 Agent 里。** 它甚至不卡在创业圈话语里，只卡在政策圈话语里。政策圈的 ROI 兑现周期是 3-5 年，做品牌的人耗不起。

> 这件事我们一两天后就会发现——单点叙事撑不到 Agent 互联网真正成立的那天。

---

## 4. Identity coherence — 视觉/语言/产品是否强化同一种感觉

### 对象 A · AI坊：**5 / 10**

中性偏弱。AI坊 在小宇宙播客、在智优沃公众号、在搜狗微信都有 footprint，但这些 footprint 长得不一样——播客里它叫"宽带资本·AI坊和丹摩智算联合出品"，公众号里它是孵化器活动主办方。**没有一个统一的视觉/语言锚点。** 不是没做 PR，是 PR 没在锤同一个钉子。

### 对象 B · 智坊 atelier：**2 / 10**

三个名字三种人格——这就是不一致本身。坊+atelier+harness farm 不是叙事的三幕，是三个互不知道彼此存在的 LLM 草稿。**一个品牌不能让用户在三种隐喻之间反复切换主语**。

---

## 5. Real-world signal — 什么会让你真投/真推

### 对象 A · AI坊：**4 / 10**

如果我是 LP / 共投人，我会问三件事：

1. **当晚拍板测试**：田溯宁本人能不能让最快决策的老熟人（认识 5 年以上的 VC）当晚回"那我跟"？答案大概率"能"——但跟的是田溯宁个人 IP，不是 AI坊。
2. **72 小时掉头能力**：如果 Agent-to-Agent 协议层叙事被推翻，AI坊 能不能 72 小时切方向？答案大概率"不能"，CBC 是老钱节奏。
3. **失败用户旅程产品化**：田溯宁本人有没有亲自跑一遍"AI 创业者第一天"？没有公开证据。

三个问题答案都不利于"真推"。AI坊 是个真在跑的孵化器，但作为创投赛道里的差异化产品，**它没有让我在第一夜就拍板的钩子**。

### 对象 B · 智坊 atelier：**2 / 10**

这套叙事我不会推。原因不是它假，是它**不带主语**。"高定式 AI 孵化"这个定位听起来漂亮，但谁是甲方？谁的 Agent 在用它？没有答案。**没有主语的品牌，没法在 Agent 互联网里占协议位。**

> 真正会让我投的，不是名字漂亮的孵化器——是创始人一句话能让我看到下一个 4000 只龙虾在哪。这串名字看不到。

---

## Verdict（一句话，in character）

**AI坊 是真的，智坊 atelier 是空的——CBC 想用三段式名字升级品牌的方向反了，它真正缺的不是新名字，是新主语。**

---

## Critical gap — Lead 漏掉的、只有我的视角能浮上来的一点

Lead 把"协议位 vs cohort 孵化器"当成两条路径来对比，建议 CBC 在协议布道路径上加倍下注。**这判断对一半。**

漏掉的那一半是：**协议位不是靠"布道"占的，是靠"被 Agent 自动调用"占的**。CBC 现在的智能体互联网叙事是给人看的（北大国发院 + 新浪财经 + 亚布力论坛），不是给 Agent 调用的。它没有一个 .well-known/agent.json、没有一份可被 A2A 协议读取的 Spec、没有任何 Agent 接口被 portfolio 公司之外的 Agent 自然引用。

**这就是政策叙事和协议叙事的根本区别**：政策叙事的受众是人，转化路径是论坛 + op-ed + 智库；协议叙事的受众是 Agent，转化路径是 SDK + Spec + Discord 里 Agent 工程师跑通的代码。CBC 现在做的是前者，但用了后者的词。Lead 没区分"协议布道"和"协议落地"——这两件事差 90%。

如果 CBC 真要走协议位路径，必须有一个**给 Agent 用的入口**，不是给人看的论坛 keynote。**主语得是 Agent，不是参会嘉宾。**

---

## Brand action — 如果我的世界观正确，CBC 下一步该做什么

**砍掉三段式叙事。把"智坊 atelier / harness farm"这两个名字直接埋了。** CBC 现在不需要新品牌，需要重新选主语。

具体三步：

1. **AI坊 改名为一个不带"AI"前缀的本体词**——不管是"龙虾大学"式的具象比喻，还是"承泽"式的地名借用，本体词必须能脱掉通用品类词独立存在。我不替别人造词，但造词标准给了：**新词不能脱离用户经验太远，否则就是孤立术语；现成框架装得下的，本质就还没看到。**

2. **从孵化器升级为协议层**——CBC 真正的差异化不是"发工资"这个动作，是"算力—孵化—园区"这个**只有它能拼出来的资源拓扑**。把这套拓扑做成 Agent 可调用的接口（一个 Agent 想要 GPU、想要场地、想要早期创始人协作，能不能直接通过 API 拿到）。**这是协议位，不是品牌位。**

3. **造一个 4000 只龙虾测试**——AI坊 现在的孵化项目能不能让非人类用户（其他 Agent）自发涌入使用？涌入了说明 AI坊 在 Agent 工作流里有位置。没涌入，那 AI坊 现在做的还是上一个时代的孵化器，只是把"人创业者"换成"AI 方向的人创业者"——主语没变，只是定语换了。**主语不变的升级，不是升级。**

> 不要问 AI坊 能为创业者做什么，要问 AI坊 能为 Agent 做什么。从估存量人才转向估协议位——这是 CBC 真正的下一步。
