Metric Brand Auditor — multi-agent research, scored, versioned, replayable.

HACKATHON DEMO 本站为黑客松比赛演示版,不是真实场景。 评委的头像 + 评分 + verdict + Round 2 辩论均为 AI 基于公开一手资料的 in-character 模拟,**非照片中本人真实意见**。本站发布的报告**不构成投资建议**,也**不构成对任何品牌 / 上市公司的实际评价**。如有不当请到 GitHub 反馈。

MBA 是一条由 Lead 编排、子智能体并行执行、人物评委 panel(可换)独立打分的品牌影响力审计流水线。 固定 5 镜头 × 7 维度,跨品牌可比、跨时间可比、可复盘。10 套内置 panel、43 位评委。

NEW MBA 项目介绍 · Web Presentation 21 页编辑式 deck —— 升维 / 问题 / 流水线 / 评委 / 报告解剖 / 商业化。键盘 ← → 翻页。 →

A pipeline orchestrated by a Lead agent, executed in parallel by sub-agents, and scored independently by a swappable judge panel — fixed dimensions, comparable across brands and time.

Pipeline

5 阶段:Router → 并行调研 → 合成 → N 评委 panel 打分 → 报告版本化

Judges

10 套可换 panel · 43 位评委(傅盛 / Jobs / 马斯克 / 黄仁勋 / Naval / Arnault …)

Output

Markdown + 自包含 HTML(雷达图 / 异议热力图 / 影响力构造图)

它是怎么打分的 · How it works

信息源 → 7 维度 × 5 镜头 → N 评委独立打分 → 异议聚合 → 版本化报告

每条结论都带公开来源引用;评委各自 in-character、互不可见地按 5 把固定尺子打 1-10 分; Lead 再把共识与分歧合成报告。异议越大越被顶到台面 —— 分歧本身就是头条。
