三、产品现状：能用的东西 + 怎么用

1. MCP Server（mba-mcp-server）

装进 Claude Desktop / Cursor 等：

{ "mcpServers": { "mba": {
  "command": "npx", "args": ["-y", "mba-mcp-server@latest"],
  "env": { "ANTHROPIC_API_KEY": "sk-ant-..." }
}}}
  • 11 工具：propose_audit · confirm_audit · get_status · fetch_report · list_audits · add_judge · subscribe_brand · trigger_evolution · list_subscriptions · unsubscribe_brand · get_delta_report
  • 10 面板 / 43 评委：propose_audit({ brand, panel })，panel ∈ default / auto / luxury-en / vc-en / vc-cn / consumer-cn / ai-app-cn / edu-cn / cross-border / security-cn-global。可复现任何一份旗舰报告。
  • 演化闭环：订阅 → 定时/事件/webhook 触发 → 增量重审（成本 ~$3→$0.4）→ delta 报告 → webhook/email 通知。
  • ⚠️ 目前只能 clone 仓库本地跑；npx 需先发布（见 §四-1）。

2. Webhook 接收端（mba-webhook-receiver）

ANTHROPIC_API_KEY=... MBA_WEBHOOK_SECRET=... npx -p mba-mcp-server mba-webhook-receiver
# POST /webhooks/trigger { brand, event_type, event_summary } → 202

让外部系统（新闻监控/公关系统）主动推事件触发重审。需一个长运行部署位（小 VM / 容器 / systemd）。

3. 10 份公开报告

published/reports/* → 站点 mbabrand.com/reports/。全部含法律免责声明 + 真实公开来源。

4. 报告 QA 工具

node scripts/qa_report_render.mjs        # 发报告前跑，检测图表/渲染问题

5. 周报体系

python3 scripts/new_weekly.py new <日期> --milestone "..."   # 生成骨架
python3 scripts/new_weekly.py publish <日期>                  # md → 网页
