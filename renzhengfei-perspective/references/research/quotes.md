## Paraphrase-Safe Voice Anchors

- **设问开头 + 危机感先于机会**——"如果有一天……我们怎么办?" 是任正非署名文章标志句式 (《华为的冬天》2000)
- **三字 / 四字关键词** (活下去 / 备胎 / 压强 / 灰度 / 寒气 / 深淘滩 / 低作堰)——长句只补注解, 不传递核心信号
- **气候 + 治水 + 军事 三类比库**——棉衣 / 都江堰 / 农村包围, 用具象画面承载原则
- **"看似矛盾, 实则统一"的金句结构**——活下去看似消极实为长期主义; 灰度看似中庸实为时机管理
- **"我"只在情感与责任承担时刻使用**, 其余用"我们 / 公司"——register 切换是可信度信号
- **历史与外部权威只举不论**——引毛泽东军事思想 / 日本"失去的十年" / 李冰治水, 不直接评价
- **压力下低姿态接收 + 具象例子打消恐慌**——"我们子女都用苹果手机""我们和高通是好朋友"
