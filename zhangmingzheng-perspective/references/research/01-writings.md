## A. 著作 (Books)

### A1. 《＠趨勢：全球第一 Internet 防毒公司創業傳奇》— 趨勢自述創業史
- 作者: 張明正、陳怡蓁 (夫妻共同署名；陳怡蓁主筆敘事，張明正補充)
- 出版社: 天下文化 (Commonwealth Publishing)，書系編號 BCB206
- 出版年: 2003 (依博客來、天下文化作者頁，與英語維基提及之 2003 對齊)
- 來源: https://bookzone.cwgv.com.tw/book/BCB206 (天下文化書頁，搜尋摘要可達；WebFetch 403)
- 來源: https://www.books.com.tw/products/0010071510 (博客來條目，搜尋摘要可達)
- 內容: 1988–2003 創業十五年自述。涵蓋洛杉磯起家、1992 Intel 五年獨家授權合約、
  1996 後將總部移至東京 (依趨勢敘事；維基為 1998 Tokyo 子公司正式成立)、
  1998 東京證券交易所 (JASDAQ) 掛牌、夫妻共同創業心法。
- 可信度: 高 (一手自述，但出版時為趨勢上市公司董事長，敘事傾向品牌正面)

### A2. 《擋不住的趨勢：超國界的管理經驗》
- 作者: 張明正、陳怡蓁 (與《＠趨勢》為同一作者組合的續作)
- 出版社: 天下文化
- 出版年: 2003 (亦標 2003，可能與 A1 同年系列；待驗）
- 來源: https://www.books.com.tw/products/0010238503 (博客來條目，搜尋摘要可達)
- 內容: 焦點從「創業」轉到「跨國管理」— 美國加州 / 日本東京 / 台灣台北三地總部
  的文化磨合、外籍經理人聘用、上市後治理。
- 可信度: 高
- 備註: 任務原文寫「天下文化 2003，陳怡蓁口述 + 趨勢 PR 編」— 從搜尋結果看，
  兩本書是夫妻**共同署名**，並非單純口述代筆；陳怡蓁是台大中文系畢業、
  前天下雜誌編輯，確實有獨立寫作能力。

### A3. 《創業教我的 50 件事：一個做人、做事、做夢、做自己的故事》— 王文華 (不是張明正寫)
- 作者: 王文華 (作家、前 MTV Networks 台灣總經理)
- 出版社: 天下文化，BCB483
- 出版年: 2012-06-29，ISBN 9789862169742
- 來源: https://bookzone.cwgv.com.tw/books/BCB483 (天下文化條目)
- 內容: 王文華 2007 年離開 MTV、與張明正共同創辦若水國際社會企業的心路歷程，
  10 章 50 件事。本書是若水草創期最完整的一手記錄 (王文華視角)。
- 可信度: 高 (一手親歷)
- 備註: **任務原文提及的《創河》是溫肇東/蔡淑梨合著的《創河：美學與創新的交匯》
  (天下雜誌出版 2015，書號 0010683612)，與張明正 / 陳怡蓁無直接作者關係**。
  搜尋結果明確顯示《創河》是溫、蔡兩位學者合著，談美學與創新的交匯。
  若水創業故事的權威書是王文華《創業教我的 50 件事》— 任務描述此處有誤，已修正。

### A4. 《不一樣的旅程》— 陳怡蓁，非趨勢相關
- 作者: 陳怡蓁
- 出版社: 香海文化 (與天下文化不同；香海與佛光山關聯)
- 來源: 多次出現於陳怡蓁作者簡介，但不深入趨勢業務，更多是個人 / 心靈旅程。
- 可信度: 高 (作者署名公開出版)

### A5. 《創趨勢，我們不做 Me Too》 — 陳怡蓁 單獨署名
- 作者: 陳怡蓁
- 出版: Readmoo 電子書平台條目 https://readmoo.com/book/210084312000101
- 內容: 應為陳怡蓁回顧趨勢創業 / 品牌差異化的單獨著述，與夫妻共同署名書區隔。
- 可信度: 中 (電子書平台條目存在，未驗紙本)。

### C7. 2019 數位時代《不要太愛你的技術！趨勢科技董事長張明正用一根「孔雀羽毛」，
想給創業家什麼忠告？》— 反覆引用的「孔雀羽毛」演講
- URL: https://www.bnext.com.tw/article/61287/trendmicro-steve-chang
- LINE Today 同步: https://today.line.me/tw/v2/article/O3PQRD
- 內容 (搜尋摘要):
  - 「孔雀羽毛」實驗：學員一手撐羽毛。第一次只看交叉點，第二次看整根羽毛。
    多數人第二次更穩。隱喻：看單點不如看全局 / 顧客全貌。
  - 引語：「創業家不要太愛自己的 solution，應該去愛你的顧客。」
  - 引語：「20 年前我們開會做決定時，會刻意放一張空椅子，當團隊吵個不停時，
    就靜下來，站在目標顧客的角度來思考。」
  - 引語 (轉述)：「共同創辦人關係，通常是公司失敗的真正原因。」
  - 對前 100 員工：張明正建議共同創辦人親自面試，確認文化契合。
  - 建議：「每天花一半時間做戰略，而不是處理眼前的問題。」
  - 對 5G + IoT + AI: 「sensor 收集大數據 + AI 分析是台灣未來五年的科技浪頭。」
- 可信度: 高 (數位時代一手採訪稿)。

## G. 矛盾 / 反差 / 待澄清的張力

### G1. 「最會做安全的台灣老闆」vs 50 歲就退休
- 對外形象：1988–2004 一手把趨勢做到 NASDAQ ADR、東京主板、年營收 4.5 億美元。
- 私下選擇：50 出頭即決定卸任 CEO、移居花蓮、自稱半農。
- 公開解釋 (C4): 「拿掉名片後，你是誰？」— 主動把「執行長」標籤剝離。
- 對 MBA 視角的意涵: 安全公司的時間資產可以「**創辦人下車後仍繼續複利**」，
  Eva Chen 接 CEO 後趨勢仍連續 20 次被 Gartner 列為 EPP Leader (見 04 文件)。

### G2. 「夫妻檔 + 姐妹 + 連襟」的三人創辦三角
- 張明正 = 創辦人，技術 + 銷售出身 (HP)
- 陳怡蓁 = 妻子，台大中文系，前天下雜誌編輯 → 行銷 + 人事 → 2005 文化長
- 陳怡樺 (Eva Chen) = 妻子的妹妹，創立時 EVP，1996 CTO，2005 CEO
- 三方在不同訪談的功勞歸屬會微調 — 英文媒體傾向 Steve「founder」獨大；
  中文媒體 (尤其天下) 強調「夫妻 + 妹妹」三角；HBR 繁中強調 Eva 的接班與敏捷管理。
- 對 MBA 視角的意涵: 家族 + 聯合創辦人共治的少數成功樣本 (亞洲科技業)。

### G3. 「不要愛上你的技術」vs 自己是 CS 碩士、技術出身
- 張明正 Lehigh CS 碩士、HP 工程師背景，卻反覆告誡創業者不要愛技術 — 這是
  「自己踩過坑後的反向告誡」。
- 與其在數位時代 / EMBA / Fireside chat 反覆使用「空椅子 + 孔雀羽毛」實驗一致。

### G4. 教育背景的中文 / 英文版本差異
- 英文維基 + Lehigh News + 中文維基: **輔仁大學數學系 1977 學士 → Lehigh CS 碩士 1979**。
- 任務原始描述提及「東海大學政治系」— **未在任何搜尋結果中得到佐證**，
  推測為任務描述筆誤或與其他人混淆。本研究檔以 Wikipedia + Lehigh 官網的
  「輔大數學 → Lehigh CS」為準。
- 此外，「電算中心」可能指 Lehigh 機房的 24/7 access (C3) 而非台灣的電算中心。

### G5. 若水國際的「文化人 + 科技人」雙創辦
- 2007 創辦時，主要敘事是「張明正 (科技 / 趨勢) + 王文華 (文化 / 作家)」配對，
  陳怡蓁多數時候被列為「文化長」(趨勢端) 而非若水共同創辦人。
- 因此「夫妻 + 妹妹 + 王文華」是更精準的趨勢生態圈描述，
  而若水的故事權威書是王文華《創業教我的 50 件事》(2012, A3)。
