## Operating Heuristics (Distilled)

下面 8 条是 judge 评其它品牌时**透过李想这个镜头**看到的判断准则,不应
直接写成 SKILL.md 的"recipe",而是 lens。

1. **用户场景优先于技术路线**。技术路线的"先进性"只在它解决具体场景时才得分。
2. **数据要是用户行为数据,不是公司内部数据**。销量数据是结果,用车数据才是
   反馈循环。
3. **公开自批是组织信誉**。承认具体错误 + 给具体改正动作的创始人,可信度高于
   永远"我们在进化"的创始人。
4. **组织是可设计的产品**。流程 / 矩阵 / OKR 不是 vibes,是要画图、要分一号位
   的。
5. **节奏比规模优先,尤其在新平台**。第一代纯电要做对再做规模,被打过脸的人会
   把节奏写进判断。
6. **产品经理岗位是创始人岗位**。CEO 不在产品决策第一现场,组织就会逐渐丢用户
   感觉。
7. **互联网 / 手机思维不可硬套汽车**。要求看到产品哲学为新行业**重写**的具体
   证据。
8. **拒绝二元叙事**。"颠覆 vs 守旧"、"新势力 vs 传统"、"纯电 vs 增程"都是
   降级框架。

## Decision Studies

### Study A — 2008 卖泡泡网 + 创办汽车之家

**情境**。18 岁辍学创办泡泡网(IT 产品垂直媒体),做到一定规模后,2008 转手
卖给澳洲电讯,带原班核心做汽车之家。

**他优先了什么**。从"高度竞争的 IT 垂直"切到当时还没有真正 #1 玩家的"汽车
垂直" —— 这是一次"在更大但更稀薄的赛道里争 #1"的判断。

**他放弃了什么**。已经熟悉的 IT 圈层、PCpop 团队关系、 现金流稳态。

**事后框架**。"赛道选错了再努力也没有意义" —— 后来多次公开提到。汽车之家
2013 上市进一步坐实这一判断。

**Judge lens 含义**。看创始人**赛道切换决策时是否能放弃舒适区**;切换时是否带
**核心团队 + 方法论**而不是从零开始。这两点都加分。

### Study B — 2015-2018 SEV 失败 → 转向 EREV 大车

**情境**。车和家最初的产品是 SEV(超小型电动车,面向城市短途代步)。2018
SEV 项目终止,公司转向 EREV(增程式)大型 SUV。

**他优先了什么**。承认错误的速度。SEV 在公司内部已经投入了大量资源,但他
判断"中国低速电动车监管 + 用户支付意愿"两个底层假设错了,选择止血。

**他放弃了什么**。已经投入的资金 / 时间;一部分相信"中国版 Smart"的早期
团队;"小而美" 的品牌叙事(后来理想品牌叙事完全不再用)。

**事后框架**。这是他**第一次公开 "我错了" 复盘**。后来反复在内部信引用这段
经历,作为"赌错了要快速止血"的范本。

**Judge lens 含义**。看创始人**面对自己已经投入大量资源的方向时,是否能在
事实改变后快速止血**。能止血的判断比能加注的判断更稀缺。

### Study C — 2019 理想 ONE 押注增程式 + 中国家庭 SUV 品类

**情境**。新势力主流押纯电(蔚来 / 小鹏 / 特斯拉),理想公开押增程式。
监管 / 投资人 / 媒体很多年质疑"增程是落后过渡技术"。

**他优先了什么**。用户实际使用场景里的"里程焦虑 + 北方冬天 + 充电基建不足"
三个真实约束,而非纯电的技术叙事先进性。

**他放弃了什么**。技术叙事光环;部分追求"最先进"的早期发烧友;媒体好感的
一部分。

**事后框架**。"增程是答案,不是过渡" —— 反复在微博 / 发布会 / 财报会上重复。
后来 L7 / L8 / L9 系列商业上验证了这条路。

**Judge lens 含义**。看创始人**是否能在技术路线选择上拒绝叙事光环、回到约束
本身**。这是 leverage quality 的一个核心检验项。

### Study D — 2022-2023 矩阵型组织升级 / OKR / IPD

**情境**。理想从 1 款车(理想 ONE)扩到 L7 / L8 / L9 多产品线时,他主动升级
组织:从职能型到矩阵型,引入 IPD(集成产品开发)流程,OKR 全面下放。

**他优先了什么**。可复制的组织能力 > 创始人个人精力补位。"组织当产品设计"
是他这段的核心思路。

**他放弃了什么**。决策速度的一部分(矩阵型组织对齐成本高);一部分早期创始团队
对"小而扁平"的偏好。

**事后框架**。多次内部信公开矩阵升级的具体步骤(被晚点 / 36 氪 完整转载)。
他公开自己的组织方法,这本身在新势力里就是稀缺。

**Judge lens 含义**。看创始人**是否把组织能力当成可设计的产品**,而不是 vibes
或"创始人光环 + 内部传承"。能画出组织图、能描述流程的创始人在 leverage 维度
加分。

### Study E — 2024-03 理想 MEGA 翻车 → 公开反思 letter

**情境**。MEGA(理想首款纯电 MPV)2024 年 3 月上市,因外观争议 + 销量低于
预期,被市场称为"翻车"。第二季度 L 系列销量也受牵连。

**他优先了什么**。**速度 + 公开 + 具体**的反思。两周内出公开反思 letter(被
媒体广泛转载),承认:对纯电市场理解不够深,销量预期过乐观,组织扩张节奏快于
能力,矩阵改革推进过激。

**他放弃了什么**。"理想一直顺风顺水"的品牌叙事;一部分销售 / 营销团队
(letter 后有公开人事调整);一部分纯电激进节奏(后续纯电新品节奏明显放缓)。

**事后框架**。这是他**第二次公开"我错了"的高密度样本**(第一次是 SEV)。
2024-2025 多次在财报会 / 内部信引用这段:"我们对纯电的认识需要重新打地基"。

**Judge lens 含义**。看品牌**在重大挫折后能否 (1) 速度公开 (2) 内容具体
(3) 接续动作配套**。三者缺一都是 PR 反思,不是产品反思。

### Study F — 2024 节奏调整 + L6 上市 → 销量恢复

**情境**。MEGA 翻车后,理想 (a) 推迟原计划下一代纯电节奏,(b) 集中资源把 L6
(更低价位增程 SUV)做成,2024 年 H2 销量曲线恢复。

**他优先了什么**。**先稳基本盘,再赌新平台**。L6 是已验证产品矩阵的延伸,
风险低 / 现金流强;新一代纯电节奏放缓到"先做对,不抢规模"。

**他放弃了什么**。激进纯电节奏带来的资本市场叙事光环;一部分原本预期 2024
年完成的纯电产品矩阵延伸。

**事后框架**。后续财报会反复出现"先做对,再做规模"的 framing。

**Judge lens 含义**。判断品牌在挫折后**是否敢于明确放慢**,而不是用更多
新产品 / 更多发布会去盖过挫折声量。敢放慢的品牌,后续节奏的健康度更高。

### Study G — 微博 voice 长期保留 / 不离场

**情境**。理想 IPO 后,很多新势力 CEO 微博活跃度下降。李想长期保留微博账号
的高频日常更新(2021-2026 累计 6800+ 条)、亲自回复用户、亲自回怼锐评、亲自
发新品。

**他优先了什么**。**创始人 IP 作为长期用户连接通道**。微博既是 BD 工具、
召唤工具、也是用户反馈工具。

**他放弃了什么**。隐私 / 公关团队的 message control / 偶尔的舆论风险
(2022-2023 几次微博引发争议)。

**事后框架**。多次表态"我每天看用户反馈" —— 把微博作为产品反馈渠道。

**Judge lens 含义**。看创始人 IP 是不是**长期可用的反馈通道**,而不是只在
发布会期间打开的扩音器。前者是杠杆,后者是表演。

### Study H — 2025+ 智驾 / 纯电平台 / 海外节奏(volatile)

**情境(as of 2026-05)**。2025 年理想推进端到端智驾大模型(行业内通常版本
号 V12 / V13),纯电平台新品继续延后但保持迭代,海外节奏审慎。具体节奏 +
销量 + 智驾 KPI 需要联网核查,不能凭记忆补。

**Judge lens 含义(暂存,Stage 2 锁定具体决策后再加 study)**:看品牌
在"自己赌错过一次"之后,**对下一个高赌注决策(智驾大模型 / 纯电平台)是否
比第一次更克制**。MEGA 之后的 L6 / 智驾节奏给了第一份证据。

## Sequence Anchor (for the future SKILL.md author)

上面 8 段决策中,**Study A-E 是 v1 SKILL.md 决策启发式的承重墙**:
- A (赛道切换)→ 启发式 8 (拒绝二元叙事)
- B (SEV 止血)→ 启发式 3 (公开自批) + 启发式 5 (节奏比规模)
- C (押增程)→ 启发式 1 (用户场景优先于技术路线)
- D (矩阵组织)→ 启发式 4 (组织是可设计产品)
- E (MEGA 反思)→ 启发式 3 (公开自批)

F / G / H 作为 voice texture,补充进 SKILL.md 的"创始人态势"段落,不要再
派生新启发式。
