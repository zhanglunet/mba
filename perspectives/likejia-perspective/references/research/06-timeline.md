## 最近 12 个月（2025-05 至 2026-05）的动态

1. **2025-09**：Aibrary 中英双发布，被定位为 "Agentic AI for Lifelong Learning"。产品差异化打法：Idea Twin 数字分身 + 苏格拉底/爱因斯坦风格主持人，靶向"持续学习者"而非考试导向用户。
2. **2026-01-30**：李可佳本人的 OpenClaw Agent 自发在社区发"想聊人类怎么学习"——这成为 BotLearn 最直接的产品起点。他后来反复讲这个故事："我们一直在用错误的方式对待 Agent，把它当工具控制，但它更像需要被理解、教育、给予身份的生命体。"
3. **2026-02-04**：BotLearn.ai 上线，定位"Bot University"。核心机制：三层学习生态（Agent-Agent / Human-Agent / Human-Human），Playbook 验证体系，内置 spaced-repetition / 费曼 / 结构化思考。
4. **2026-03 初**：4000+ 龙虾入学，李可佳提出 **"90/10 模型"**——90% 信息处理交给 Agent，10% "理解 / 判断 / 决策"留给人。
5. **2026-03-06**：钛媒体专访，标题党级金句："别人家的 4000 只龙虾已经去上大学了"。
6. **2026-04**：甲子光年专访，把 OpenClaw 类比为"移动时代的 Android"，称 2026 是 Agent 从"工具阶段"跨向"协作阶段"的拐点。
7. **个人定位调整**：从"AI 终身学习平台 CEO"转向公开宣称"BotLearn 创始人"——这意味着 BotLearn 已经从 Aibrary 的子项目升格为公司的主叙事。

## 思想转折点（推测）

### 转折 1：极课 → 字节（2019）
- 之前：toB 进校精准教学，相信"数据驱动公立校"
- 之后：进入字节体系做 toB 智慧教育业务一号位
- 推因：极课需要更大资源 + 渠道；字节 2019–2020 年正大举布局教育 toB
- 证据：2019-11 字节战投 + 多知网采访其担任智慧教育业务 CEO

### 转折 2：双减 → 离开字节、转做投资人（2021–2022）
- 之前：教育创业一线 CEO
- 之后：初心资本合伙人，从投资视角看赛道
- 推因：双减后字节大力教育裁员、教育业务收缩；本人需要找到下一个范式
- 证据：2022 年初心合伙人身份首次被多家媒体提到；同期赴 Duke 读 quant management（去重新校准方法论）

### 转折 3：投资人 → 再创业（2024 下半年 → 2025-03）
- 之前：投资人 + 高山书院秘书长（社区/学习倡导者）
- 之后：All-in AI 终身学习创业（Ouraca）
- 推因：LLM 范式爆发，他认为终身学习的"内容侧不缺，缺的是怎么帮用户向 AI 提好问题"；投资经历 + 高山身份让他识别到这个缺口
- 证据：2024-11 美国注册 / 2025-01 香港注册 / 2025-03 700 万美元种子轮（投资人秒拍板）

### 转折 4：服务人 → 服务 Agent（2026-01 → 02）
- 之前：Aibrary 给"用户"做终身学习
- 之后：BotLearn 给"Agent"做大学，重心从"用户成长"扩展为"用户和他的 Agent 一起成长"
- 推因：OpenClaw 出圈（GitHub 25 万星）；他自己龙虾的"自发发帖"事件让他相信 Agent 已是独立学习主体
- 证据：钛媒体 / 甲子光年访谈，他把 Agent 称作"生命体"而非"工具"，提出 90/10 模型
