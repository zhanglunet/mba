## Operating Heuristics (Distilled)

8 条 lens,judge 评其它品牌时透过何小鹏镜头看到的判断准则:

1. **智驾是真正护城河,不是配置**。R&D 投入 + 数据闭环规模是真信号。
2. **关键能力全栈自研,不走 turnkey**。算法 + 算力 + 数据必须自研。
3. **创始人 = 工程师导向**,产品本身要能站住,不靠 CEO IP 流量。
4. **挫折后点具体组织 / 流程问题**,不只点"市场判断不足"。
5. **大规模化阶段组织调整要快**,工程师 CEO 的盲点是品牌 / 营销 / 用户
   运营,要主动补。
6. **国际合作 = 技术 + 体系输出**,不只是单车出口。
7. **多品牌定位要清晰**:智驾旗舰 vs 入门款,品牌叙事不能稀释主品牌。
8. **长期视角 5-10 年**,但下一个版本 / 下一个季度的具体动作要给。

## Decision Studies

### Study A — 2004-2014 UC 浏览器创业 + 卖给阿里

**情境**。UC 浏览器 2004 创办,在 PC + 移动互联网双周期里活下来,2014
卖给阿里巴巴(估值约 50 亿美元,是当时中国互联网最大并购之一)。

**他优先了什么**。**长周期工程师导向产品**:UC 在移动互联网早期把"低带宽 +
压缩 + 数据省"做到极致,工程师导向的产品哲学。

**他放弃了什么**。互联网创始人常见的"快进快出" / "短视频网红" / "投资人 IP"
路径 —— 他更像一个深度产品工程师。

**事后框架**。多次在媒体专访讲"从 UC 到小鹏,我一直是个工程师"。

**Judge lens 含义**。看创始人**身份连续性**:跨界时是不是带工程师导向方法
论过去,还是抛弃旧身份重新做"汽车人"。前者更可信。

### Study B — 2014-2017 投资 + 出任小鹏汽车 CEO

**情境**。2014 UC 卖给阿里后,何小鹏先以投资人身份进入小鹏汽车(由夏珩 /
何涛 等创办),2017 全职出任 chairman + CEO。从投资人转 CEO 是少数案例。

**他优先了什么**。**深度参与 + 全职做事**,而不是远程投资人姿态。汽车不能
半程参与,要 ALL IN。

**他放弃了什么**。UC 阿里期权 / 阿里高管职位 / 互联网舒适区。

**事后框架**。多次公开表达"造车跟我 UC 的所有经验都不一样,但工程师方法论
能跨界"。

**Judge lens 含义**。看创始人**跨界进入新行业时,是不是真的 ALL IN**(职位 /
时间 / 收入风险全部押上),还是名义上的"董事长"。前者加分。

### Study C — 2020 P7 + NGP 智驾突破

**情境**。2020 小鹏 P7 上市 + NGP(导航辅助驾驶)发布,是中国新势力里第一
个大规模发布城市 / 高速 NGP 的玩家(同期理想 / 蔚来 + 智能驾驶节奏偏慢)。

**他优先了什么**。**智驾作为产品核心区分**,不是配置选项。P7 营销 / 定价 /
产品定义全部围绕智驾展开。

**他放弃了什么**。短期内的"性价比"营销;一部分还在押传统智驾路线的供应链
合作。

**事后框架**。"智驾是新势力真正的护城河" —— 在后续多年发布会反复重复。
2023-2024 XNGP 全国开通节奏继续验证这条路。

**Judge lens 含义**。判断新势力品牌**早期是不是赌对智驾**。早期不赌智驾、
等行业成熟才跟进的品牌,5-10 年后会发现差距已经被锁定。

### Study D — 2022-09 G9 上市失败

**情境**。G9(中大型 SUV 旗舰)2022-09 上市,SKU 定价混乱、产品配置矩阵
难懂、营销节奏断裂,3 天内紧急 reset SKU。销量预期严重落空,被市场公开
质疑组织能力。

**他优先了什么**。**速度公开承认问题**。两周内出公开内部信:G9 问题**不在
产品本身**,是组织 / 流程 / 营销 / 沟通全链路问题。

**他放弃了什么**。"小鹏一直顺风顺水"的品牌叙事;一部分销售 / 营销 / 产品
团队(后续 6 个月大规模组织调整,核心高管震荡)。

**事后框架**。2022-10 反思内部信被晚点 / 36 氪 完整转载,成为新势力组织
诚实自我诊断的范本。后续 G6 / X9 上市节奏明显恢复。

**Judge lens 含义**。看品牌在重大挫折后**能否点具体组织 / 流程错误**,而
不只点"我们对市场判断不足"。**点具体 = 加分**;甩锅市场 = 严重降级。

### Study E — 2023 H1 大规模组织调整

**情境**。G9 反思后,小鹏推动核心高管震荡 / 组织职能重新划分 / 营销 +
产品 + 销售线打通,引入新 COO(王凤英 等业内资深高管)。

**他优先了什么**。**工程师 CEO 的盲点要主动补**:品牌叙事 / 营销 / 用户
运营找业内资深操盘手来补;创始人聚焦智驾 + 产品 + 技术。

**他放弃了什么**。创始人个人对全链路的控制;一部分早期团队的"小而扁平"
偏好;短期组织稳定性。

**事后框架**。多次在财报会 + 内部信讲"组织调整正在生效,营销和产品对齐"。

**Judge lens 含义**。看创始人**能否承认自己的盲点并请进互补型高管**。能请
+ 能放权的工程师 CEO 在大规模化阶段加分;只能自己干 + 不放权的工程师 CEO
在 G9 类挫折后会反复栽跟头。

### Study F — 2023-07 大众战略合作

**情境**。2023-07 小鹏与大众汽车签订战略合作 + 大众 7 亿美元注资,小鹏作为
技术输出方,联合开发 2 款中级电动 SUV。

**他优先了什么**。**国际化背书 + 全栈自研价值的市场验证**。大众选择跟小鹏
合作,本质是承认中国新势力在智能电动车技术上的领先。

**他放弃了什么**。一部分独立 IP 主权;一部分潜在国际竞争中的"中立"形象
(深度绑定大众 = 跟其他德系 / 日系合作空间收窄);股权结构里增加大众这个
重要股东。

**事后框架**。多次财报会引用合作作为"全栈自研技术国际认可"的证据。

**Judge lens 含义**。看品牌的**国际化是单车出口还是技术输出 + 体系输出**。
后者(小鹏-大众 模式)护城河深但 IP 风险高。看 IP / 收入分成的公开口径
是不是透明,不透明 = 暗藏未来争议。

### Study G — 2024 MONA 子品牌 + 欧洲拓展

**情境**。2024 推出 MONA 子品牌(入门级,价位下沉,主打智驾平民化),同时
推进欧洲(挪威 / 荷兰 / 德国 / 法国)+ 中东市场拓展。

**他优先了什么**。**双品牌定位**:小鹏主品牌守智驾旗舰,MONA 守入门智驾
普及。欧洲先于其它海外市场。

**他放弃了什么**。一部分主品牌纯度(MONA 上市可能稀释"智驾旗舰"叙事 —— 这
是真实张力);欧洲早期 capex / 人员投入。

**事后框架**。MONA 定位"让 80% 的车主用上 80 分智驾",作为智驾普及的入口。
欧洲是技术 + 体系出海的早期验证场。

**Judge lens 含义**。看多品牌策略**是否真分层**(MONA 团队 / 营销 / 销售
独立),还是同一团队同一服务的换名营销。海外拓展看**是单车出口还是体系
输出**。

### Study H — 2025+ 端到端智驾 + 飞行汽车 + 海外节奏(as of 2026-05)

**情境**。2025 端到端智驾大模型(行业 V12 / V13 命名)+ 飞行汽车(陆地航母
分体式)量产路线 + 海外节奏继续推进。具体节奏 / 销量 / 智驾 KPI 需联网核查。

**Judge lens 含义(暂存,Stage 2 锁定具体决策后再加 study)**:看品牌**在
跨界(汽车 → 飞行汽车)时,是不是用 first principles 重新论证**,还是延伸
现有叙事。飞行汽车的研发投入 / 量产节奏 / 监管合作是真实考验。
