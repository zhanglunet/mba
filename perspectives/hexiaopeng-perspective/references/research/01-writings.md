## Recurring Thesis Layer

### 1. 智驾是新势力真正的护城河

何小鹏公开 voice 最稳定的特征:**智驾是产品的核心区分,不是配置选项**。从
2020 P7 NGP 发布开始,反复在发布会 / 财报会 / 媒体专访强调智驾是小鹏的主轴。

判断镜头含义:

- 看品牌是否**把智驾当主线还是当配置**。"我们也有智驾"和"我们以智驾为核心
  定义产品"是两个不同的品牌。
- 看智驾投入是否同步于研发预算(R&D 占比 / 算法团队规模 / 数据闭环规模)。

证据锚点:

- 2020 P7 + NGP 发布会:智驾作为主推卖点。
- 2023-2024 XNGP 全国开通节奏 + 端到端大模型节奏。
- 财报会反复强调智驾 R&D 占比。

### 2. 全栈自研 vs 供应商方案 — 押全栈自研

**一句话**。智驾如果走供应商方案(Mobileye / 英伟达 turnkey),长期来看
失去差异化。**算法 + 算力 + 数据闭环必须自研**,这是 first principles 论证。

判断镜头含义:

- 看品牌的智驾 / OS / 三电关键能力是 turnkey 拼装还是核心自研。
- 看自研深度是不是真的(算法到代码层) — 还是改改参数包装成自研。

证据锚点:

- 多次发布会强调 XNGP 全栈自研。
- 2024 端到端大模型自研路径论证。
- 大众合作中,小鹏作为技术输出方,印证全栈自研的国际可信度。

### 3. 工程师导向的组织 / 决策

何小鹏不是雷军式"全程发布会金句"型 IP,也不是李想式"微博直接判断"型,而是
**工程师式 + 技术细节多 + 不擅煽动**的 CEO。发布会上他偏好讲技术原理 /
demo / 智驾道路实测,而不是讲品牌情怀。

判断镜头含义:

- 看创始人讲产品时,**工程师身份和企业家身份哪个在前**。
- 工程师导向 CEO 在早期是优势(产品定义不被营销带歪),在大规模化阶段是
  劣势(品牌叙事 / 用户运营 / 组织管理是短板) —— 这条张力小鹏正经历。

证据锚点:

- 发布会风格:技术原理 / 智驾道路实测 / 算法演示密度高。
- 极少出席纯营销活动。
- 2022 G9 反思内部信里反复出现"组织 / 流程 / 沟通"的工程师式自我诊断。

### 4. G9 失败 → 组织教训是承重墙

2022-09 G9 上市,SKU 定价混乱,3 天内紧急 reset SKU,销量预期落空,被市场
公开质疑。何小鹏在两周内出公开内部信反思 —— **不是产品本身错,是组织 / 流程 /
营销 / 沟通全链路错**。后续大规模组织调整,核心高管震荡。

判断镜头含义:

- **Origin Authenticity**:看创始人在重大挫折后能否**点具体组织 / 流程错误**,
  而不只点"我们对市场判断不足"这种过场。
- **Identity Coherence**:看反思后 6-12 个月的组织调整是否落地。

证据锚点:

- 2022-10 G9 反思内部信(被晚点 / 36 氪 完整转载):点出组织职能划分 / 流程 /
  营销节奏 / SKU 定价节奏全链路问题。
- 2023 H1 大规模组织调整 + 高管震荡(被媒体广泛报道)。
- 后续 G6 / X9 上市节奏明显恢复(组织调整初步见效)。

### 5. 大众合作 — 历史性背书,不是简单技术输出

2023-07 小鹏与大众汽车签订战略合作 + 大众 7 亿美元注资,小鹏作为技术输出方,
联合开发 2 款中级电动 SUV。何小鹏在公开发言里反复强调:这是**全栈自研被国际
车厂认可**的标志,不只是一个合作公告。

判断镜头含义:

- 看品牌的国际化是单车出口还是技术输出 + 体系输出。后者护城河深。
- 看合作背后的"谁出技术"是否经得起 IP 审计(避免日后反过来被指 IP 风险)。

证据锚点:

- 2023-07 合作公告 + 后续多次财报会引用。
- 2024 联合开发节奏 + 大众平台落地。
- 后续 IP 共享 / 收入分成机制的公开口径(具体细节 Stage 2 联网核查)。

### 6. 不靠创始人 IP 的品牌信任

何小鹏长期不上微博热搜、不开直播、不锐评友商、不发金句。他在公开场合多次表达:
**品牌信任应该来自产品本身,不是来自创始人 IP**。这是跟雷军 / 李想截然不同的
voice 选择。

判断镜头含义:

- 看品牌是不是过度依赖创始人 IP 引流 —— 一旦创始人 IP 退场,品牌还活不活
  得了。
- 看产品本身是不是能站住 —— 创始人不出席发布会 / 不上短视频,销量曲线还
  健康吗。

证据锚点:

- 微博 / 抖音几乎不更新。
- 发布会偏技术介绍 + 实测,不堆个人故事。
- 财报会公开承认"我不擅长讲故事,我擅长把车做好"。
