# 张一鸣著作、长文、内部演讲调研笔记

研究范围：张一鸣本人撰写或发表的系统性内容、字节跳动内部演讲全文、公开推荐的书单。
研究方法：黑名单（知乎/微信公众号/百度系）信息源不直接采信，仅作为发现一手原文的索引。
完成日期：2026-05-09。
分类标记：[一手] = 张一鸣自己写/说的逐字原文；[二手] = 媒体或他人整理转述；[推测] = 来源链条不可考。

## 0. 整体格局：为什么张一鸣的"著作"很少

张一鸣**没有出版任何书**。他的全部"系统性长文"分三块：
1. **早年微博 / 饭否 / blog 51cto**（2010-2013 创业初期，最暴露原始思考）
2. **字节跳动周年庆内部演讲**（2017-2021，对外曝光的几次最重要）
3. **零星采访长文**（极客公园、财经、晚点 LatePost、36氪、第一财经）

他公开演讲非常少。2014 年 4 月在极客公园 IF 大会与 Elon Musk 同台是他的"首次重要公开演讲"（来源：极客公园 https://www.geekpark.net/news/241561 ，[二手]）。在那之前的 2011 年他公开表态过"自己的产品要好好做两年，这两年绝对不出去说"（来源：极客公园 [二手]，转述其受周鸿祎影响），这条信息可解释为什么 2011-2013 张一鸣公开发声极少而微博却很活跃 —— 微博在他那里替代了公开演讲。

## 1. 周年庆演讲（按可信度从高到低）

字节成立日 2012-03-09，周年庆每年 3 月。

### 1.1 5 周年演讲（2017）：《CEO 要保持 "Context, not Control"》

这是网络上传播最广、被反复引用的一次演讲。**注意：用户在任务中标注的"6周年演讲 2018《Context, not Control》"在源头追溯后应为 5周年（2017），2018 年码荟年会演讲张一鸣本人开场已说"去年，我讲了如何应对公司变大之后的管理挑战，分享了我们公司的一个实践，'Context, not Control'"（来源：Source Code Capital 官网 https://sourcecodecap.com/code-class-post/zijietiaodongzhangyiming-waibushijiaoyufanxiangshangguanliruhebimianqiyedefuguimoxiao/ ，[一手 transcript]）。**

核心论点（[一手] 直接引文经媒体转述，无字节官方原始全文 PDF）：
- 用计算机两种处理方式作类比：超级计算机（密集）vs 分布式（拆解任务，群智涌现）。张一鸣选择后者。
- 反对"自上而下的一人思考法"，"避免 CEO 理性的自负"。
- 主张"信息透明，让大家都参与思考和决策，群智涌现"。
- 实现机制：飞书协同文档、CEO 面对面会议、头条圈跨部门提案。

**矛盾/争议点**：「Context, not Control」这个英文短语并非张一鸣原创 —— Netflix CEO Reed Hastings 在 2009 年的 *Netflix Culture Deck* 中已使用 "Lead with context, not control" 作为管理原则。张一鸣的贡献是引入并内化到字节实践，但中文圈很多文章把它说成是张一鸣的发明，这是不准确的。**张一鸣本人是否承认借鉴 Netflix 文献尚未在一手源中找到明确记录。**

**仅有二手转述，无字节官方发布的演讲全文 PDF/录音公开可访问。** 即使是 Source Code Capital 转载的也是节选 + 整理稿，不是逐字稿。

来源：
- Source Code Capital（一手主办方，2018 年码荟，张一鸣亲口提及） https://sourcecodecap.com/code-class-post/zijietiaodongzhangyiming-waibushijiaoyufanxiangshangguanliruhebimianqiyedefuguimoxiao/
- Runwise 整理 https://runwise.co/applied-leadership/23958/ （[二手]）

### 1.2 7 周年演讲（2019-03-14）：《放小自己，放大格局 / 我们是务实而浪漫的人》

这是相对较完整能拿到全文的一次内部演讲（经济观察网公开报道版本最接近，36氪/极客公园也有完整稿流传）。来源链：
- 经济观察网 http://www.eeo.com.cn/2019/0314/350331.shtml （[一手 transcript]，主流媒体公开报道，可信度高）
- 极客公园专访 https://www.geekpark.net/news/241561 （[二手] 整理）

**核心金句 [一手]**：
- "**我们是一群务实而浪漫的人**" / "**务实浪漫就是把想象变成现实，face reality and change it**"
- "**做正确的事，而不是容易的事**"
- "**大力出奇迹**"（这次演讲是该术语在公开场合最典型的语境）
- "**有同理心是务实，有想象力是浪漫；同理心是地基，想象力是天空，中间是逻辑和工具**"
- "**ego 小，格局大**"
- "**AB 测试是测不出用户需求的，只是一个工具而已**"

**张一鸣"浪漫"的反向定义** [一手]：
- "**精致的文艺不是浪漫，粗糙的宏大是浪漫，新事物都是粗糙的**"
- "**晒情怀故意感动别人不是浪漫，独立思考穿越喧嚣是浪漫**"
- 浪漫 = 拥抱不确定性、保持可能性、面向未来、有生命力

**演讲中讲的 5 个故事**（这些是反复在张一鸣表达体系里出现的母题，比金句更能揭示其信念）：

1. **2012 锦秋家园做推荐引擎**（张一鸣最常讲的故事，至少在 5/7/9 周年演讲中均出现）
   - 团队"没有算法基因"，但张一鸣坚持"可以学"。
   - 自己上网找资料，向《推荐系统实践》作者发邮件求电子版被拒。
   - "**只能网上找资料，自己想象着写出了第一版的推荐引擎**"。
   - 寓意：不被既有能力边界约束。

2. **国际化 ByteDance 命名**（2012 年底就预设英文名）
   - "**当时团队鲜少出过国，但已经决定要做国际化**"。
   - 寓意：超前规划。

3. **印度 APEC 证拒签**（一位同事拿 APEC 证去印度被海关拦）
   - 寓意：执行层在不确定性中硬闯。

4. **头条寻人**（拒绝低效 404 方案，已找到 8521 人）
   - "**做正确的事，而不是容易的事**"的具体例证。

5. **短视频赛道反复**（2014 犹豫、2015 错过美拍快手、2015 冲绳年会决定不放弃、2016 国内国外都做）
   - 寓意：屡败屡战，"大力出奇迹"。

**关于多闪 [一手]**：
- "**对多闪的预期就是没有预期**"
- "**我们可以放弃商业利益，避免竞争，不做某件事，但如何面对这 20 万用户的吐槽？**"
- 这段很重要：揭示张一鸣的"用户视角"修辞如何用于合理化竞争行为。

**引用文献**：
- 《史蒂夫·乔布斯传》（沃尔特·艾萨克森）— 引用乔布斯关于"破旧汽车"的比喻
- 《爱丽丝梦游仙境》— 皇后"早餐前相信六件不可能的事" "Why, sometimes I've believed as many as six impossible things before breakfast"

### 1.3 9 周年演讲（2021-03-30）：《平常心做非常事》

这是张一鸣**作为 CEO 的最后一次周年庆演讲**（5 月 20 日宣布卸任）。是网络可获得全文最完整的一次。

来源（按可信度）：
- 澎湃新闻 https://m.thepaper.cn/newsDetail_forward_11979231 （[一手 transcript]，主流媒体）
- 晚点 LatePost 转载于新浪 https://finance.sina.com.cn/tech/2021-03-30/doc-ikknscsk5047837.shtml （[一手]，晚点是字节内部信首发渠道之一）
- 华尔街见闻 https://wallstreetcn.com/articles/3625622 （[二手] 转载晚点）

**核心金句 [一手]**：
- "**平常心做非常事**" / "**外部波澜起伏，内心平静如常**"
- "**平常心的直白定义就是：吃饭的时候好好吃饭，睡觉的时候好好睡觉**"
- "**保持平常心的人，比较放松，内心没有扭曲，观察事物细腻，实事求是，比较有耐心。他们往往更能把事情做好**"
- "**don't rush to conclusion，不要匆忙下结论。不要太容易说'说白了就是这样'**"
- "**竞争对手是很好的蓝军**" / "**哪怕是竞争对手推动的批评文章，我们也认真看……也许 80% 都有问题，但 20% 是对我们有启发的，那我们应该吸收那 20%**"
- "**随便说 All In 的公司有很大问题**" / "**想要 All in 本质上就是'我不想了，就这样赌一把'**"
- "**大部分事情，都是有原因、有道理的，没有特别难以解释或者特别不寻常的部分**"
- 应对失误的"四步曲" [一手]：**realize it、correct it、learn from it、forgive it**

**演讲中讲的故事 [一手]**：

1. **微软 vs Google**：微软长时间把打败 Google 当目标狂砸搜索，"**直到几年之后才意识到，真正影响核心业务的是新兴的云计算，对手是亚马逊**"。寓意：紧盯错误对手的危险。

2. **Alex Honnold（《徒手攀岩》纪录片）**：张一鸣在加州见过他本人。引用他的状态作为"平常心"具象化。

3. **个人例子**：晚上计划工作但被西瓜（字节内部论坛）有意思的内容吸引，看了很久，睡前懊恼。— 用来讲"自我接纳"。

4. **跑步**：尽可能放松，不理会酸痛带来的干扰，很容易能跑 3-4 公里。— 平常心 = 不抗拒不强迫。

5. **早期短视频用户留存**：城市用户留存差，一同事猜测是"城市白领脑力劳动多更倾向图文"。张一鸣用这个案例反驳"先入预设"。

**引用文献**：
- 《The Power of Now》(《当下的力量》, Eckhart Tolle) — "**在'开言英语'上听到的**"，作为平常心的概念来源
- 《徒手攀岩》(Free Solo) 纪录片

**矛盾/张力**：这次演讲已经能感觉到张一鸣对 CEO 角色的疲惫 — 整篇通篇在劝大家"放松""不要紧绷""不要 All In"，这与他一贯"大力出奇迹"的风格形成反差。两个月后他就宣布卸任。可以解读为他在为自己的退出做铺垫，但也可以解读为他真的开始相信"平常心"是更高境界。这个矛盾不要调和。

### 1.4 卸任 CEO 内部信（2021-05-20）

来源：36氪原文 https://36kr.com/p/1232165369369221 （[一手] 36氪是字节官方公开渠道之一，此次内部信由字节同步给媒体公开）。

**核心原话 [一手]**：
- "**我感觉过去几年很大程度都在'吃老本'，比如，在 17 年之前我还能保持关注机器学习技术的新进展，近三年已经没有太多学习了**"
- "**当业务和组织变复杂规模变大的时候，作为中心节点的 CEO 容易陷入被动**"
- 自我评价对"过去三年的 OKR 执行都挺不满意"
- 未来要研究的方向：**虚拟现实、生命科学、科学计算、教育公益、脑疾病、古籍数字化整理**
- 给自己 10 年时间："**以十年为期，为公司创造更多可能**"

**这封信是理解张一鸣的关键文档**。它揭示：
1. 他真正的智识焦虑是"跟不上技术前沿"，不是商业焦虑。
2. 他认为 CEO 角色阻碍了他的学习。
3. 他对自己的标准是"机器学习论文阅读频率"。

### 1.5 7 周年之前的周年演讲（5/6 周年）

**5 周年（2017）**：仅可推断主题为 "Context, not Control"（详见 1.1）。**没有完整全文流通。**

**6 周年（2018）**：用户任务中提到的"6 周年演讲 2018《Context, not Control》"在调研中**未找到独立的 6 周年演讲全文**。最接近的是 2018 年码荟年会（4-5 月）那场演讲（Source Code Capital 主办），主题是"外部视角与反'向上管理'：如何避免企业的负规模效应"，张一鸣自己说"去年讲了 Context not Control，今年讲负规模效应"。所以 6 周年与 2018 年码荟可能是不同的两场，但 6 周年原文调研中未找到。**结论：仅有二手转述，无 6 周年独立全文。**

### 1.6 2018 码荟年会演讲：《外部视角与反"向上管理"》

来源：Source Code Capital 官网 https://sourcecodecap.com/code-class-post/zijietiaodongzhangyiming-waibushijiaoyufanxiangshangguanliruhebimianqiyedefuguimoxiao/ （[一手] 主办方发布，可信度高）

**核心论点 [一手]**：
- "**企业越大，能力越强。那反过来想想，企业是不是也有负规模效应？其实有的。而且企业越大，负规模效应也可能越大**"
- "**外界比你更了解你的公司**" / "**投资人一个月的 DD 调查有时比 CEO 更了解公司**"
- 三大反规模症状：**信息失焦、向上管理、ego 变大**

**讲的故事 [一手]**：
- 诺基亚 CEO 2007 年评 iPhone："**苹果不会对诺基亚造成任何影响**" → 3 年后衰落
- 亚马逊贝索斯重视用户邮件反馈
- 任正非主张保留有效竞争对手防止傲慢
- 王兴提出"**ego 缩小到原子那么小**"（这个说法张一鸣后来反复用）

**汇报 PPT 反模式**：
- 多页无逻辑，用数据堆砌
- 频繁改换统计口径（DAU→MAU→收入，天→周切换）
- 报喜不报忧

这场演讲是理解张一鸣"外部视角"思维的关键源头。

### 2.3 早期工作经历的反思 [一手] 微博

- "我去年自己有一个感想：觉得之前的老板都对我不太苛刻"（暗指饭否、九九房经历）
- "在这样没人干活的公司你都做不出点成绩来，你还能干什么?"
- "工作基本上是一个积累信誉的过程"
- "如果给过去 5 年的自己一个建议，就是激进再激进一点"

这条最后的"激进再激进一点"很有意思 — 与他后来推崇的"平常心""延迟满足"形成微妙张力。可能解释：早期他对自己不够进取的反省，转化成了对字节"大力出奇迹"的推崇。

## 4. 重复出现 ≥ 3 次的母题故事（这些才是真信念）

把不同时间点的演讲、采访横向比对，**反复讲的故事**比金句更能揭示张一鸣的真正信念：

1. **"我们当年没有算法基因，自己学着写第一版推荐引擎"** — 出现于至少 3 次：7 周年演讲、9 周年演讲、2016 年 Bootcamp 演讲、CSDN 2013 采访
   - 真信念：**能力可学，不被既有边界约束** = 成长心态
2. **"地铁上越来越多人用智能手机，我决定做今日头条"** — 出现于至少 4 次
   - 真信念：**与趋势站在一起 / 第一性原理观察用户**
3. **"南开四个标准（综合、靠海、不近、下雪）"** — 多次（南开演讲、各采访）
   - 真信念：**独立思考，不从众**（叙事自我塑造）
4. **"修电脑认识梁汝波"** — 多次
   - 真信念：**伙伴的重要性 + 兴趣驱动**
5. **"和王兴在饭否，2009 年被关停"** — 多次
   - 真信念：**经历过失败，知道边界在哪里**（很少展开）
6. **"周鸿祎建议我两年别出去说"** — 至少 3 次
   - 真信念：**默默做事，沉默是力量**（与张一鸣后期减少公开发言一致）
7. **"诺基亚 CEO 评价 iPhone"** — 多次
   - 真信念：**反 ego、警惕成功者认知偏见**

## 5. 核心矛盾与张力（不调和，直接记录）

调研中发现以下**张一鸣自我陈述之间的矛盾**：

1. **"大力出奇迹" vs "平常心" / "不要 All In"**
   - 7 周年（2019）赞颂大力出奇迹；9 周年（2021）批评 All In 是偷懒。
   - 中间发生了什么：TikTok 美国危机（2020）、抖音商业化、与拼多多/快手对垒。
   - 可能解读：成功后对早期方法论的修正。或者：对外说"平常心"，对内仍要"大力"。

2. **"激进再激进一点" vs "延迟满足感"**
   - 早期微博自我反省"应该激进点"，但同时反复推延迟满足。
   - 可能解读：延迟满足 ≠ 不激进，而是在长时间尺度上激进。

3. **"独立思考穿越喧嚣" vs "Context, not Control 的群智涌现"**
   - 一边强调个人独立思考，一边强调集体智慧。
   - 可能解读：决策做法 vs 信息流通。但实操中往往混淆。

4. **2021 卸任信"过去几年都在吃老本" vs 2019 年高调"大力出奇迹"**
   - 同一个人，两年间叙事反转。

5. **"务实"vs"浪漫"框架本身的张力**
   - 张一鸣极力强调"务实的浪漫"是一体的，但具体决策中（多闪、抖音 vs 微信、TikTok 美国）几乎全是务实驱动，"浪漫"修辞用于事后正当化。
