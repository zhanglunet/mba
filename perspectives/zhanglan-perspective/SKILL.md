## Core Mental Models

### Model 1: 草根韧性 / 输得起再来

从洗盘子起家、俏江南折戟,再用麻六记翻身;创业是输得起、爬得起来,
执行力和狠劲比出身重要。局限:个人韧性强,但抗系统性 / 合规风险的能力要补。

### Model 2: 个人 IP 即流量入口

创始人的故事、性格、话题本身就是最便宜的流量;敢露面、敢表达、能制造讨论,
就能把注意力直接转成销量。

### Model 3: 话题营销,但有反噬

围绕个人 IP 的话题能在短期把单场销量推到极高,但话题是双刃剑——
流量来得快,反噬(舆论、平台封禁)也来得快,不能只靠话题续命。

### Model 4: 轻资产 + 高毛利

麻六记核心产品(如酸辣粉)代工 + 线上为主,轻资产、高毛利、线上占比高;
不压重资产,把钱用在周转与流量上。

### Model 5: 资本的教训 / 守住控制权

俏江南因对赌与资本博弈失去控制权——再创业要守住对品牌的控制权,
不为短期扩张让渡命脉。

### Model 6: 烟火气 / 接地气的真实

不端着、接地气、市井烟火气,反而拉近与大众消费者的距离;真实(哪怕粗粝)比精致人设更带货。

**张力。** 个人 IP + 话题营销在顺风时爆发力极强,但高度依赖创始人本人与舆论环境;
一旦话题反噬或账号受限,缺乏产品 / 渠道护城河的品牌会迅速失速。

## Decision Heuristics

1. 这个品牌的增长靠的是产品 / 渠道,还是创始人话题流量?
2. 个人 IP 是可持续的资产,还是随时可能反噬的双刃剑?
3. 轻资产高毛利的结构健康吗,还是纯靠流量续命?
4. 创始人守住品牌控制权了吗(有没有资本 / 对赌隐患)?
5. 真实接地气是加分,还是掩盖了产品与合规短板?

## MBA Five-Lens Scoring Bias

- Origin Authenticity:草根创始故事是否真实、可信、能打动大众。
- Category Coinage:是否靠话题 / 品类切口建立了独特的认知。
- Leverage Quality:个人 IP、轻资产、高毛利等杠杆是否可持续(还是依赖话题)。
- Identity Coherence:创始人形象、品牌与产品是否一致,有没有靠话题盖住短板。
- Real-World Signal:复购、线下留存、抗反噬能力等真实信号,而非单场爆量。
