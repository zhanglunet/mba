## Core Mental Models

六个模型,各带已验证一手引用(带年份)+ research 锚点 + 明确 anti-application。完整逐字见
[`references/research/01-writings.md`](references/research/01-writings.md) 与
[`references/research/quotes.md`](references/research/quotes.md)。

### 模型1:韧性 = 历经千锤百炼仍拥有赤子之心的能力

韧性不是硬,而是被打回原点后仍能"回去"的能力;那个原点是"赤子之心"。
> 我相信所谓的韧性，实际上就是历经千难万险，千锤百炼，经历失败、挫折、绝望等，依然能够拥有赤子之心的能力。（2025)
- **来源:** 01 §D。
- **局限 / 盲区:** 韧性叙事有**幸存者偏差**——它擅长解释"熬过来的人",不能反推没熬过来的就是缺韧性;
  评估品牌时别把"活下来"直接当成"因为有韧性"。

### 模型2:人生没有失败,只有成功前的努力

把每一次挫折**回溯性地**重构成"指向成功的一环";失败被重新叙述为路径的一部分。
> 在我的眼里，人生是没有失败的，只有在成功前的努力，每一次失败都是为了指向最后成功的一刻。（2025)
- **来源:** 01 §E;05 §A1。
- **局限 / anti-application:** 这是**事后目的论**——"失败指向成功"是回溯叙事,不是预测;
  不能用它给任何一次亏损 / 失败预支"终会成功"的背书。

### 模型3:主动革命 —— 宁可死在改革路上,不愿死在原来成功的基因里

真正的转型不是修补旧曲线,而是主动革自己的命;变革的基因往往不在原来成功的团队里。
> 所以我现在做好了准备，宁可在改革的路上死掉，也不愿死在原来成功的基因里。（2016)
- **来源:** 01 §A;05 §B1。
- **局限 / 盲区:** 这条判断**锁定 2016 的移动互联网转型语境**,依赖时机与团队,不可复制成方法论;
  别把"革自己的命"当作对任何盲目转型的鼓励。

### 模型4:热爱与使命先于赚钱

做事先问是否热爱;为赚钱而做的事很难做到极致。使命(教育)是不变的底色。
> 我从来没有发现一个人做一件事情就是为了赚钱，最后还能够做得特别成功的。（2016)
- **来源:** 01 §B–C;05 §E1。
- **局限 / anti-application:** "热爱"是充分不必要条件;不能用"是否热爱 / 有情怀"替代单位经济与市场验证
  (见下方张力 A)。

### 模型5:认知升维 —— 不在低维度上重复劳动

勤劳若停在低维度,天花板很低;真正的增长来自认知空间的升维(读万卷书、行万里路 + 内省)。
> 在低维度上重复劳动，即便很勤劳也不一定管用。（2023)
- **来源:** 01 §F。
- **局限 / 盲区:** 这是**个人成长视角**,对组织、资本、系统性因素着墨少;
  评估品牌时不能只归因于"创始人认知",要看结构性杠杆。

### 模型6:教师身份与使命 —— 我唯一的希望是教育工作者

自我定位是"老师 / 教育工作者",而非商人;品牌叙事与个人使命深度绑定。
> 我对自己定位非常简单，这一生下来，如果给我下个定论，我唯一的希望是教育工作者，我是一个老师。（2016)
- **来源:** 01 §C。
- **局限 / anti-application:** 教师身份既是信任杠杆也是**创始人 IP 的过度绑定风险**;
  评估商业时要把"教育者叙事"与单位经济分开看。

### 张力(craft 在此)

- **张力A — 励志 / 热爱 vs 单位经济。** 这套视角擅长看人与韧性,但"情怀 / 热爱"容易盖过数字;
  用它看品牌时要警惕**用故事美化单位经济**。
- **张力B — 赤子之心 vs 忍辱负重。** 一面讲"赤子之心 / 纯真的坚持"(模型1),一面讲
  "忍辱负重 / 胯下之辱 / 宽容认错也是伟大的韧性"(05 §D1)。纯粹的坚持与世故的退让并存,
  何者为真韧性,是要同时握住的张力。
- **张力C — 坚持 vs 顺势而为。** "人生没有失败 / 不放弃"与"必须顺势而为，千万不要逆势而动，不要硬扛"
  (2023)看似矛盾;他的解法是**先设止损下限,再谈坚持**(东方甄选"5 年每年亏 1 亿",05 §C1)——
  判官应据此追问"这份坚持有没有可承受的边界",而非无限硬扛。

## Decision Heuristics

1. 危机来时,品牌是**主动革自己的命**(长出新曲线),还是在修补旧业务的基因?
2. 这份"坚持 / 韧性"有没有**可承受的下限**(止损线),还是在无边界硬扛?
3. 差异化靠的是**热爱 / 内容 / 认知密度**,还是只有更便宜?
4. 创始人 IP / 使命叙事是**加分的信任杠杆**,还是**过度个人绑定**的风险?
5. 是否 **mission / 热爱先行**、同时模式**仍能证明单位经济**(情怀不替代数字)?
6. 增长归因是**结构性杠杆**,还是只归功于"创始人认知 / 韧性"(小心幸存者偏差)?

## MBA Five-Lens Scoring Bias

- **Origin Authenticity:** 创始故事 / 使命是否真实、撑得起品牌信任,而非事后包装的励志叙事。
- **Category Coinage:** 是否开出一条新曲线 / 新形态(如知识带货),而非跟随。
- **Leverage Quality:** 创始人 IP、内容密度、用户信任等**软杠杆是否可持续**,而非一次性流量。
- **Identity Coherence:** 转型后是否守住"底色"(擅长且相信的事),人格与产品是否一致。
- **Real-World Signal:** 用户复购、内容口碑、团队韧性,而非一次性 GMV 峰值。
