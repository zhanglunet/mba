## 七、立场演化时间线（2010 → 2026）

| 时间 | 立场关键词 | 变化方向 |
|---|---|---|
| 2010-2011 | 3Q 大战、扣扣保镖、红衣大炮 | 对抗、争议、流量 |
| 2011-2014 | 纽交所上市、免费杀毒打到瑞星江民出局 | 颠覆者叙事固化 |
| 2014-2016 | 360 手机、奇酷、酷派 | 多线扩张、走捷径 |
| 2015-2016 | 私有化、回 A 准备 | 退出美股 |
| 2017-2018 | 江南嘉捷借壳、A 股市值 4400 亿 | 国家叙事起步 |
| 2018-2019 | 安全大脑、网络战、敌已在我 | 政企转向 |
| 2020-2022 | 数字安全、看不见就是最大痛点 | 概念扩张 |
| 2022-2023 | 360 智脑、360 鸿图、大模型新工业革命 | AI 全面押注 |
| 2024 初 | 16 年和傅盛和解 / 与马化腾握手 / 卖迈巴赫 | 性格转变 |
| 2024 中 | 抖音 1000 万粉丝 / 哪吒直播 / 红衣大叔 | IP 转型 |
| 2024 末 | 安全大模型 / 把"免费"贯彻到底 | 经典策略复用 |
| 2025 | 干掉市场部 / 360 纳米 / 智能体黑客 | AI 替代式管理 |
| 2025-08 | ALL IN AGENT | 第三次"全部押注" |
| 2026 初 | AI 智能体安全主场域 | 仍在演变 |

跨度观察: 周的核心安全观 (攻防对抗、免费策略、看不见的攻击、国家级威胁) 从 2010 到 2026 几乎完全一致；变的是产品形态 (杀毒 → 安全大脑 → 数字安全 → 安全大模型 → 安全智能体) 和品牌人设 (大炮 → 教主 → 大叔)。
