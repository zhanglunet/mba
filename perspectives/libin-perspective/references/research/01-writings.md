## Recurring Thesis Layer

### 1. 用户社区是核心资产,不是营销结果

李斌公开 voice 的最稳定特征:把 NIO 车主 / NIO House 用户活动 / 用户口碑当作
**资产负债表**项目,而不是 PR 输出。NIO Day 由用户志愿者参与策划,NIO House
本质是用户线下空间。他在多次演讲里反复说"用户口碑是蔚来活下来的原因"。

判断镜头含义:

- **Origin Authenticity**:看品牌讲故事时,用户是不是核心叙事人 —— 真用户
  上台讲故事,还是 PR 拍的用户视频。
- **Real-World Signal**:看用户复购 / 推荐 / 自发传播率,不是单看销量。
- **Identity Coherence**:用户社区不是营销活动,是组织架构里的部门(用户
  发展 / 用户运营 / 用户服务三条线分立)。

证据锚点:

- NIO Day 历届:用户志愿者参与;用户上台讲自己的故事。
- 2019 现金流危机时,大量用户自发为 NIO 站台 / 推荐购车,被反复在内部信引用。
- 2020 合肥救助签约后,内部信里把"用户没放弃我们"列为决策依据。

### 2. 换电不是技术理想,是用户体验

李斌为换电体系的辩护一直围绕用户体验:**车主不应该被充电焦虑、电池衰减、电池
升级路线绑架**。他反对"换电是落后路线"的批评,论证是 —— 在高端用车场景里,
3 分钟换电 + 电池 BaaS 模式让用户拥有更高的灵活性,这是产品决策不是技术
路线决策。

判断镜头含义:

- 看品牌的"基础设施 / 用户体验"层做得多深 —— 是不是真把基建当成产品的一
  部分。
- 看品牌的"长期成本"是不是 transparent 给用户(电池租赁 vs 电池买断的清楚
  机会)。

证据锚点:

- NIO Power Day 多次主题演讲。
- 2024 换电联盟扩张(中石化、长安合作)的公开讲话。
- 2023+ 多次回应"换电过时"质疑。

### 3. 高端服务网络比硬件更难复制

李斌的核心战略判断:汽车品牌的护城河在售后服务网络 + 用户体验长链,不在硬件
本身。NIO 上海中心 / NIO House / NIO Service / 救援电话 / 一键加电 / 上门
取送车 ——他坚持这套全栈服务是高端品牌的真正壁垒。

判断镜头含义:

- 看品牌的高端化是不是停留在"产品 + 营销 + 价格"三件套,还是真做了服务网络
  深耕。
- 看服务网络的投入是否同步于销量扩张(销量翻倍但服务点没翻倍 = 透支)。

证据锚点:

- NIO Day 演讲反复出现"服务是品牌的本体"。
- 财报会上,服务网络的 capex / opex 单列,公开承认这是负担也是壁垒。
- 2022-2024 多次为"服务网络拖累毛利"辩护。

### 4. 公司可以慢,但服务网络不能断

蔚来在销量上不追逐"快增长",在服务网络上反而不允许打折。李斌在 2019 危机
后形成的判断:"如果我们让现有用户失望,品牌就死了" —— 资本市场宁可短期不
开心,服务承诺不能压缩。

判断镜头含义:

- 看品牌在资本紧绷期是先砍服务还是先砍发布会 / 营销 / 海外。前者降级品牌
  价值,后者只是节奏调整。
- 看品牌对老用户的承诺 (软件升级 / OTA / 换车权益 / 旧车残值) 有没有兑现。

证据锚点:

- 2019-2020 危机期间的全员信:服务承诺不打折。
- 2024 多品牌矩阵推出时,反复保证 NIO 主品牌服务体系不变。

### 5. 多品牌矩阵覆盖全价格带

2022-2024 蔚来推出乐道(中端) + 萤火虫(入门),完成 30 万 + / 15-25 万 /
10-15 万的全价格带覆盖。李斌的论证是:**不同价格带是不同人群 + 不同服务标准**,
合在一起才是规模 + 利润 + 品牌矩阵的完整盘。

判断镜头含义:

- 看多品牌策略是不是带不同服务标准的真正分层,还是同一团队同一服务的换名营销。
- 看子品牌之间的内部竞争是否有制度防火墙(独立团队 / 独立销售 / 独立服务)。

证据锚点:

- 2024 乐道上市 + 萤火虫上市发布会。
- 多次内部信讲多品牌组织架构(独立一号位 / 独立 P&L)。

### 6. 创始人 IP 不靠激进言论,靠服务承诺

李斌不是雷军式的"全程发布会金句"型 IP,也不是李想式的"微博直接判断"型,而是
**长篇 + 用户社区 + 长期出席**型 IP —— 出席 NIO Day、月度直播、用户聚会,
不追求短视频热点。他公开表达过:"我不追求每天一个金句,我追求每年一次跟车主
当面把话讲完"。

判断镜头含义:

- 看创始人 IP 是不是稳定 / 长期 / 跟用户社区绑定,还是短视频流量驱动。
- 看 IP 是建立在"承诺兑现"上,还是"反共识断言"上。

证据锚点:

- 历届 NIO Day 主持人风格(用户故事 → 产品 → 承诺,而不是反共识开炮)。
- 2019 危机期间,反复公开承认错误,不甩锅,不反共识。
- 2024 多品牌矩阵发布时,语气依然温和,不靠攻击对手获得声量。
