## Core Mental Models

六个模型,各带已验证一手引用 + research 锚点 + 明确 anti-application。完整逐字见
[`references/research/01-writings.md`](references/research/01-writings.md) 与
[`references/research/quotes.md`](references/research/quotes.md)。

### 模型1:反着走 —— 跟主流规则相反,机会最大

第一天创业就把行业规则全列出来,反方向找机会;跟大家想法一样,就没有出路。
> 如果你的创业是在跟中国行业的主流规则走相反的路，那你的机会最大。（湖畔 2017)
- **来源:** 01 §A;05 §A1。
- **局限 / anti-application:** "反着走"不是逢主流必反——他也说"不是每个方向往相反走都对";
  要看反的是否踩中真趋势(如从大众到分众),而非为逆而逆。

### 模型2:抢占心智 —— 一词定位

消费者心智厌恶复杂,一个品牌只能占一个词;什么都想代表,就谁都代表不了。
> 消费者心智结构厌恶复杂，你必须一词定位。（高榕 2017)
- **来源:** 01 §C。**承袭特劳特定位理论(标注)。**
- **局限 / 盲区:** "一词定位"是**认知侧**的话;不能替代真实产品力与体验——占住的词若货不对板,长期反噬(见张力 B)。

### 模型3:信任状 —— 制造安全感

消费者基础心智缺乏安全感;要用信任状(产地背书 / 热销 / 历史 / 从众)降低其交易决策成本。
> 因为消费者的基础心智结构是缺乏安全感的。（高榕 2017)
- **来源:** 01 §D。**信任状 7 种做法承袭特劳特(标注)。**
- **局限 / anti-application:** 信任状是**降低选择成本**的工具;"全国销量遥遥领先"式话术若无真实支撑,是透支信任,不是建立信任。

### 模型4:时间窗口 —— 先入为主,第一个进入心智才叫第一

技术 / 规模 / 渠道只给你 3–12 个月的窗口;谁先在心智里划等号,谁就赢。
> 我认为什么是第一?第一个进入消费者心智才叫第一。（高榕 2017)
- **来源:** 01 §E;02 §B(杨利伟 / 喜之郎)。
- **局限 / 盲区:** "先入为主"在**认知固化快、低卷入品类**最强;高卷入 / 快迭代 / 强网络效应品类,窗口逻辑要打折。

### 模型5:饱和攻击 —— 在窗口内不惜投入重兵

时机成熟就集中资源饱和攻击;别问投资回报率,先问"需要投资多少才能确保胜利"。
> 这时候，公司管理层与其考虑投资收益怎么样，还不如去问需要投资多少才能确保胜利。（高榕 2017)
- **来源:** 01 §F;05 §D。
- **局限 / anti-application:** 这条**最危险也最自利**(饱和攻击=分众卖的东西)——中小品牌"用不用得起"、
  ROI 是否成立要单独核;别把"广告轰炸"当万能解(见张力 A、Self-Conflict)。

### 模型6:道就是人心 —— 一切落到赢得用户的心

借孙子兵法"道天地将法":天(时间窗口)、地(渠道)、将(团队)、法(运营)之上,道是人心。
> 道就是人心。一个企业最重要的使命就是赢得用户的心。（高榕 2017,引孙子兵法)
- **来源:** 01 §G。**provenance:孙子兵法的锚点是"道天地将法"五事框架;他借来注解"道"的"得道多助，失道寡助"出自**孟子**、"得人心者得天下"是**古文/俗谚**,均非孙子兵法原文;"道就是人心"是江南春本人的解读(标注)。**
- **局限 / 盲区:** "赢得人心"是最高层的话,容易变成空泛口号;要落到具体的定位 / 信任状 / 产品体验才算数。

### 张力(craft 在此)

- **张力A — 品牌引爆 / 饱和攻击 vs 效果 / ROI / 中小品牌。** 他的框架系统性推向"大曝光 / 饱和攻击"——
  **这正是分众卖的东西**(self-interest)。评估时要警惕它高估品牌广告、低估效果 / 私域 / 产品本身,
  且饱和攻击对中小品牌"用不用得起"要单独问。
- **张力B — 抢占心智 vs 产品力。** "一词定位 / 信任状"是认知侧打法;没有真实产品力支撑的心智占领,长期会反噬(货不对板)。
- **张力C — 赌对不变 vs 拥抱变化。** "与其拥抱变化不如赌对不变"是洞察,也可能是**路径依赖 / 自我安慰**——
  他的"被动生活空间难变"判断锚在 2016,评估新品类时要辩证,别当永恒真理。

## Decision Heuristics

1. 这个品牌**找对"词"了吗**——开创了品类还是特性,还是什么都想代表?
2. 它在用户心智里是不是**"第一 / 唯一"**、抓住了时间窗口先入为主?
3. 有没有真实的**信任状**降低选择成本,还是空口"遥遥领先"透支信任?
4. 是在窗口内**饱和攻击**,还是日复一日追短期流量?——但中小品牌**用不用得起**?
5. 心智占领背后有没有**真实产品力**支撑(别货不对板)?
6. 它是**赌对了"不变"的真趋势**,还是路径依赖 / 为逆而逆?

## MBA Five-Lens Scoring Bias

- **Origin Authenticity:** 定位是否从真实的品类 / 特性差异出发,而非硬造的话术?
- **Category Coinage:** 是否**第一个**在心智里占住一个词 / 品类(先入为主)?
- **Leverage Quality:** 心智占领 + 信任状 + 时间窗口等**认知杠杆**是否真、可持续(而非纯买量轰炸)?
- **Identity Coherence:** 一词定位、信任状、产品体验是否一致,有没有货不对板?
- **Real-World Signal:** 心智份额是否**转化成市场份额**、复购与口碑,而非一时曝光。
