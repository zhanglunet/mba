## 赛手第一 / 投人(反"只投赛道"误区)

- 这其实是一个巨大的误区。我们是既要关注赛道也要关注赛手，而且赛手应该放在第一位。 `[陆]`（明确反驳"红杉只投赛道")
- 再好的赛道，如果赛手不行，即便有过短暂的领先，也很难长期持续。 `[陆]`
- 用我们行业的话来说叫"盲注"。 `[陆]`（投一个还没定案的创业者)
- 你认可他的价值观、能力、以前的业绩，不那么在乎他具体走上哪一条细分的赛道。 `[陆]`
- 成功的、伟大的公司背后一定有一个鲜活的优秀创业者的面孔。 `[陆]`

## 看公司三点标准(15 年不变)

- 这个标准，我们15年来没有改变。 `[陆]`（说起来三点,做起来每点都很难判断)
- 成功的企业都非常有"企图心"，但关键是它得在一个"好"的行业里，只有天空才是它的限制。 `[陆]`
- 应该关注那些具有巨大发展空间的行业。 `[陆]`
- 有些商业模式天然地容易形成竞争壁垒 `[陆]`（第三点:商业模式)

## 长期主义 = 抵御诱惑 + 理智诚实 + 反人性

- 长期主义说起来很容易，但是做起来很难。 `[陆]`
- 长期主义的另一面是你能抵御短期的诱惑，而人性往往让你去寻找短期的利益。 `[陆]`
- 有所为，有所不为，这对投资人和企业家都是很重要的选择。 `[陆]`
- 投资过程有一些反人性的东西需要你去克服。 `[陆]`
- 要勇于承认错误，以清零的心态来了解企业。 `[陆]`
- 我们希望企业家是长期主义，这样投资人实践长期主义才有可能。 `[陆]`
- 我们并不认为要通过砸钱的方法去"支持"公司，而应该变成市场上聪明的"钱" `[陆]`

## 创业者背后的创业者 / 好奇心 / 司机旁看地图

- 我感觉好奇心是一个非常重要的驱动因子。 `[陆]`
- 参与创造的这个过程是这份工作的最大乐趣。 `[陆]`
- 红杉1972年创立时的定位是"创业者背后的创业者" `[秦]`（红杉定位,标注)
- 我认为我们永远是在司机旁边帮他看地图的人，创始人永远是驾驶员 `[秦]`
- 我们需要一个好司机，同时要帮他看好路。 `[秦]`
- 我曾经是一个互联网领域的创业者，今天也还是一个创业者，是投资领域的创业者。 `[秦]`

## 优秀创业者的共通性

- 首先，他们都极其专注、市场化驱动 `[陆]`
- 但首先是把一个产品做到极致。 `[陆]`
- 你有多大的抱负，决定了你能成就多大的事业。 `[陆]`
- 他们将尊重商业规则放至首位 `[陆]`（同时又重塑了行业规则)
- 非常成功的企业家都有国际视野。 `[陆]`
- 首先是要尊重商业规则，这是我们投资创业者最基本的底线，不能想着在哪打擦边球。 `[陆]`

## 做多中国 / 好奇心 / 选择

- 凡是成功地做出很大事业的中国企业家，他们身上都有一个共同的词叫"long China"。 `[秦]`（做多中国)
- 像我这样年纪的投资人，今天必须还要有很强的好奇心去了解崭新的东西，否则就会被时代抛弃。 `[秦]`
- 人生其实就是不断的选择 `[秦]`
- 有时候在你理性的思考之外，必须有那么一点冲动，一点激情 `[秦]`（判断来自"第六感")
- 成功不仅在于投了什么，也在于不投什么 `[秦]`（有所为有所不为)

## 全站式投资 / 本地化 / 知识沉淀

- 是一个全站式的投资 `[陆]`（种子/风投/成长/并购/一级半/二级)
- 赢家通吃 `[陆]`（新经济尤其互联网的特征)
- 我们相信每个地方都应该是一个最优秀的团队，有自己的决策机制 `[陆]`（本地化决策)
- 全球平台，全球知识，本地执行，本地决策 `[秦]`
- 这些累积的知识我认为是最重要的资产。 `[秦]`（知识沉淀 / SMS 知识平台)
- 我们主要投四个行业 `[秦]`（科技传媒/消费服务/医疗健康/新能源清洁技术)
- 拿什么来比喻我们整个团队的协同模式呢？我感觉就像是一个交响乐团 `[陆]`（One Team)

## 犯错 / 认知边界(self-conflict:均为 portfolio)

- 像字节跳动的投资就是这样。 `[陆]`（一度理解不够,九个月后主动给张一鸣打电话补投——near-miss 已纠)

- 我们总会犯错误，这在风险投资中不可避免 `[秦]`
