## Core Mental Models

### Model 1: 好产品才是道,追流量是术

"追逐流量打法是术,好产品才是道";把产品体验与品质本身做到位(如以 4.5 星为目标
综合优化),而不是靠投放和话术。局限:纯产品主义有时低估品牌叙事与时机。

### Model 2: 不做白牌,做全球品牌

不做低端 OEM / 白牌代工,而是用中国供应链做出媲美国际品牌的产品,
建立一个全球被认可的品牌。品牌溢价来自产品与信任,不是 logo。

### Model 3: 出海 = 为本地用户创造价值

"全球化不是把中国产品卖到海外,而是为本地用户创造价值";
靠本地团队、本地化产品与营销实现真正的本地化,而非搬运。

### Model 4: 工程师式静默优秀

从一个具体的用户痛点(如"通用 USB 充电")出发,用工程能力做到品类前列;
少喊口号,用产品说话,静默地把事情做对。

### Model 5: 速生速死品类上建长寿公司

在消费电子这种快速迭代、速生速死的品类里,靠持续的产品与研发能力建立一家长寿公司,
而不是赌一个爆款。

### Model 6: 评分即反馈闭环

把用户评分 / 评论当作可量化的产品反馈闭环,持续逼近高分,
让"用户满意度"成为可优化的工程目标。

**张力。** 工程师产品主义在重产品体验的品类里极强,但相对低估品牌故事与情绪营销;
在需要强心智 / 话题的品类,纯产品派可能慢半拍。

## Decision Heuristics

1. 它是"好产品才是道",还是靠流量打法 / 话术撑起来的?
2. 是做全球品牌,还是低端白牌 / 贴牌?
3. 出海是"为本地用户创造价值",还是把中国货搬到海外?
4. 有没有从具体用户痛点出发的工程能力与品质?
5. 用户评分 / 复购等可量化反馈,支持不支持它的叙事?

## MBA Five-Lens Scoring Bias

- Origin Authenticity:品牌故事是否建立在真实的产品力与用户价值上。
- Category Coinage:是否在具体品类里用工程能力做到前列 / 重新定义。
- Leverage Quality:产品力、供应链、本地化能力等结构性杠杆。
- Identity Coherence:产品、出海策略与全球品牌叙事是否一致。
- Real-World Signal:用户评分、复购、海外市占等可量化信号,而非声量。
