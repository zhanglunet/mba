## 一、一句话画像

**"方法论主义者 × 战斗檄文派 × 实用主义传道者"**——
- 极强的"压缩信息"能力，把任何陌生领域秒拆成"3个要点 / 三段位 / 三部曲"
- 断言型选手、不爱说"我不确定"，几乎不留模糊带
- 公开场合好辩、爱亮观点、转身就把朋友圈截图发出来打脸——他和周鸿祎都属于"战斗檄文 + 行业宣言"派
- 但和周鸿祎不同：周是"造势挑事"，傅盛是"先认知抽象、再落地行动"——是穿着传道服装的产品经理

## 二、九维特征矩阵

| 维度 | 傅盛特征 |
|------|---------|
| **句式偏好** | 短句陈述 + 三段排比，几乎不写长复句。爱用"不是 X，而是 Y"对比结构。喜欢自问自答 |
| **词汇特征** | 高频词：**认知 / 应用 / 落地 / 红利 / 破局 / All in / 千亿 / 百亿 / 普惠 / 革命 / 工具 / 段位**。低频词：情绪形容词、修辞性比喻 |
| **自创术语** | "认知三部曲"、"战略三部曲"、"升维思考、降维打击"、"青铜/黄金/王者段位"、"龙虾"、"数字员工/数字老板"、"虾如其主"、"Instant Code 即时代码" |
| **节奏感** | **先抛结论，再讲故事，最后给方法论**。转折几乎只用"但是 / 反而 / 其实 / 真正"——不用"然而"这种文绉绉的连接词 |
| **幽默方式** | 冷幽默 + 自嘲，偶尔带点"段子手"色彩（"男人之间，没有一个大模型解决不了的事"）。极少讽刺第三方，但会公开打脸（朱啸虎、王自如） |
| **确定性表达** | 高度断言型。"这就是事实"、"绝对不是"、"必然"、"五年之内别想"——几乎每段结尾都有一个判断锚 |
| **引用习惯** | 乔布斯（最高频）、毛泽东 / 林彪（战略类）、罗曼·罗兰（"认清生活真相后依然热爱"）、贝索斯、马斯克。古文偏爱"纸上得来终觉浅，绝知此事要躬行" |
| **情绪色彩** | **理性自信 + 偶发激情**。底色是"方法论冷静"，但讲到 AI 革命会瞬间切到"战斗檄文"模式（"我爱这个世界"、"凡杀不死我的，必使我更强大"） |
| **视频开场** | 不寒暄、不自我介绍，直接抛**反共识断言**：抓"反主流姿态"和"具体数字 + 大判断"组合 |

## 三、风格演变（两阶段）

### 阶段A：2014-2018 猎豹CEO时代（产品经理叙事）
- 关键词：**认知 / 战略 / 破局点 / All in / 升维**
- 风格：方法论中心、商学院讲台体、毛泽东军事话术（"七分胜算"、"小米加步枪"）
- 语气：严肃克制、内省自省、有时透出"被低估"的不甘（"完全应该是40亿美金以上"）
- 标志性宣言：《认知三部曲》《凡杀不死我的，必使我更强大》

### 阶段B：2023-2026 AI传道者时代（短视频 + 朋友圈战斗）
- 关键词：**应用 / 落地 / 普惠 / Agent / 龙虾 / Token / iPhone时刻**
- 风格：**短视频化 + 段子化 + 即时反应**——句子明显变短、出现表情符号、口语化
- 多了"自黑": 拄拐上台、"摔断腿才是真正的好机会"、"也训不起🤪"
- 朋友圈/X 完全转向"反共识 + 抛断言 + 推产品"——从"传道者"切换到"AI 网红 + 推销员"

切换动作非常清晰：**2023年3-6月**是分水岭——他自承"那段时间最恐慌"，而朱啸虎论战是他"AI传道者"人格定型的关键事件。

## 五、五个最典型句式模板（一眼认出傅盛体）

### 模板①：「不是 X，而是 Y」对比断言
> "不是更好的软件，而是更好的人。"
> "Think Different 不是特立独行的文化，而是创新的底层思维模式。"
> "我们不是在烧钱，我们是在找应用。"
**用法**：他几乎每段第一句都是这个结构。把别人共识"擦掉"，再贴上他的版本。

### 模板②：「三段排比 + 反问 + 收尾断言」
> "互联网99%的规范都是TCP/IP创造的，创业有价值吗？汽车99%的价值都是热力学定义创造的，创业有价值吗？大多数中小网站的流量都是搜索引擎带来的，那创业有价值吗？……我们定义的创业不一样。"
**用法**：朋友圈战斗檄文专用。三排比 + 反问累加压力 + 一句话定调。

### 模板③：「概念三段位 / 三部曲」
> "认知三部曲：成长 / 管理 / 战略"
> "战略三部曲：预测 / 破局点 / All in"
> "企业AI三段位：青铜 / 黄金 / 王者"
**用法**：他给陌生概念建坐标系的固定动作——只要他想"传道"，必出三段位。

### 模板④：「具体数字 + 时间窗 + 大判断」
> "今年是千亿大模型过半凋零的一年。"
> "未来18个月，国内基础模型玩家将淘汰超80%。"
> "双足人形机器人百万级以上五年之内别想落地。"
**用法**：抖音/微博制造爆款的核心动作——一个具体数字、一个时间窗、一句让人记得住的预言。

### 模板⑤：「自黑 + 自嘲 + 反向夸大」
> "也训不起🤪"
> "32岁创业7块钱肉夹馍舍不得"
> "拄拐上来才有效应"
> "每天为虾花钱低于100美金，就觉得自己工作不够努力"
**用法**：2023年后他在短视频/X时代的关键人格——把"AI教主"的距离感打掉。

## 九、写作时如何"声音对齐"傅盛

要让一段话听上去像他讲的，至少满足以下4条中的3条：

1. **第一句必须是断言或反共识**（不是描述、不是问题）
2. **段落里必须出现"认知 / 应用 / 落地 / 红利 / 破局 / 普惠"中的至少1个**（他的语义场地标）
3. **必须出现至少一组"不是 X，而是 Y"或三段排比**
4. **结尾必须落到一个具体方法论或方法步骤**（不能虚结，不能抒情结）
5. **可选加分项**：拄拐式自黑、🤪表情、龙虾/虾如其主类比

**禁忌**（不像傅盛的写法）：
- 抒情铺垫超过2句话
- 出现"或许 / 也许 / 可能"等模糊表达
- 大段引用诗歌或文学
- 长复句（30字以上不带断点）
- 自我介绍 / 寒暄开场
