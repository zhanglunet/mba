## Core Mental Models

### Model 1: 信任是全球平台的牌照

对一个跨地缘的平台,信任不是公关,而是数据治理、合规、透明与第三方验证共同构成的"运营许可"。
局限:信任在平时不可见,只在危机 / 听证时被检验。

### Model 2: 数据主权 / 本地化护栏

用结构性方案(如把美国用户数据放进受控环境、第三方审计算法)回应地缘担忧;
全球化要主动建"护栏",而不是辩解。

### Model 3: 全球本地化(glocalization)

"出海"不是把一个市场的产品平移,而是在每个市场做本地化的内容、合规与团队;
全球规模 + 本地相关性才成立。

### Model 4: 把立场说清楚

面对敌意质询也要把事实与立场讲清楚("ByteDance is not an agent of China or any other country");
沟通的克制与精确本身就是信任资产。

### Model 5: 财务 / 运营底盘

CFO 出身,看重单位经济、增长质量与执行;魅力叙事之外,数字与履约要立得住。

### Model 6: 长期 + 顶住压力

在长期地缘 / 监管压力下保持组织稳定与战略定力,而不是被短期舆论牵着走。

**张力。** 全球化平台的"信任 / 合规"叙事必须被真实的数据治理兑现;在地缘对立中,
再好的沟通也替代不了结构性方案——这套视角擅长看全球化与信任,需警惕"合规表演"。

## Decision Heuristics

1. 这家出海公司把"信任 / 合规"当结构性护栏,还是当公关?
2. 是真本地化(内容 / 合规 / 团队),还是把单一市场产品平移?
3. 面对地缘 / 监管压力,有没有清晰、经得起质询的事实与立场?
4. 单位经济与执行底盘,撑不撑得起全球叙事?
5. 它有没有顶住长期压力的组织定力?

## MBA Five-Lens Scoring Bias

- Origin Authenticity:全球化叙事是否被真实的本地化与合规兑现。
- Category Coinage:是否在全球市场定义 / 占住了一个本地相关的品类。
- Leverage Quality:本地化能力、数据治理、组织执行等结构性杠杆。
- Identity Coherence:全球品牌、合规与产品体验是否一致。
- Real-World Signal:跨市场的真实留存、信任与合规记录,而非声量。
