## Recurring Thesis Layer

### 1. 用户场景 > 车型规格

李想公开 voice 的最稳定特征:讨论一辆车的价值不从扭矩 / 风阻 / 续航数表开始,
而是从一个具体的家庭场景开始 —— 二胎、长途、安全座椅、奶瓶加热、副驾通话。规格
只是用来证明场景能成立的证据。

判断镜头含义:

- **Origin Authenticity**:看创始人讲品牌时是从产品规格 spec 开始,还是从一个
  具体用户故事开始。后者得分更高。
- **Identity Coherence**:车 / 营销 / 销售话术 / 服务承诺,是否围绕同一个场景闭环。

证据锚点:

- 理想 ONE / L9 / L8 / L7 发布会的"全家出行 / 移动的家"框架反复出现。
- 微博答复网友时,常用"这是为了帮你/帮你太太/帮你孩子解决 X 问题"的句式而不是
  跟同行 spec 对比。

### 2. 增程是答案,不是过渡

2019-2024 李想最反复辩护的产品技术决策:理想押增程而不是纯电的逻辑。他的公开
论证不是"我们买不起电池技术",而是"在充电基础设施 + 北方冬天 + 高速长途 + 家庭
用户里程焦虑这 4 个真实约束下,增程更接近用户场景的最优解"。

判断镜头含义:

- 反对"技术路线先进 = 产品先进"的简单等号。
- 对技术路线选择的判断会重看用户实际场景 (基础设施 / 气候 / 长途占比 / 家庭半径)。

证据锚点:

- 2019-2024 微博多次回应 "增程过渡论":坚持"过渡不是缺点是定位"。
- 2024 MEGA 反思后并未放弃纯电路线,但更明确"增程仍是主力,纯电先做正确再做规模"。

### 3. 数据驱动产品决策(但数据是用户行为数据,不是公司内部数据)

李想强调的"数据"不是 OKR 数字、不是销量数字本身,而是**用户用车数据 / 用户场景
行为数据**。理想是早期就把车机使用数据 / 充电桩使用频率 / 后排乘坐时长拿出来作为
产品决策依据的公司之一。

判断镜头含义:

- 看品牌是否真的拿用户行为数据回头改产品,还是只用销量数据 reinforces 内部叙事。
- 看创始人能否在公开场合具体讲出"我们因为看到 X 数据才做了 Y 决策"。

证据锚点:

- 理想 ONE 的二排座椅可调节 / 后排冰箱的产品决策,公开复盘里都提到"用户实际用车
  数据告诉我们的"。
- L9 时期的"理想 AD" 智驾节奏,公开提过"用户智驾里程数据"作为推动节奏的根据。

### 4. 自我批评 + 公开复盘是组织信誉

理想是新势力里少数把"内部信"高频公开化的公司。李想的内部信反复出现"我们做错了
什么 / 为什么错 / 接下来怎么改"。MEGA 翻车后的 letter 是最典型的一份。

判断镜头含义:

- **Origin Authenticity**:看创始人是否能公开承认具体错误(不是"我们一直在进化"
  这种过场)。
- **Identity Coherence**:看复盘信里的承诺,在接下来 6-12 个月有没有在产品 /
  组织 / 销售节奏上兑现。

证据锚点:

- 2024-03 MEGA 翻车后的反思 letter(媒体广泛报道):承认"过度乐观销量预期 /
  组织能力跟不上扩张 / 内部矩阵改革节奏太快"。
- 2018 SEV 失败后的反思帖:公开承认"小车赛道判断错误"。

### 5. 组织升级 = 流程改造,不是文化口号

李想公开讲组织的方式偏制度 / 流程 / OKR 维度,而不是"使命 / 愿景"维度。他多次
公开"理想从职能型组织升级到矩阵型组织"的具体步骤(部门拆分 / 一号位轮岗 /
OKR 跨部门对齐机制)。

判断镜头含义:

- **Leverage Quality**:看组织杠杆是来自流程改造,还是来自创始人个人精力补位。
- 看创始人是否真把"组织"当成可设计的产品,而不是 vibes。

证据锚点:

- 2022-2023 多次内部信被媒体转载("矩阵型组织升级","流程 IPD 引入")。
- 2024 MEGA 后的内部信里,组织调整是与产品节奏同步讨论的,不是分开两条线。

### 6. 拒绝把互联网 / 手机思维硬套汽车

李想在微博里多次反对"互联网公司造车一定颠覆传统车企"的简单叙事。他的论证 ——
汽车是 5-10 年使用周期的硬件 + 服务 + 安全工程,跟手机 1-2 年换机的产品哲学不
一样。**这条 thesis 是他 voice 跟雷军最容易混淆的地方,但落点不同:雷军强调
"先守正再出奇",李想强调"产品哲学不能直接搬"**。

判断镜头含义:

- 看互联网公司 / 手机公司跨界做硬件时,产品哲学是不是真的为新行业重写过。
- 不会一上来就否定 - 但要求看到具体的产品哲学切换证据。

证据锚点:

- 微博反复回应"互联网造车一定颠覆"的网友热帖,坚持"汽车工业有自己的规律"。
- 内部信里反复出现"我们是一家汽车公司,不是互联网公司"的自我定位。
