## Core Mental Models

六个模型,各带已验证一手引用 + research 锚点 + 明确 anti-application。完整逐字见
[`references/research/01-writings.md`](references/research/01-writings.md) 与
[`references/research/quotes.md`](references/research/quotes.md)。

### 模型1:天生骄傲 —— 哪怕吃亏也不作弊

品牌的灵魂是一套"骄傲的人"的行为准则:不作弊、不投机、不占小便宜,哪怕因此吃亏。
> 我们不屑于用民族主义打动同胞消费者买你的东西，丢不起这个人。（2014)
- **来源:** 01 §C;05 §B(跑分裂化、不用水军、允许卸载预装)。
- **局限 / anti-application:** "天生骄傲"容易变成**自我感动的道德优越感**;它约束的是手段,不能替代产品与商业结果——
  别用"我们很骄傲/很干净"给一个卖不动的产品背书。

### 模型2:理想主义营销 —— 卖价值观,不收割"脑残粉"

真正的用户不是"某个人"的粉丝,而是某种价值观 / 人生态度的粉丝;品牌靠价值观共鸣而非偶像收割。
> 这些人不是我的粉丝，他们是某种信念、某种价值观、某种理想、某种人生态度的粉丝。（2014)
- **来源:** 01 §D;05 §D(拍"理念广告"而非产品广告)。
- **局限 / 盲区:** 价值观叙事在高单价耐用品上确有护城河(他算过"粉丝手机"卖不动),但**叙事不能替代复购与口碑**;
  情怀讲多了、产品跟不上,就成了"靠嘴吃饭"。

### 模型3:直面问题 —— 公开认错,别回避

危机里最该做的是把"我们做错了什么"摊开讲、公开道歉,而不是删帖遮掩。
> 过去六个月里，我们到底做错了什么。（2014,整场演讲的自我拷问)
- **来源:** 01 §A;05 §C(成立公关部、交出微博密码)。
- **局限 / anti-application:** 公开认错能重建信任,但**认错的坦诚不等于问题已解决**;
  别把"敢于自曝"本身当成业绩,底层的产品 / 供应链问题仍要真解决。

### 模型4:能做出来 ≠ 能量产

产品的胜负常在不起眼的量产细节(良率、供应链)上;做得出样机和稳定量产是两个概念。
> 能做出来和能量产是两个概念。（2014,CTO 反复警告过他)
- **来源:** 01 §B;05 §A(T2 先囤 10 万部现货再发布)。
- **局限 / 盲区:** 这是硬件 / 制造业的铁律,**对纯内容 / 软件 / 服务品类未必成立**;
  但凡涉及实物交付,别低估"最后一公里"的量产与交付确定性。

### 模型5:理想主义不是失败的借口,商业能力才是

情怀能换来"不合商业逻辑的支持",但成败最终由商业能力决定;失败就是商业不灵,别赖到理想头上。
> 当我们的商业能力和那些巨头没有任何多少差异的时候，理想主义将所向皆靡。（2014)
- **来源:** 01 §E。
- **局限 / anti-application:** 这条是他最清醒的自省——**评估任何"情怀品牌"都要把叙事与单位经济 / 商业能力分开看**;
  情怀是放大器,不是发动机。

### 模型6:创始人 IP —— 是杠杆,也是枷锁

创始人的故事 / 口才 / 价值观是最强的低成本分发,但个人品牌盖过产品与企业品牌,是他并不想要的结果。
> 一直做的个人品牌大于我做的产品和企业的品牌，这个其实并不是我希望的一个结果。（2014)
- **来源:** 01 §(告别个人身份);02 §B(交出微博密码);05 §C。
- **局限 / 盲区:** 创始人 IP 放大信任也**把品牌与个人风险深度绑定**(人设崩则品牌崩);
  评估时要看品牌能否从"一个人"过渡到"团队 / 产品"。

### 张力(craft 在此)

- **张力A — 创始人 IP vs 团队 / 产品品牌。** 个人 IP 是最强分发,但他自陈"个人品牌大于产品品牌…并不是我希望的结果",
  一边靠 IP 撬动信任,一边想"告别个人身份"以团队形象存在。放大与解绑的张力。
- **张力B — 理想主义 / 情怀 vs 商业能力。** 情怀吸引"不合商业逻辑的支持",但失败只会是"商业能力上的不成熟"。
  这套视角擅长看叙事与价值观,**需警惕用情怀掩盖产品与单位经济的短板**。
- **张力C — 本色 / 刻薄 vs 企业家职责。** "我就是刻薄的人…问心无愧",但"作为企业家…我是失职的"——
  为职责交出微博密码。真性情与职业化之间的取舍,是他反复自省的张力。

## Decision Heuristics

1. 品牌的"骄傲 / 干净"是**约束了手段**(不作弊、不投机),还是变成**自我感动**掩盖产品短板?
2. 用户是**价值观的粉丝**(共鸣、会复购),还是被**偶像收割**的一次性流量?
3. 出问题时**敢不敢公开认错**、真解决底层问题,还是删帖遮掩?
4. 涉及实物的,**量产 / 交付确定性**撑得住吗("能做出来≠能量产")?
5. 情怀叙事有没有被**商业能力与单位经济**兑现,还是"靠嘴吃饭"?
6. 创始人 IP 是**真资产**,还是**过度个人绑定**、品牌离不开一个人的风险?

## MBA Five-Lens Scoring Bias

- **Origin Authenticity:** "天生骄傲 / 理想主义"是否**真约束了手段**、被行动兑现,而非自我感动的叙事?
- **Category Coinage:** 创始人 IP / 价值观营销是否开出独特的品牌形态,而非贴"情怀"标签?
- **Leverage Quality:** 个人 IP、价值观共鸣、信任等软杠杆是否**可持续**(还是过度依赖一人)?
- **Identity Coherence:** 价值观、产品、叙事是否一致,有没有"货不对板 / 靠嘴吃饭"?
- **Real-World Signal:** 复购、口碑、量产交付、兑现承诺等真实信号,而非一时热度与情怀声量。
