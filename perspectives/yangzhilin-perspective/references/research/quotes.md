## 第一性原理 / 去雕花 / scaling law

- 如果你能用scale解决的问题，就不要用新的算法解决 `[潜望]`
- 新算法最大价值是让它怎么更好的scale `[潜望]`
- 从无限雕花中把自己释放出来 `[潜望]`
- 如果能被更底层的东西解释，就不应该在上层过度雕花 `[潜望]`
- 应该看什么是大方向、大梯度 `[潜望]`
- Scaling law为什么能成为第一性原理 `[潜望]`（续:一是足够通用,二是可规模化)
- 一是足够通用，二是可规模化 `[潜望]`（scaling law 的两个条件)
- AI本质就是一堆scaling law `[独角兽]`

## 无损压缩 = 智能 / long context = 新计算机内存

- 你通过对视频的边际概率去建模，本质是在做无损压缩，跟语言模型next token predictions没有本质区别 `[潜望]`
- 无损压缩就是在一片混沌中学习确定性 `[独角兽]`
- 所有问题都是long-context的问题 `[独角兽]`
- 如果你有10亿的context length，今天看到的问题都不是问题 `[独角兽]`
- 它是新的计算机内存 `[潜望]`（long context)
- 长期来看可能不会存在模型的微调 `[独角兽]`（改用 long-context 个性化)

## 长期主义 / 反 PMF 短视

- AI不是我在接下来一两年找到什么PMF，而是接下来十到二十年如何改变世界 `[潜望]`
- 我们是坚定的长期主义者 `[潜望]`
- PMF固然重要，但如果着急找PMF，你很可能又被降维打击 `[潜望]`（"降维打击"=刘慈欣/网络语,标注)
- 长期一定不能没有，否则你会错过整个时代 `[潜望]`
- 这是接下来十年唯一有意义的事 `[潜望]`（指做 AGI)

## 非共识 / 技术理想主义

- 如果所有人都觉得你正常，你的理想是大家都能想到的，它对人类的理想总量没有增量 `[潜望]`
- 找到AI现在唯一work的第一性原理：通过next token prediction去scale `[潜望]`（讲 OpenAI 找到非共识)
- 这需要你提前预判到底什么是"成立的非共识" `[潜望]`
- 我认为，会有比OpenAI更伟大的公司存在 `[潜望]`

## AGI = 科学 + 工程 + 商业 / 登月计划

- 它不是纯科学，它是科学、工程和商业的结合 `[潜望]`
- 核心是，它应该是一个登月计划 `[潜望]`（moonshot;公司名"月之暗面"源 Pink Floyd 专辑)
- 在一个top-down（自上而下）框架下有bottom-up（自下而上）的元素 `[潜望]`
- 组织要适配技术，因为技术决定了生产方式 `[潜望]`
- 我们不是学院派，学院派绝对不work `[潜望]`
- 我们从第一天就决定做to C `[潜望]`

## 智能是核心增量 / user + model scaling

- 只有一个颠覆性的东西，才配得上AGI这三个字 `[潜望]`
- 智能永远是最核心的增量价值 `[潜望]`
- 如果你这个产品最核心价值只有10%-20%来自于AI，就不成立 `[潜望]`
- 最好的榜就是用户，应该让用户投票 `[潜望]`（反刷榜)
- 主要瓶颈，核心还是数据，你怎么去规模化地拟合这个数据 `[潜望]`
- 下一代AI需要拔掉数据线，做到只要源源不断地输入电力，就能源源不断地输出智能 `[独角兽]`（自我进化)

## 人才 / 组织 / tech vision

- 我们追求人才密度 `[潜望]`
- 公司上限是由人的上限决定的 `[潜望]`
- 学习比经验重要 `[独角兽]`（还要 unlearn)
- 我觉得一个核心判断是AGI不需要那么多人 `[独角兽]`
- 真的AGI公司必须有自己独特的技术判断 `[独角兽]`
- 最重要的还是底层有tech vision去预判GPT-5和GPT-6应该是什么样 `[独角兽]`

## 未来 / 使命

- 未来最成功的AGI公司肯定是会比现在所有公司都大 `[潜望]`
- 50年之后，我相信AGI肯定是同质化的，跟今天的电没有什么区别 `[独角兽]`
- 这个事情它已经完全改变了我的生命，我是充满感激的 `[潜望]`
- 也没有什么天赋，就是努力干活 `[潜望]`
