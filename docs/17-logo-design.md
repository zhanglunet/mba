# 17 — MBA Logo 设计说明与使用规范

> **Status**: v1 · 2026-07 · **线上版(唯一权威,含可交互演示)**: [mbabrand.com/logo-design.html](https://mbabrand.com/logo-design.html)
> 落位:`site/index.html`(masthead + favicon)· 规范页 `site/logo-design.html`

## 1. 设计理念:产品即符号

MBA 的方法论是"固定 **5 镜头** × 7 维度,N 评委独立打分"——每份报告的第一张图都是五镜头雷达。
站标不另造抽象图形,直接把这张雷达图提炼成标志:**五镜头雷达 glyph + MBA. 字标**。

| 元素 | 语义 |
|---|---|
| 外层正五边形 | 评分框架 —— 5 个镜头,固定、可比、不漂移 |
| 内部数据形 | 一次审计 —— 五个镜头各有高低,真实品牌从不是满分正多边形 |
| 峰值评分点 | 最高分镜头上的强调点 —— 每个品牌都有最锋利的一面 |
| MBA. 字标 | 编辑部刊头风格,句点收尾 = "审计有结论" |

> ⚠️ **Anti-fabrication**:数据形五个半径是纯造型比例,**不是任何真实品牌的评分**,禁止当示例数据引用或标注分数。

## 2. 几何

- viewBox `0 0 64 64`,中心 `(32, 33)`,外接圆 `R = 26`,顶点朝上(−90°/−18°/54°/126°/198°)
- 网格:50% 内环 + 五条辐条,ink 12% 透明度
- 数据形半径比例(自顶点顺时针):`0.94 · 0.66 · 0.78 · 0.55 · 0.72`(峰值在顶,重心偏右下,不对称张力)
- 评分点:数据形顶点 `(32, 8.6)`,r `3.2`,纸色描边 `1.5`

## 3. 色彩

| Token | 值 | 用途 | 对比度 |
|---|---|---|---|
| `--ink` | `#111111` | 外框、字标(纸底) | ≈16:1 vs paper ✓ |
| `--accent` | `#c1440e` | 数据形、评分点、句点(纸底) | ≈4.8:1 vs paper ✓(图形/大字号) |
| `--paper` | `#faf8f3` | 底色、评分点描边 | — |
| `--accent-dark` | `#d95f27` | **深色底专用**强调色 | ≈5.0:1 vs #111 ✓ |

渐变填充:`#c1440e` 自上而下 `stop-opacity .30 → .06`。深色底:外框/字标改纸色,强调色换 `--accent-dark`(#c1440e 在墨底仅 ≈3.4:1,不达标)。

## 4. 动效("打分"开场,只播一次)

| 元素 | 动画 | 参数 |
|---|---|---|
| 数据形 | 描线(`pathLength=100`,dashoffset 100→0)+ 填充渐显 | 1.1s · cubic-bezier(.6,0,.25,1) · delay 0.2s |
| 评分点 | scale 0→1 弹出 | 0.4s · cubic-bezier(.2,1.6,.4,1) · delay 1.2s |
| 评分点 hover | scale 1→1.35 | 0.25s 回弹 |

全部包在 `@media (prefers-reduced-motion: no-preference)` 内,关动效 = 直接成品态。

## 5. 变体与使用规范

**变体**:标准横排 lockup(masthead)· 仅 glyph(头像/小尺寸)· 深色底版 · favicon(去网格、加粗描边、圆角纸底,SVG data URI)。

**规则**:glyph ≥16px、lockup 高 ≥20px;四周留白 ≥0.25× glyph 宽;单色场景用全 ink 版。
**禁止**:拉伸/旋转/改半径比例;换配色;当真实雷达图用或标分数;加投影发光 3D;低对比底色上用彩色版。

## 6. 嵌入

内联 SVG,零外部资源;SVG 内颜色写死(favicon 无 CSS 上下文,不能用 CSS 变量)。完整代码与动效 CSS 见 [线上规范页 §08](https://mbabrand.com/logo-design.html) 或 `site/index.html` 的 `.mark-glyph` 段。无障碍:容器 `role="img"` + `aria-label`,SVG `aria-hidden`。
