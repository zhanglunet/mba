# MBA 项目 5 分钟 Pitch 稿(高级版)

> **核心升维 / Key reframe**
>
> **AI 时代,品牌不只是人类心智资产,而会变成 Agent 选择、推荐、排序、判断公司的基础信号。MBA 要做的,就是品牌影响力的 Agentic Audit Protocol。**

---

各位好,我今天要讲的项目叫 **MBA — Metric Brand Auditor**。

但我不想从“品牌分析工具”开始讲。

因为如果我们只是说“用 AI 生成品牌报告”，这个项目马上就会变得很普通。

我真正想讲的是一个更大的变化：

**在 AI 时代，品牌正在从“人类心智资产”，变成“智能体世界里的可识别、可比较、可调用的信号资产”。**

过去，一个品牌强不强，主要取决于人怎么记住它。

消费者怎么感知它，投资人怎么理解它，媒体怎么叙述它，候选人愿不愿意加入它。

但未来，越来越多的判断会被 Agent 参与甚至代理。

投资 Agent 会筛公司，采购 Agent 会推荐供应商，招聘 Agent 会理解雇主品牌，消费者 Agent 会帮人比较产品，企业战略 Agent 会持续追踪竞争格局。

这时候，一个公司的品牌影响力就不再只是 slogan、logo、PR 或社媒声量。

它会变成一个更底层的问题：

**当 AI Agent 观察一家公司时，它能不能稳定地识别这家公司是谁、代表什么、定义了什么品类、拥有多少真实信号、是否值得被推荐、被投资、被合作、被加入？**

今天，没有一套系统能严肃回答这个问题。

这就是 MBA 要解决的事。

**一、问题：品牌很重要，但今天仍然不可审计**

所有人都知道品牌重要。

创业公司融资要讲品牌。

上市公司做 IR 要讲品牌。

AI 公司要讲生态、开发者心智和模型认知。

消费公司要讲用户身份。

B2B 公司要讲信任、渠道和行业位置。

但今天我们判断品牌，主要还是靠三类东西：

第一类，是咨询公司报告。

它很漂亮，但经常是一次性的、主观的、不可复盘的。

第二类，是舆情和社媒工具。

它能告诉你声量、曝光、情绪，但它很难告诉你：

这个品牌到底有没有品类定义权？

它的叙事是不是被市场接受？

它的影响力是结构性的，还是短期 PR 造成的？

第三类，是投资人、创始人、CMO 自己的判断。

这种判断很重要，但高度依赖个人经验，难以标准化，难以跨公司比较，更难以跨时间追踪。

所以我们看到一个巨大空白：

**财务有审计。产品有数据分析。销售有 CRM。舆情有 monitoring。**\
**但品牌影响力，这个决定公司长期定价权、资本吸引力、人才吸引力和市场话语权的资产，今天还没有真正的审计协议。**

MBA 就是要做这个协议。

**二、解决方案：把品牌影响力拆成 Agent 可以执行的审计流程**

MBA 的全称是 **Metric Brand Auditor**。

它不是单个 prompt，也不是一个报告模板。

它是一条多智能体品牌影响力审计流水线。

现在的网站已经把核心机制表达得很清楚：MBA 是由 Lead Agent 编排、子智能体并行执行、多个评委视角独立打分的品牌影响力审计流程，并且强调 scored、versioned、replayable，也就是可打分、可版本化、可复盘。

它的工作方式可以分成五步。

第一步，**Router**。\
Lead Agent 先判断这是一个新品牌审计，还是已有品牌的版本更新。

第二步，**并行调研**。\
系统不会简单问“这个品牌好不好”，而是把品牌拆成多个维度：创始叙事、产品定位、分发渠道、社区与 PR、视觉语言、竞品格局、用户情绪。GitHub README 里也明确写到，MBA 的调研输入是这 7 个维度。

第三步，**综合判断**。\
Lead Agent 把不同维度的调研结果合成，找出品牌杠杆、叙事矛盾、脆弱边缘和证据链。

第四步，**多视角打分**。\
不是让一个模型拍脑袋给分，而是用多个 Judge Skills，从不同认知模型审视同一个品牌。当前系统使用 5 个打分镜头：原创性、范畴命名、杠杆质量、身份一致性、真实信号。

第五步，**生成版本化报告**。\
输出不是一段文字，而是 Markdown 和自包含 HTML，包括雷达图、异议热力图、影响力构造图，并沉淀为版本快照。网站首页也明确展示了 Markdown + HTML 图表输出。

所以 MBA 真正做的事情是：

**把品牌判断，从主观表达，变成一个 Agent 可以重复执行的审计流程。**

**三、为什么这件事现在重要？**

因为 AI 正在改变品牌的存在方式。

过去，品牌主要影响人。

未来，品牌还要影响 Agent。

人类会记住一个品牌，因为它有故事、有审美、有身份、有文化符号。

但 Agent 会识别一个品牌，是因为它有结构化信号：

它是否有稳定叙事？

是否有清晰品类？

是否被高质量来源反复引用？

是否在竞品对比中拥有独特位置？

是否有真实用户、真实增长、真实生态？

这意味着未来品牌管理会发生一个变化：

**品牌不只是要被人喜欢，还要能被机器正确理解。**

如果一个公司对外表达混乱，官网讲一套，财报讲一套，创始人采访讲一套，客户案例讲一套，那么未来不仅人会困惑，Agent 也会困惑。

而当 Agent 困惑时，它就不会推荐你，不会解释你，不会把你放在正确的比较集合里。

这对 AI 公司、B2B 公司、上市公司、消费公司、创业公司都会非常关键。

所以 MBA 的长期价值不是“帮你写一份品牌报告”。

而是帮助公司回答：

**在 AI Agent 的世界里，我这个品牌是否足够清晰、可信、可比较、可推荐？**

**四、我们的第一个切入点：投资人与创始人**

我们不会一开始就做一个泛泛的营销工具。

那样很容易被压成普通 SaaS。

MBA 的第一个高价值场景，是 **投资研究和创始人战略叙事**。

对投资人来说，品牌影响力其实是判断长期复利的重要变量。

一家公司的增长，到底来自真实品牌势能，还是渠道红利？

一家公司的估值溢价，到底来自品类定义权，还是市场短期情绪？

一家 AI 公司讲了很多宏大故事，但它有没有真正进入用户心智？

一家消费公司很火，但这种热度能不能沉淀为定价权？

MBA 可以成为投资研究里的一个新模块：

**Brand Influence Due Diligence。**

它帮助基金经理、VC、PE、二级市场研究员判断：

这个公司有没有真正的品牌资产？

这个品牌资产是在增强，还是在衰减？

它是否支撑未来估值？

对创始人来说，MBA 是一个战略镜子。

很多创业公司最大的问题不是没有故事，而是故事太多。

融资 deck 是一个故事，官网是一个故事，产品 demo 是一个故事，销售材料是一个故事，媒体采访又是另一个故事。

MBA 可以帮助创始人判断：

我的公司到底代表什么？

我有没有定义一个新品类？

客户、投资人、候选人、媒体看到的是不是同一家公司？

我的品牌势能能不能调动人才、资本、渠道和生态？

所以 MBA 一开始不应该卖给普通市场部。

它应该卖给那些真正关心“品牌如何转化为战略资源”的人：

基金经理、创始人、CEO Office、IR、战略部、咨询公司。

**五、商业化路径：从审计报告，到持续监控，到 Skills 平台**

MBA 的商业化可以分三层。

第一层，是 **高价值单次审计**。

我们可以为一家创业公司、上市公司或投资标的做一次完整的 Brand Influence Audit。

输出包括品牌影响力评分、叙事一致性诊断、竞品定位、品类定义权、风险点和行动建议。

第二层，是 **持续监控订阅**。

对基金来说，可以监控一个 portfolio 或 watchlist。

对上市公司来说，可以监控自己和核心竞品。

对创业公司来说，可以每月追踪品牌叙事是否变强、市场是否更理解自己。

这里 MBA 从一次性报告，变成品牌影响力的时间序列。

README 里已经写到，MBA 支持 evolution 模式：如果已有报告，系统会只研究发生变化的维度、只重判受影响维度，并写入新版本快照。

这是非常关键的能力。

因为真正有价值的不是“今天品牌几分”，而是：

**这个品牌的影响力在过去 30 天、90 天、180 天，到底发生了什么结构性变化？**

第三层，是 **Skills 平台化**。

MBA 最终可以拆成一组可调用 Skills：

Brand Audit Skill。

Category Mapping Skill。

Narrative Diff Skill。

Competitor Benchmark Skill。

Investor Memo Skill。

IPO Narrative Check Skill。

Monthly Brand Monitoring Skill。

Founder Story Audit Skill。

这些 skills 可以接入企业知识库、投研系统、咨询工作台，也可以成为更大 Agent 平台中的品牌智能模块。

**六、壁垒：我们沉淀的不是报告，而是品牌影响力数据结构**

这个项目的壁垒不是调用大模型。

任何人都可以让 AI 写一篇品牌分析文章。

MBA 的壁垒在于五件事。

第一，是 **审计框架**。\
我们把品牌拆成 7 个调研维度和 5 个判断镜头。它不是随便写，而是有固定口径。

第二，是 **多智能体流程**。\
Lead Agent 负责拆解和合成，Research Agents 负责并行调研，Judge Skills 负责独立判断，最后形成可复盘报告。

第三，是 **版本化数据库**。\
每一次审计都会成为品牌影响力的一次快照。时间越长，系统越知道一个品牌如何演化。

第四，是 **跨品牌 benchmark**。\
当我们审计足够多公司之后，就可以比较：谁真正拥有品类定义权？谁只是流量强？谁的叙事在增强？谁的品牌资产在透支？

第五，是 **可治理的输出机制**。\
当前网站已经明确声明这是黑客松演示版，并提示评委头像、评分和辩论是 AI 模拟，不代表本人真实意见，也不构成投资建议或真实品牌评价。\
商业化版本会进一步去掉真人模拟风险，升级成角色化 Lens，比如 Investor Lens、Founder Lens、Product Lens、Category Lens、Distribution Lens、Culture Lens。

这会让 MBA 从 demo 走向企业级可用系统。

**七、结尾：我们的愿景**

我们相信，未来每一家重要公司，都需要一份持续更新的 **Brand Influence Ledger**。

就像公司有财务报表，产品有数据看板，销售有 CRM，舆情有监控系统。

品牌影响力也应该有自己的审计系统。

因为品牌不是装饰。

品牌是一个公司调动社会资源的能力。

它决定你能吸引什么人才。

能进入什么渠道。

能获得什么资本。

能拥有多少定价权。

能不能定义一个品类。

能不能在 AI Agent 的世界里被正确理解和优先推荐。

MBA 要做的，就是把这件事变成一套可执行的系统。

**我们不是在做 AI 品牌报告。**\
**我们是在做 AI 时代的品牌影响力审计协议。**

一句话总结：

**MBA 是一个 Agentic Brand Influence Auditor。**\
**它把品牌影响力从主观感觉，变成可调研、可打分、可比较、可复盘、可持续追踪的智能资产。**

谢谢大家。
