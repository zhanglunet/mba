## A. The Founding Moves (2014)

### A1. Appointed CEO — 2014-02-04

- Announcement: https://news.microsoft.com/source/2014/02/04/microsoft-board-names-satya-nadella-as-ceo/ (Microsoft Stories; verification indirect — news.microsoft.com returned 403 to WebFetch; content cross-confirmed via https://blogs.microsoft.com/blog/2014/02/04/introducing-microsofts-new-ceo-satya-nadella/ and https://blogs.microsoft.com/blog/2014/02/04/microsoft-employees-give-new-ceo-satya-nadella-a-12th-man-welcome/ )
- First-day email to employees: https://news.microsoft.com/source/2014/02/04/satya-nadella-email-to-employees-on-first-day-as-ceo/
- Direct quote (cross-cited): "Today is a very humbling day for me. It reminds me of my very first day at Microsoft, 22 years ago."
- Direct quote (cross-cited): "The opportunity ahead for Microsoft is vast, but to seize it, we must move faster, push harder and continue to transform."
- Bill Gates simultaneously stepped down as chairman and took a "technology advisor" role: "During this time of transformation and remarkable opportunity for Microsoft, there is no better person to lead the company than Satya."
- John Thompson became chairman of the board.

### A2. Office for iPad — 2014-03-27 (first public event as CEO)

- Recode live coverage: https://www.recode.net/2014/3/27/11625000/live-as-satya-nadella-makes-office-for-ipad-his-ceo-debut
- Fortune: https://fortune.com/2014/03/27/microsoft-ceo-satya-nadella-announces-office-for-ipad-at-public-debut/
- Bloomberg: https://www.bloomberg.com/news/articles/2014-03-27/microsoft-ceo-satya-nadella-announces-office-apps-for-ipad
- Day-52 of his tenure. Office for iPad had been ready for years; Ballmer had withheld it while Windows touch devices were ready.
- Rationale: monetize Office where users are, not where Microsoft wants them to be — explicit "post-Windows-first" signal.
- Stratechery 2022 retrospective: this was the move that, with hindsight, marked the strategic break with Ballmer https://stratechery.com/2022/microsoft-full-circle/

### A3. "Microsoft loves Linux" — 2014-10-20

- YouTube clip (Azure Cloud Briefing slide): https://www.youtube.com/watch?v=54hHr8ye2kE
- Background: https://www.idginsiderpro.com/article/3584911/how-microsoft-went-from-linux-is-a-cancer-to-microsoft-loves-linux.html (CIO)
- Slide displayed by Nadella at a San Francisco Azure event reading "Microsoft loves Linux."
- Reversed Ballmer's 2001 "Linux is a cancer" framing. Set up Azure as a multi-OS cloud and prepared the GitHub / open-source pivot. Microsoft joined the Linux Foundation in 2016.

### A4. "Mobile-first, cloud-first" doctrine — Q3 / Q4 FY14

- SEC 8-K filing with Q3 FY14 earnings: https://www.sec.gov/Archives/edgar/data/0000789019/000119312514157077/d713611dex991.htm
- SEC 8-K filing with Q4 FY14 earnings: https://www.sec.gov/Archives/edgar/data/0000789019/000119312514275529/d759928dex991.htm
- Q4 FY14: "We are galvanized around our core as a productivity and platform company for the mobile-first and cloud-first world."
- The phrase is the strategic banner for 2014–2017; "intelligent cloud, intelligent edge" replaces it from 2017 onward; "AI platform shift" replaces that from 2023.

## B. The Acquisitions That Defined the Pivot

### B1. Mojang (Minecraft) — 2014-09-15, $2.5B

- Inherited from Ballmer's process but closed under Nadella; first major signal that Microsoft would invest in cross-platform brand IP (Minecraft on PlayStation, mobile, etc.).
- See American Bazaar acquisitions retrospective https://americanbazaaronline.com/2025/08/21/the-billion-dollar-blueprint-microsofts-biggest-acquisitions-under-satya-nadella-466570/

### B2. LinkedIn — 2016-06-13 announced, 2016-12-08 closed, $26.2B

- Announcement: https://news.microsoft.com/source/2016/06/13/microsoft-to-acquire-linkedin/ (canonical; news.microsoft 403)
- GeekWire: https://www.geekwire.com/2016/microsoft-buy-linkedin-26-2b-largest-acquisition-history-blockbuster-deal-tech-industry/
- SEC 8-K with press release: https://www.sec.gov/Archives/edgar/data/0000789019/000119312516619689/d116815dex991.htm
- $196/share all-cash, Microsoft's largest deal at the time.
- Jeff Weiner continued as LinkedIn CEO reporting to Nadella; Reid Hoffman stayed as chairman/controlling shareholder, called it "a re-founding moment for LinkedIn."
- Nadella rationale: data graph of professional identity + productivity data + a publishing surface for thought-leadership (which became his own primary long-form channel).

### B3. GitHub — 2018-06-04 announced, 2018-10-26 closed, $7.5B (stock)

- Announcement: https://news.microsoft.com/source/2018/06/04/microsoft-to-acquire-github-for-7-5-billion/ (canonical; cross-confirmed via Open Source blog https://opensource.microsoft.com/blog/2018/06/07/github-acquisition-microsoft-commitment-open-source-developers and Satya tweet https://x.com/satyanadella/status/1055821526539231235 )
- Microsoft Blog: https://blogs.microsoft.com/blog/2018/06/04/microsoft-github-empowering-developers/ (403 to WebFetch; verified content via https://www.geekwire.com/2018/microsoft-completes-7-5b-github-acquisition-says-user-base-grown-31m-developers/ )
- Direct quote, Nadella (cross-cited across recaps): "Microsoft is a developer-first company, and by joining forces with GitHub we strengthen our commitment to developer freedom, openness and innovation."
- Stewardship pledge: "We will retain its developer-first ethos, operate independently and remain an open platform."
- Nat Friedman appointed CEO, reporting to Scott Guthrie. Chris Wanstrath became a Microsoft technical fellow.
- This deal made the open-source / "developer-first" claim concrete and became the substrate for GitHub Copilot in 2021.

### B4. Activision Blizzard — 2022-01-18 announced, 2023-10-13 closed, $68.7B

- Announcement: https://news.microsoft.com/source/2022/01/18/microsoft-to-acquire-activision-blizzard-to-bring-the-joy-and-community-of-gaming-to-everyone-across-every-device/
- M&A retrospective: https://mnawatch.com/legal-retrospectives/v4iq25w03z596aw5ywvvbqykik12mb
- Wikipedia composite: https://en.wikipedia.org/wiki/Acquisition_of_Activision_Blizzard_by_Microsoft
- All-cash, $95.00/share. Microsoft's largest deal ever.
- Direct quote, Nadella (cross-cited): "Gaming is the most dynamic and exciting category in entertainment across all platforms today and will play a key role in the development of metaverse platforms."
- Faced FTC challenge (US), CMA initial block then re-approval (UK), EU approval with remedies. Microsoft made multiple platform-openness commitments (Call of Duty on PlayStation, Switch, cloud).
- Post-close total cost ~$75.4B with hold-period charges.

### B5. Nuance Communications — 2021-04-12 announced, 2022-03-04 closed, ~$19.7B

- Speech/AI/healthcare-document company that became substrate for Microsoft's vertical healthcare AI play and several Copilot features. Often missed from popular timelines but materially important to the Copilot productized story.

### B6. Inflection AI team-lift — 2024-03-19

- Microsoft blog: https://blogs.microsoft.com/blog/2024/03/19/mustafa-suleyman-deepmind-and-inflection-co-founder-joins-microsoft-to-lead-copilot/ (403 to WebFetch; cross-confirmed via https://techcrunch.com/2024/03/19/microsoft-hires-inflection-founders-to-run-new-consumer-ai-division/ and https://fortune.com/2024/03/19/microsoft-surprise-deal-inflection-ai-mustafa-suleyman-reid-hoffman-questions/ )
- Mustafa Suleyman (DeepMind, Inflection co-founder) joined as EVP and CEO of Microsoft AI, reporting to Nadella. Karén Simonyan joined as Chief Scientist. Most of Inflection's engineering/research team came with them.
- Microsoft paid Inflection a ~$650M licensing fee; structured as not-an-acquisition to dodge antitrust review (UK CMA later cleared without in-depth probe — https://www.cnbc.com/2024/09/04/microsoft-avoids-uk-probe-into-hiring-of-inflection-ai-employees.html ).
- Rationale: consumer-AI capability without the antitrust exposure of an acquisition; insurance against OpenAI relationship volatility.

## C. The OpenAI Bet (the most consequential single decision)

### C1. Initial $1B investment — 2019-07-22

- Microsoft Stories: https://news.microsoft.com/source/2019/07/22/openai-forms-exclusive-computing-partnership-with-microsoft-to-build-new-azure-ai-supercomputing-technologies/
- Direct quote, Nadella: "By bringing together OpenAI's breakthrough technology with new Azure AI supercomputing technologies, our ambition is to democratize AI — while always keeping AI safety front and center — so everyone can benefit."
- Made Azure OpenAI's exclusive compute provider. Bill Gates reportedly told Nadella "you're going to burn this billion dollars" (Fortune https://fortune.com/article/microsoft-ceo-satya-nadella-bill-gates-openai-sam-altman-youre-going-to-burn-this-billion-dollars-big-tech/ ).

### C2. Multi-year extension — 2023-01-23

- Microsoft Blog: https://blogs.microsoft.com/blog/2023/01/23/microsoftandopenaiextendpartnership/ (403; cross-confirmed via WSJ / Reuters at time)
- Reported total commitment ~$10B (mix of cash, Azure credits, revenue-share).
- Made Copilot / Bing Chat possible at scale.

### C3. November 2023 OpenAI board crisis — Nadella's stabilizer role

- Sam Altman was fired by OpenAI's board 2023-11-17, briefly accepted a Microsoft AI lead role offered by Nadella, returned to OpenAI 2023-11-22 with a reconstituted board.
- Treated by media (WSJ, Bloomberg, NYT) as Nadella's calmest crisis-management moment. URLs paywalled; treat reproduction of direct quotes as requires live verification.

### C4. 2026 partnership renegotiation

- TechCrunch: https://techcrunch.com/2026/04/29/satya-nadella-says-hes-ready-to-exploit-the-new-openai-deal/
- GeekWire "Microsoft-OpenAI Files": https://www.geekwire.com/2026/the-microsoft-openai-files-internal-documents-reveal-the-realities-of-ais-defining-alliance/
- Post-2026-Q1 the relationship is restructured; exact terms — revenue share, exclusive-compute clauses, AGI carve-out — require live verification before quoting.

## E. The Costly No: Killing Nokia Phones — 2015-07-08

- Computerworld: https://www.computerworld.com/article/1620732/microsoft-writes-off-76b-admits-failure-of-nokia-acquisition-2.html
- TechSpot retrospective: https://www.techspot.com/news/102922-decade-later-how-microsoft-flushed-76-billion-down.html
- Thurrott: https://www.thurrott.com/mobile/windows-phone/135827/satya-nadella-admits-nokia-acquisition
- Announced $7.6B impairment + 7,800 job cuts (primarily phone hardware). Largest write-off in Microsoft history at the time (23% larger than the $6.2B aQuantive impairment of 2012).
- Reporting indicates Nadella had been against the Nokia deal before becoming CEO. Acting on that conviction early signaled willingness to admit and reverse Ballmer-era mistakes.

## F. The Product / Brand Re-architecture

### F1. Copilot brand launch — 2023-03-16

- GitHub Copilot existed since 2021. The 2023-03-16 event made "Copilot" the company-wide brand: Microsoft 365 Copilot.
- Future of Work transcript (2023-03-16): https://news.microsoft.com/wp-content/uploads/prod/2023/04/Satya-Nadella-The-Future-of-Work-with-AI.pdf
- Direct quote (per Microsoft Blog recap; URL 403): "This new generation of AI will remove the drudgery of work and unleash creativity."
- Ignite 2023 codified the umbrella: "We are the Copilot company."

### F2. LinkedIn Pulse as Nadella's preferred long-form channel — 2016 onward

- Strategic decision in 2016 (post-LinkedIn acquisition) to make LinkedIn his primary first-person publishing surface (rather than the Microsoft blog or Twitter/X). See `01-writings.md` C-block.

### F3. Cobalt + Maia custom silicon — 2023-11

- Announced at Ignite 2023; Maia 100 AI accelerator + Cobalt 100 ARM CPU.
- Strategic rationale: reduce dependency on Nvidia and on x86; integrate accelerator design with Azure / Copilot stack.
- Coverage: https://redmondmag.com/articles/2023/11/16/nadella-ignite-2023-keynote.aspx

### F4. Chairman role added — 2021-06-16

- CNBC: https://www.cnbc.com/2021/06/16/microsoft-ceo-satya-nadella-will-also-become-chairman-of-the-board.html
- GeekWire: https://www.geekwire.com/2021/microsoft-names-ceo-satya-nadella-chairman-board/
- John Thompson moved to lead independent director. Unanimous board vote.

### F5. AI agents pivot — 2024 → 2025

- Build 2024 introduces Copilot Studio (configurable agents).
- Build 2025 makes Azure AI Foundry Agent Service GA; multi-agent workflows as first-class building blocks (https://www.reworked.co/digital-workplace/microsoft-build-2025-microsoft-pins-its-future-on-ai-agents/ ).
- LinkedIn outlook 2026: "spectacle vs. substance"; from models to systems; from chat to agents (`01-writings.md` C4).

## G. Decision Rationale Patterns (= the playbook)

| Pattern | Decisions that fit it |
|---|---|
| **Concede an OS war Microsoft already lost; meet users where they are** | Office for iPad (A2); Microsoft loves Linux (A3); GitHub independent operation (B3); Activision keep-on-PlayStation commitment (B4) |
| **Buy / partner for the next surface, not the last one** | LinkedIn for professional graph (B2); GitHub for developer surface (B3); Activision for gaming/consumer (B4); Inflection for consumer AI (B6); OpenAI for frontier models (C) |
| **Admit and reverse predecessor's mistake quickly** | Nokia write-off (E); Windows Phone wind-down; killing of Cortana as consumer assistant |
| **Trust as platform precondition, not marketing layer** | SFI memo (D4); leadership comp tied to security (D4); chairman role added to align governance (F4) |
| **Brand consolidation under one umbrella when a category becomes strategic** | "Copilot" replacing Bing Chat / GitHub Copilot / Microsoft 365 Chat (F1); "Microsoft AI" division (B6) |
| **Refuse to commit to AGI as a product promise; commit to productivity** | Dwarkesh 2025; Davos 2026 — substance-over-spectacle framing |

## I. Sequence Anchor (for the SKILL.md author)

The decisions that most define Nadella's CEO operating system, in order of
analytic weight:

1. **OpenAI $1B + Azure exclusive compute (2019-07-22)** — single highest-leverage capital allocation.
2. **Secure Future Initiative + "Prioritizing security above all else" (2023-11 / 2024-05-03)** — defining trust pivot.
3. **GitHub acquisition (2018-06-04)** — defining developer-first move.
4. **Office for iPad + "Microsoft loves Linux" (2014-03 / 2014-10)** — defining cultural break with Ballmer era.
5. **Activision Blizzard ($68.7B, 2022→2023)** — defining regulatory-courage test of "platforms not walled gardens."

Layer **Inflection team-lift (2024-03-19)** and **Nokia write-off (2015-07-08)** as the
two best examples of his "buy the future, kill the past" twin moves.
