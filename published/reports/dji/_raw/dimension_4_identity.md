# Dimension 4 — Identity / 身份系统
# DJI Brand Audit · Phase 2F
# Audit Date: 2026-06-26

## Core Finding

DJI 的身份系统是"极简产品美学 + 功能性产品命名"，缺乏情感叙事层，视觉和语言都指向工程理性而非品牌共鸣。在跨境语境下，这是一个有意为之的选择——让产品本身成为唯一的品牌代言人。

## 视觉身份

- **主色调**：黑色机身（工程感、高端感）+ 银白色 + DJI 红/橙色 accent
- **Logo**：DJI 三字母，字体简洁无衬线，与产品硬件质感统一
- **产品形态**：极简工业设计，所有产品族群有强烈家族辨识感
- **包装**：苹果式极简白盒包装，开箱体验被广泛称赞
- **对比**：与 GoPro（橙色极限运动感）形成鲜明差异，DJI 更偏专业创作者调性

## 产品命名体系

DJI 的命名系统以功能性 + 数字迭代为主，缺乏情感驱动的命名逻辑：

| 系列 | 定位 | 命名逻辑 |
|------|------|---------|
| Phantom | 专业/消费旗舰（已停产） | 幽灵·飞行感 |
| Mavic | 可折叠主流消费 | Maverick（特立独行）→ 独立探索感 |
| Mini | 超轻量合规 | 功能描述性（<250g） |
| Air | 中端消费 | 空气·轻盈 |
| FPV | 沉浸式第一人称视角 | 功能描述性 |
| Inspire | 电影专业级 | 激励·专业创作 |
| Matrice | 企业/行业级 | Matrix·系统集成感 |
| Osmo | 手持相机/稳定器 | 希腊语"推动"，流畅感 |
| Ronin | 电影级稳定器 | 浪人（日语）·武士精神·独立精神 |
| Agras | 农业植保 | Agriculture + 拉丁词根 |

**命名评估**：Phantom、Mavic、Ronin 有一定情感共鸣；Mini、Air、Matrice 是功能描述。整体命名体系并无统一的文化叙事逻辑，是务实的工程师命名思维。与 Anthropic/Claude 的文学诗体命名相比，DJI 缺乏审美统一性。

## 品牌分裂：大疆 vs DJI

- **中国市场**："大疆"两字具有强品牌认知（"大"=宏大，"疆"=边界/疆域）
- **海外市场**："DJI"—— 缩写，消费者多不知道全称（大疆创新 / Da-Jiang Innovations）
- 两套认知系统并行，但海外消费者对"DJI"的品牌理解主要来自产品体验，而非品牌叙事
- **结论**：身份系统依赖产品本身承载所有品牌权重，是高风险的"产品即品牌"策略

## 内容身份："Shot on DJI"

- 这是 DJI 最成功的身份系统外化：用用户创作内容（UGC）定义品牌美学
- 数十亿条使用 DJI 拍摄的视频形成了"DJI 调色 / DJI 质感"的群体认知
- 无需 DJI 主动输出视觉叙事，用户帮助建立了品牌的视觉语言
- 风险：一旦平台（YouTube/TikTok）或用户偏好转移，品牌美学随之漂移

## Brand Signal Score

**6 / 10** — 产品设计系统强，但品牌叙事层薄。"大疆 vs DJI"双轨并行无统一叙事。命名体系缺乏情感统一性。完全依赖产品和 UGC 支撑身份，是"极致产品力 + 薄品牌层"的典型出海路径。
