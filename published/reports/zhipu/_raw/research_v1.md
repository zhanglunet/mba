## 2. 核心事实（全部可溯源至 §1）

### 2.1 起源（Origin）
- 智谱成立于 **2019 年 6 月**，由**清华大学计算机系知识工程实验室（KEG）技术成果转化**而来；核心研发团队由**唐杰教授**（清华计算机系教授、智谱创立发起人兼首席科学家）领衔（S2/S5/S11）。CEO 张鹏毕业于清华计算机系（创新领军工程博士）（S11）；董事长刘德兵（S8）。
- 愿景自第一天即"**让机器像人一样思考**"（刘德兵上市致辞，S5/S8 逐字）；"智谱的Z是字母表中的最后一个，代表终极境地"（S8 逐字）。
- 早期资本：2021-04 天使轮，**智源研究院以价值 2037 万元的知识产权注资**（全资子公司北京创新智源）；同月 A 轮达晨财智/华控基金/将门创投等 1.52 亿元，投后估值 8.5 亿元（S6）。
- 技术线：2021 提出 GLM 算法架构并发布 GLM-10B；2022 训练出**国内第一个千亿开源模型 GLM-130B**（斯坦福大模型中心 2022-11 全球 30 个主流模型评测中亚洲唯一入选，S9/S11）；2023 推出 ChatGLM，开源单卡版 **ChatGLM-6B**（至 2023-05 全球下载超 160 万、连续两周 Hugging Face 全球趋势榜第一，S9）。

### 2.2 GLM 版本线与开源生态（Category / Leverage）
- **GLM-4 时代**：2024-01 GLM-4；2024-06 开源 GLM-4-9B；2024-10 GLM-4-Voice 与 Agent 产品 AutoGLM；2025-03 AutoGLM 沉思（DeepResearch + Operator"边想边干"）；2025-04 开源 32B/9B 系列（MIT）并启用全新域名 **z.ai**（S10）。
- **GLM-4.5**（2025-07-28）："专为智能体应用打造的基础模型"，推理/代码/智能体综合能力**开源 SOTA**；12 个代表性评测基准综合平均分**全球第三、国产第一、开源第一**；MoE 355B（激活 32B）+ Air 106B（激活 12B）全面开源；API 上线 BigModel.cn，"可以一键兼容 Claude Code 框架"；API 低至输入 0.8 元/百万 tokens（S17）。参数效率：GLM-4.5 参数量为 DeepSeek-R1 的 1/2、Kimi-K2 的 1/3（S17）。
- **GLM-4.6 / GLM-4.7**（2025 下半年）：招股书口径"GLM-4.5/GLM-4.6/GLM-4.7 模型在 2025 年下半年发布后在海外爆火"，Windsurf、Cerebras、Vercel 等官宣接入（S8）；GLM-4.7 发布后登顶 Hugging Face 全球趋势榜第一，AA 智能指数全球开源与国产双料榜首，Code Arena 全球开源第一（S5）；刘德兵："GLM-4.7 的发布让我们的模型跻身世界领先"（S5/S8 逐字）。
- **GLM-5**（2026-02 发布；上市日唐杰内部信预告"很快将推出新一代模型 GLM-5"，S5）：744B（激活 40B 量级）；稀疏注意力架构（DSA）；Artificial Analysis Intelligence Index v4.0 得 50 分刷新开源纪录，LMArena 文本/代码开放模型第一，整体表现接近 Claude Opus 4.5；**深度适配华为昇腾、摩尔线程等七大国产平台**（S18，2026-02-22 技术细节公开）。
- **GLM-5.1**：2026-03 发布，2026-05 出高速版（S12 内文"据IT之家此前报道"）。
- **GLM-5.2**（2026-06-13 面向 GLM Coding Plan 全量用户开放；2026-06-17 上线并开源，MIT）：全球百万用户盲测的 Code Arena **全球可用模型第一**；Solid 1M 上下文，多个长程任务基准介于 Claude Opus 4.7 与 4.8 之间、排名最高的开源模型；**Day 0 完成华为昇腾、平头哥、摩尔线程、寒武纪、昆仑芯、沐曦、海光、壁仞等国产算力平台推理适配**（S12/S16）。智谱官方（S12 逐字）："在一些前沿模型突然变得不可用的时刻，智谱选择相信另一条路：前沿智能不应只属于少数人，也不应被少数规则随时收回。它应该开放、可用、可构建，并服务于每一位开发者。"
- **国际视角**（S13，CNBC 2026-06-26）：GLM-5.2 在一项关键 agentic 基准上与 Anthropic Opus 4.8 差距在 1 个百分点以内、成本约其 1/5；OpenRouter token 流量攀升快于 DeepSeek V4（2026-04）发布后；"intelligence per dollar" 正成为企业最看重的指标；Harvey 联合创始人（CNBC 逐字）："I've been consistently surprised by how quickly the open source has caught up. GLM 5.2, you're seeing the first model where it's really competitive with some of these closed-source frontier models."；语境：Anthropic Fable 因政府命令下线、OpenAI 应政府要求限制 GPT-5.6——"a model that no one can revoke increasingly look like the safer bet"。
- **GLM-5.2 对股价**（S15，SCMP）：2026-06-15（周一）盘中最高涨 48% 至 1620 港元，收涨 32.8% 报 1457 港元；GLM Coding Plan 订阅定价约为 Anthropic Claude Code / Max 高档的 1/10。

### 2.3 商业化与财务（Signal）
- 营收：2022 / 2023 / 2024 = 5740.9 万 / 1.25 亿 / 3.12 亿元（三年 CAGR 130%）；**2025 全年 7.24 亿元（+131.9%）**（S2/S3/S8）。按 2024 收入计，国内独立通用大模型厂商**第一**，整体市场份额 6.6%（S2，招股书口径）。
- 结构：2025 本地化私有化部署收入 5.34 亿元（占 73.7%），客户以央国企、金融、能源为主，**项目续约率 95%**；云端 MaaS 收入 1.90 亿元（**+292.6%**）（S2）。截至 2025-09 末累计服务超 12000 家企业客户、赋能超 8000 万台终端设备、超 4500 万名开发者（S2，公司口径）；MaaS 开放平台 bigmodel.cn 有 270 万+ 企业及应用开发者、中国前十大互联网公司 9 家在用 GLM（S8，招股书/公司口径）。
- **截至 2026Q1，MaaS 平台 ARR 达 17 亿元**（S2）；**GLM Coding Plan 上线两个月全球付费开发者超 15 万，ARR 快速破亿**（S8）。
- 亏损：2022/2023/2024 净亏 1.44 / 7.88 / 29.58 亿元；**2025 全年净亏 47.18 亿元（+59.5%），经调整净亏 31.82 亿元**；研发开支 2024 年 21.95 亿元、2025 年 31.80 亿元（算力采购为核心构成）；2025 综合毛利率 41.0%（S2/S3）。截至 2025-06-30 现金及等价物 25.5 亿元、研发团队 657 人（占比 74%）（S8）；新财富稿另载截至 2025 年 6 月末负债 112.52 亿元、净资产 -61.51 亿元（S4；原文"2055年6月末"按上下文为 2025 年之笔误，仅留痕不入报告正文）。
