## TL;DR

智谱是中国大模型公司里"起点最早、上市最先、开源最激进"的样本：2019-06（ChatGPT 前三年）
由清华 KEG 技术成果转化设立（天使轮为智源研究院以 2037 万元知识产权注资），2022 年做出
国内第一个千亿开源模型 GLM-130B，2025-07 起以 GLM-4.5 → 4.6/4.7 → GLM-5 → 5.2 的节奏
持续开源（MIT），至 2026-06 GLM-5.2 拿下 Code Arena 全球可用模型第一，CNBC 口径"在关键
agentic 基准上与 Opus 4.8 差距 1 个百分点以内、成本约 1/5"。2026-01-08 登陆港交所成
"全球大模型第一股"（发行价 116.2 港元，公开发售 1159.46 倍认购），2026-06-17 科创板 IPO
辅导验收、拟募资 150 亿元回 A。ai-app-cn 面板给出 **177 / 250（7.08 / 10）**。

**共识最强处在 Origin（7.6）**：先于风口的非共识起点 + 可核验的学术血缘，全场无争议。
**最大分歧在 Leverage（Δ4）**：傅盛 9（"MIT 开源 + 国产算力 Day 0 = 没人能下架的基础
设施，这一代最大的分发杠杆"）vs 朱啸虎 5（"开源不是护城河是让利；正面对撞大厂，一年
净亏 47.18 亿是军备竞赛账单"）——这是 2023 年傅朱大模型论战的 2026 续集，攻守易位。

**共识最弱在 Identity（6.2）**："说的是 AGI 登月，赚的是政企私有化（收入占比 73.7%）"
的两张皮，加上智谱/Z.ai/GLM/BigModel/智谱清言的多名并存——五条不同的推理路径抵达同一个
低分。**信号面双刃**：MaaS ARR 17 亿（2026Q1）、私有化续约率 95%、Coding Plan 两个月
15 万付费开发者是同类中最硬的商业化数字；但 2025 年收入 7.24 亿对净亏 47.18 亿、市值
7401 亿港元（2026-06-17，较发行价 +1328.57%）——朱啸虎的反向指标在此报警："大家都不说
泡沫的时候，那肯定是泡沫。"同面板参照：kimichat v1 为 6.85（137/200，2026-06-26）。

## 五镜头评估

### Origin（起源叙事）— 7.6/10 ⭐ 全卷最高

**先于风口的非共识，且留有可核验的学术血缘**。2019-06 由清华大学计算机系知识工程实验室
（KEG）技术成果转化设立，唐杰教授领衔、张鹏任 CEO；天使轮由智源研究院以价值 2037 万元的
知识产权注资——起源不是故事，是可查的股权结构。2021 年提出 GLM 自研架构，2022 年训练出
国内第一个千亿开源模型 GLM-130B（斯坦福大模型中心评测中亚洲唯一入选）。愿景"让机器像人
一样思考"从第一天讲到 2026 年上市致辞。杨植麟△（8）："2019 年做通用大模型是先于风口的
'成立的非共识'。"王慧文（8）：从异常现象出发押对规律的教科书起点。扣分共识：起源资产
的大众转化率低（普通用户听不懂 KEG 是什么），且"起点纯正不等于会活"（傅盛 7）——破局
时刻（2025 年开源爆发）部分借了 DeepSeek 炸开的势。

### Category（品类定义）— 7.4/10

**"开源前沿模型 + 大模型第一股"的复合占位，成立但不独享**。技术品类：GLM-5.2 在百万
用户盲测的 Code Arena 拿下全球可用模型第一，1M 上下文，长程任务表现介于 Claude Opus 4.7
与 4.8 之间（开源最高）；CNBC 给出品类判词——GLM-5.2 与 Opus 4.8 在关键 agentic 基准
差距 1 个百分点以内、成本约 1/5，"intelligence per dollar"正在成为企业最看重的指标。
资本品类：全球首家上市的独立通用大模型公司——"大模型第一股"是智谱自己铸的词。傅盛
（8）："品类的记分牌换成了 Token——智价比。"周鸿祎（8）：三句能被客户复述的心智
（开源最强 / 国产芯片能跑 / 第一个上市）。保留意见构成另一半共识：开源品类里挤着
DeepSeek 与 Qwen，"开源第一"逐季易主（杨植麟△ 7："它证明了自己在曲线上，还没证明自己
定义曲线"）；朱啸虎（7）："基座大模型是共识集中度最高的赛道，跟'错开十五度'正好相反
——但政企私有化与第一股卡位，位置选得不差。"

### Leverage（杠杆点）— 7.0/10 ★ 最大分歧（Δ4）

**同一组事实，两种定价**。事实层：① MIT 开源 + 全球开发者生态（HF 趋势第一、OpenRouter
流量攀升快于 DeepSeek V4 发布后、Windsurf/Cerebras/Vercel 接入）；② 国产算力 Day 0 适配
（GLM-5.2 发布即适配昇腾、平头哥、摩尔线程、寒武纪、昆仑芯、沐曦、海光、壁仞八平台；
40 余款国产芯片）——实体清单（2025-01）把断供风险倒逼成了先发能力；③ 政企私有化 +
95% 续约率的信任存量；④ "A+H"资本平台：已募 43.48 亿港元 + 在途 150 亿元；⑤ 反面：
2025 年研发开支 31.8 亿、净亏 47.18 亿，与腾讯（2024 资本开支 767.6 亿）、阿里（725 亿）
正面对撞。

傅盛（9，全场最高格）："开源 + 国产算力 = 不当任何大平台的附庸——Fable 被下线、GPT-5.6
被限制的那一周，'没人能撤回的模型'成了更安全的选择，智谱股价一天涨 32.8%。这不是运气，
是把独立赛道刻进结构里的回报。"朱啸虎（5，全场最低格）："开源不是护城河，是让利——
大家都开源的时候，比的还是谁烧钱快。它离大厂几条马路？零条。真杠杆只有政企私有化那一个，
撑得起 5 分的现实。"王慧文（7）拆出结构：MaaS 是有万有引力的规模效应业务，私有化交付是
反规模效应业务——公司整体还挂在反规模效应一侧。杨植麟△（7）与周鸿祎（7）落在中间：
三层杠杆皆结构性，但燃料是融资借来的。

### Identity（身份系统）— 6.2/10 ★★ 最弱镜头

**没有分歧的低分**。五条路径抵达同一结论：① 名字系统失控——智谱、智谱华章、Z.ai、GLM、
BigModel、ChatGLM、智谱清言并存（周鸿祎："一个品牌七个名字，客户记住你的成本太高了"）；
② 身份两张皮——对资本市场讲 AGI 终极境地（"智谱的 Z 是字母表中的最后一个，代表终极
境地"——刘德兵上市致辞），对客户做政企项目制交付（73.7% 收入），杨植麟△："登月计划的
组织形态和政企集成的组织形态是两种物种——目前是两个公司共用一个愿景"；③ C 端"智谱
清言"基本隐身。正面资产同样真实：使命句"让机器像人一样思考"七年未变；GLM-5.2 发布时
那句"前沿智能不应只属于少数人，也不应被少数规则随时收回"是中国大模型公司第一次布道级的
身份宣言——唯一的 7 分来自傅盛，正是给这句话的。

### Signal（真实信号）— 7.2/10

**同类中最硬的商业化数字，与最激进的估值定价并存**。正面：2025 年收入 7.24 亿（+131.9%，
三年 13 倍）；MaaS ARR 17 亿（2026Q1）、云端收入 +292.6%；GLM Coding Plan 上线两个月
15 万付费开发者、ARR 破亿；私有化续约率 95%、12000 家企业客户；按 2024 收入计为国内独立
通用大模型厂商第一（份额 6.6%）；港股公开发售 1159.46 倍认购、科创板辅导验收（2026-06-17，
同日上交所表态支持尚未形成收入规模的优质大模型企业上市）。负面：净亏 47.18 亿（+59.5%）
且经调整仍亏 31.82 亿；毛利率 41% 处在价格战的下行压力中；市值 7401 亿港元对 7.24 亿收入
（约千倍市销率）。周鸿祎（8）："8 分给证据链，不是给股价。"朱啸虎（6，最低）："付费
信号 8 分，估值信号 3 分——'大家都不说泡沫的时候，那肯定是泡沫'。"

## 评委 R2 对话：核心分歧点

*（以下评委发言均为 AI 基于其公开发表资料的 in-character 模拟，非本人真实意见；带引号并
注明出处的中文原句均逐字出自各评委已核验语料——周鸿祎无逐字语料，其发言全部为风格化
模拟，不含署名引语。）*

**傅盛 vs 朱啸虎：开源是这一代最大的杠杆，还是最贵的让利？（Leverage Δ4）**

> **傅盛**："不是智谱选择了开源，是断供时代替所有人选好了。我说过'今年是千亿大模型过半
> 凋零的一年'（甲子光年，2024）——凋零的都凋零了，活下来的这家把模型 MIT 一开、国产
> 算力 Day 0 一适配，就成了没人能下架的基础设施。我被 Google 一夜下架过 45 款应用，我
> 太知道'命脉在别人手里'是什么滋味——Fable 说停就停的那一周，全世界的开发者都在替
> 智谱做论证。杠杆我给 9。"
>
> **朱啸虎**："傅总，你讲的是情怀，我算的是账。开源让 DeepSeek 开、让 Qwen 开、让它开，
> 大家都开的时候开源就不稀缺了，比的还是谁烧钱快——它一年净亏 47.18 亿，腾讯一年资本
> 开支 767.6 亿，这仗怎么打？'移动互联网我们说离开大厂一条马路，AI时代可能要离开大厂
> 三条马路'（未竟之约，2025）——它离大厂零条马路，正面对撞。我认它一个杠杆：央国企
> 私有化部署，95% 续约，那是大厂不愿意弯腰干的活。所以是 5 分，不是 2 分。"
>
> **王慧文（折中）**："你们俩说的是同一家公司的两条曲线。'规模效应则是商业世界里的万有
> 引力'（清华产品课，整理稿）——傅盛看的是 MaaS 和开源生态那条有引力的曲线，朱总看的
> 是私有化交付那条反规模效应的曲线加上军备开支。真正的问题不是开源对不对，是哪条曲线
> 先长成收入的大头：现在是 73.7% 对 26.3%，市场却按 26.3% 那一边定价——这个剪刀差，
> 就是这家公司未来三年全部的戏。"

**杨植麟△ vs 朱啸虎：长期主义与算账主义，谁给智谱定价？（Signal / 估值）**

> **杨植麟△（利益披露：月之暗面创始人，智谱直接竞对）**："先说披露：这是我的竞对，我
> 按'利益无关者会怎么打'自检过。我不同意用市销率给基座模型公司定价——'AI不是我在接
> 下来一两年找到什么PMF，而是接下来十到二十年如何改变世界'（潜望，2024）。智谱真正
> 该被审计的是智能增量曲线：Code Arena 百万用户盲测第一、与 Opus 4.8 差 1 个百分点，
> 这些是真进展。我的保留只有一条：让利换生态和烧钱换 scale，这两个循环最终要在同一张
> 资产负债表上闭合——这是它的考题，也是我们所有人的。"
>
> **朱啸虎**："植麟，你这套话我 2023 年就听过，对吧？'到底有没有人在用，到底够不够用，
> 这是核心问题'（新浪《未竟之约》，2025）——我先表扬：智谱是六小虎里这道题答得最好的，
> ARR 17 亿、15 万付费开发者，这些我真金白银地认。但一千倍市销率、IPO 半年涨 13 倍、
> 公开发售 1159 倍认购，'大家都不说泡沫的时候，那肯定是泡沫'（未竟之约，2025）。公司

## 90 天品牌行动建议

**立即执行（0-30 天）**

1. **把"智价比（intelligence per dollar）"抢注成自己的品类语言**（承傅盛/朱啸虎）：
   CNBC 已经替它总结了口径（与 Opus 4.8 差 1 个点、成本 1/5）——下一步是每次发版官方
   发布"每美元智能"对比卡（基准分 / 百万 token 成本 / 国产算力可用性），让市场引用
   智谱的口径而不是媒体的转述；品类定义权从来是喊出来的

2. **在科创板招股书里主动给出"收入结构演进路线图"**（承王慧文/朱啸虎）：披露 MaaS/
   订阅收入占比从 26.3% 走向 50% 的时间表与驱动假设，把估值锚从"AGI 信仰"换成三个
   可验证的数（MaaS ARR 增速、私有化续约率、单 token 毛利）——给市场一套跌得下去也
   涨得回来的定价框架

**中期执行（30-90 天）**

3. **做一次品牌收敛手术**（承周鸿祎/杨植麟△）：收敛到"智谱 / Z.ai"双层主品牌（国内/
   国际），产品线全部归入 GLM 一个字母之下；明示 C 端"智谱清言"的战略取舍——要么给它
   独立叙事，要么公开聚焦 B 端与开发者。一个敢说"不做什么"的品牌，Identity 分自己会涨

4. **公开"开源→付费"转化漏斗**（承杨植麟△/朱啸虎）：以 GLM Coding Plan（两个月 15 万
   付费开发者、ARR 破亿）为模板，按季度披露开源模型用户转 MaaS/私有化客户的转化口径——
   把"开源 SOTA"从信仰叙事变成可计算的获客杠杆，终结"让利无捕获"质疑

5. **把地缘位置转化为主动的信任叙事**（承周鸿祎/傅盛）：实体清单与"没人能撤回的模型"
   已是既成事实——发布面向政企与海外开发者的"自主可控与开放承诺"白皮书（国产算力
   适配清单、开源许可不可撤回承诺、数据边界），让被动的防御回应沉淀为可复述的采购论据
