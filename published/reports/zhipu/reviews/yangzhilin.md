## Origin — 8

先说披露：智谱是我的直接竞对，这一格我按"如果我不是同行，会不会也给 8"来自检——会。
我的标准是"如果所有人都觉得你正常，你的理想是大家都能想到的，它对人类的理想总量没有增量"
（潜望，2024-02）。2019 年 6 月从清华 KEG 出发去做通用大模型，那个时点在中国就是没人觉得
"正常"的选择——ChatGPT 还有三年才出现，天使轮是智源研究院拿 2037 万元的知识产权注资的
（证券时报资本版图稿）。这是学术血缘可核验的 tech vision，不是风口叙事的逆向包装。扣两分：
一是 GLM-130B 之后的路线（对话模型、开源、agentic）里，承袭领域共识的部分与原创非共识的
部分需要分开记账；二是起源的非共识成色要用后续每一代模型是否仍在"提出问题"来续期，而
不能吃 2019 年的老本。

## Category — 7

品类判断我只问一件事：核心价值是不是智能本身。"如果你这个产品最核心价值只有10%-20%来自于
AI，就不成立"（潜望，2024-02）——智谱通过这个测试，它卖的就是智能（模型权重、API、
Coding Plan），没有贴皮。GLM-5.2 在 Code Arena 拿全球可用模型第一、与 Opus 4.8 在关键
agentic 基准差 1 个百分点（CNBC, 2026-06-26），这是品类内的真实位置。但"开源前沿模型"
这个品类不是智谱独享的——DeepSeek、Qwen 都住在里面，头衔逐季易主；而"大模型第一股"是
资本市场的品类，不是技术的品类。用 scale 的语言说：它证明了自己在曲线上，还没证明自己
定义曲线。

## Leverage — 7

"如果你能用scale解决的问题，就不要用新的算法解决"（潜望，2024-02）——智谱的杠杆设计
是我认可的第一性结构：开源做获客漏斗（HF 趋势第一 → OpenRouter 流量 → Windsurf/Cerebras/
Vercel 接入），资本平台做算力弹药（43.48 亿港元已募 + 150 亿在途），国产算力 Day 0 适配做
地缘对冲（八大平台，实体清单倒逼出来的能力）。这三层都是结构性的，不是买量。扣三分的
理由也是结构性的：开源是让利，让利换生态的捕获链条到今天只有 Coding Plan ARR 破亿一个
正样本；一年 31.8 亿研发、47.18 亿亏损的投入结构里，model scaling 与 user scaling 必须
同时成立，任何一边断了，杠杆就变成成本。这也是我自己每天在面对的问题——同行的诚实。

## Identity — 6

这一格我下手最重。"AI不是我在接下来一两年找到什么PMF，而是接下来十到二十年如何改变世界"
（潜望，2024-02）——智谱嘴上讲的是同一种话（"让机器像人一样思考"，从第一天讲到上市
致辞，这我尊重），但收入结构说的是另一种话：73.7% 来自政企私有化部署（21经济网,
2026-06-22）。登月计划的组织形态和政企集成的组织形态是两种物种——前者要人才密度和
top-down 的技术判断，后者要交付团队和续约率。智谱现在两个都要，名字系统也跟着分裂：
智谱、智谱华章、Z.ai、GLM、BigModel、智谱清言。技术信仰、产品形态与组织是否同构，是我
评 Identity 的唯一标准——目前是"两个公司共用一个愿景"。6 分，不是否定，是张力未解。

## Signal — 7

最好的榜就是用户。Code Arena 百万用户盲测第一（IT之家, 2026-06-17）比任何自报 benchmark
都硬；Coding Plan 两个月 15 万付费开发者、ARR 破亿（36氪, 2026-01-08），MaaS ARR 17 亿
（2026Q1），这些是"有人用真金白银投票"的信号。为什么不是 8：智能增量的领先是流动的——
GLM-5.2 与 Opus 4.8 的 1 个百分点差距，既可以读成"追平"，也可以读成"下一代模型发布前
的快照"；而 7401 亿港元市值是市场对"持续领先"的预付款，不是信号本身。我把估值从信号里
剔除，只看技术与用户曲线：真实、上行、但未终局。

## Verdict（in character）

"利益披露在先：这是我的直接竞对。智谱做对的事情是把 2019 年的非共识坚持成了 2026 年的
基础设施；它没有解决的问题和我们是同一个——当智能本身是产品，让利换来的生态和烧钱换来的
scale，如何在十到二十年的时间尺度上闭合成商业。这不是它一家的考题。"

## Critical gap

没人追问"开源 SOTA"的续期成本：每一代保持开源第一的边际算力投入在上升，而开源本身
不产生对应的边际收入——除非开源→付费的转化漏斗被公开证明，否则这个头衔是品牌资产，
同时是财务负债。

## Brand action

把"开源模型用户 → MaaS/私有化付费客户"的转化率做成公开口径（Coding Plan 已经给了一个
样本），让开源从"信仰叙事"变成"可计算的获客杠杆"——这一步走通，Leverage 和 Signal
各值一分。
