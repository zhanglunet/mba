# 雷军 · Scorecard — Anthropic (Claude)
# vc-cn Panel · PANEL-MERGE 2026-07-11

> 免责声明:以下是**以雷军视角、基于公开演讲与访谈推断**的角色化打分,非本人观点。我这次不是以小米创始人身份、而是以顺为资本投资人身份看这家美国 AI 公司;Anthropic 不是小米系、也不是顺为被投,所以不触发利益冲突,可以给中立分。我只依据两份 Lead 简报打分,带 flag 的数字(Sonnet 5「更便宜」、$47B ARR 毛/净、分类器「>99%」「debug 掉 70%」、IPO 日期)一律当**未审计线索**,不当既成事实,更不编数、不假装我私下了解他们内部。

---

## 镜头 1 · Origin 起源真实性 — 8/10

这件事我认。从 OpenAI 出走这条起源,有清楚的人、清楚的时间线、公开记录的价值观分歧——不是事后包装的苦难叙事。更打动我的是那个「研究之后的勇气」:2021 年被人当面质疑「safety 和 capability 不能兼得」,他们没有先讲大故事,而是把整个赌注绑在一条当时被嘲笑、成本很高的路上。Dario 管技术、Daniela 管政策,这个搭档是**刻意设计**的,不是碰巧凑的班子。苦难本身我从不加分,苦难之后的**决策转折**才加分,他们有这个转折。我扣两分,是因为这条起源到今天还有点「我比 OpenAI 更安全 / 更好」的影子——真正的原创者,不需要一直拿对手当参照系来定义自己。

**你真会说的话**:「他们不是讲了个好故事才让人信,是当年真的押上了声誉去走一条别人不看好的路——这个我尊重。」

## 镜头 2 · Category 品类定义 — 7/10

「AI Safety-First」这个品类,在受监管的行业(医疗、金融、法律、联邦政府)已经变成采购标准,Constitutional AI 和 RSP 是能被评估的具体凭据,这一点很扎实。而且他们把安全从一句口号,做成了**自己起草的行业标准**(绑定框架、越狱严重度标准),这是真本事。但我看品类,第一条永远是:**品类词是不是让用户一听就秒懂**。在开发者社区,不少人还是把 Claude 当「更安全的 GPT 替代」,不是一个独立范式;「AI Safety」更像是 B2B / 监管语言,不是大众的生活语言。真正让我犹豫的是**安全税**:分类器认词不认意图,连「Hello」、SSH、iptables 都误伤,还有单一来源说 debug 掉了七成体验——这是把品类招牌做成了**用户用起来的摩擦**。守正(把安全基本功做到位)我完全同意,但如果守正守到把旗舰「关进笼子」,那品类的确定性就打了折。Claude Code 这个子品类反而更清楚,开发者一听就懂。综合给 7。

**你真会说的话**:「品类词做到监管认、采购认,这一步很扎实;但让用户用起来觉得被安全策略绊了一下,那这个『安全』就还没完全变成用户的确定性。」

## 镜头 3 · Leverage 杠杆点 — 8/10

我最欣赏的一点:他们的杠杆**不是靠创始人流量**——Dario、Daniela 在大众媒体上声量并不大,可公司照样跑起来了,说明杠杆是结构性的,来自产品、渠道、生态的组合。三朵云都上、零采购摩擦;Claude Code 渗进 IDE;再加一个自托管网关,把切换成本做成了「要重构安全策略、不是换个 API key」——这正是我做生态链最看重的东西:**生态要真的减少用户摩擦、顺手抬高转换成本**,而不只是给公司多一个交叉销售的机会,他们这一层是真的。加上自己起草标准、拿到政府快速通道,这种关系型杠杆钱买不到。我扣两分,是两个依赖:一是压在别人云平台和 $100–200B 级别算力承诺上;二是这次安全通关很大程度靠华盛顿的善意,这是**单点故障**。还有 Sonnet 5 那一刀降价——它同时暴露了底层模型正在被商品化,杠杆越往「平台层」搬越好,越留在「模型层」越危险。

**你真会说的话**:「他们的护城河已经从『我训练一个更安全的模型』搬到了『我写规则、我占企业和政府的渠道』——前者在蒸发,后者更耐打,但别忘了这条命脉一半攥在监管手里。」

## 镜头 4 · Identity 身份系统 — 6/10

这一镜我给得保守,先说清楚我的偏好:我不做 Jobs 式的审美裁决,不会因为 Haiku / Sonnet / Opus / Fable 这套文学命名「高级」就多给分——命名好看不等于身份自洽。让我愿意给分的,是那个**清楚的产品档位阶梯**(Fable 5 溢价 / Opus 4.8 精度 / Sonnet 5 便宜跑 agent):一听就知道贵在哪、便宜在哪,像一个想清楚的产品线,这是加分。但扣分的地方更实:一是 **Anthropic / Claude 双品牌分裂**——大众记得 Claude,企业记得 Anthropic,两套语言没有互相加强,IPO 迟早逼他们做一次品牌架构决断。二是更根本的**身份自相矛盾**:一边是「负责任、要减速的安全实验室」,一边是跑着高速商业化的最强产品公司;前面说的安全税更是把「最能干」这个身份自己戳了一下——旗舰被关进笼子的时候,「最有能力」和「最安全」这两句承诺就在打架。价格、产品、服务、语言要讲**同一个承诺**,他们现在讲了两个。所以 6。

**你真会说的话**:「档位阶梯我喜欢,一看就懂;但『我是安全实验室』和『我是最强产品公司』这两句话,用户到底该信哪句,他们自己还没完全对齐。」

## 镜头 5 · Signal 真实信号 — 8/10

真实信号是我最认死理的一镜:用户有没有拿**钱、时间、复购、推荐**投票,而不是社媒热度。这一块他们的底子是硬的——1000+ 个客户年付百万美元(从半年前的 500 翻倍)、编码支出份额 54% 对 21%、企业 LLM 支出约 40% 对 OpenAI 27%、第一次整体超过 OpenAI。这种**老客户扩容 + 复购**,是「交付后信任」级别的信号,是真金白银,不是发布会热度,这部分我很服。但我不会像有些人那样给满分,原因就在我的老规矩:**发布会数字必须带来源,没审计过就不能替他们当结论**。$47B 这个头条 ARR 带着毛/净的 flag(可能被砍两到四成)、Sonnet 5「更便宜」是存疑的(挂牌价降了,按 token 每任务可能反而贵约 15%)、$965B 被简报点明是**私募 Series H 后估值、不是 IPO 定价**、IPO 日期还自相矛盾。头条数字对我来说,在审计成净口径之前,仍带着「发布会数字」的性质。所以我把最干净的那部分(企业扩容)当强信号,把还没对过账的头条数字扣下来,给 8。

**你真会说的话**:「$47B 挂在那儿很响,但没审计成净口径之前,我只当它是发布会数字;真正让我放心的是那 1000 多个年付百万、还在续费扩容的企业——那是用户不后悔的证据。」

## 一句话总评(角色内)

这是一家**基本功扎实、护城河从「更安全的模型」搬到「自己写规则 + 占企业渠道」的公司**——底子我认;但它同时欠着两笔要还的账:头条 ARR 的净口径没对完,「更便宜」和「更安全」这两句承诺还没做成用户能自己核查的东西。是好公司,不是没有交付债务的公司。

## 我这个产品 / 性价比 / 生态视角独有的关键洞察

别的评委在争「安全是不是护城河、模型层是不是被商品化」。我从**性价比与厚道**这条线看,看到的是另一层,而且是更危险的一层:

**Sonnet 5 降价抢 agent 默认位,这个打法我太熟了——它就是性价比抢默认位。但性价比只有在「真降价」时才叫厚道。** 简报把「更便宜」标成存疑:挂牌价降了,可换了 tokenizer 之后,按每任务真实成本可能反而贵约 15%。这在我这儿正好踩中一条扣分红线:**用低价拿注意力,却在实际使用里让用户多付**。而且要命的是——**开发者是全世界最不好骗的一群用户**,他们按 cost-per-task 精算,你挂牌降价、暗地里 token 涨回来,他们一算就知道。你最想抢的默认位人群,恰恰是最会拆穿「假厚道」的人群;口碑一旦在他们那儿裂,裂得最快,而他们正是你的护城河本身。

第二层,**安全税就是口碑税**。守正(把安全基本功做到位)我百分百同意,这是复杂行业该有的敬畏。但如果安全分类器把旗舰「关进笼子」——误伤「Hello」、SSH、iptables,单一来源说 debug 体验掉七成——那用户买你是冲着「最安全的 AI」,买回去却发现工具更难用了,这就是**「用户会不会后悔」**的问题。别人把安全当护城河 / 商品化之争,我把它当**交付后体验**之争:安全是承诺,可承诺不能反过来把产品做差,否则你收的是口碑税。

第三层,生态这块我反而给好评:自托管网关把切换成本做成「要重构安全策略」,这是真·减摩擦 + 抬转换成本,不是把多 SKU 叫生态。这一点他们做对了。

## 若判断成立,Anthropic 接下来该做的一个品牌动作

**对开发者做一次公开的「厚道对账」**:

把「更便宜」和「更安全」这两个正在存疑、正在收口碑税的承诺,**做成用户可核查、可自己调的信任合同**,而不是发布会上的两个形容词。具体一件事——发一版**经第三方审计的「每任务真实成本」对照(不是挂牌价)**,同时给开发者一个**明确的「安全档位」开关**,让合规用户能把误伤自己调下来。前者把「更便宜」从话术变成能对账的数字,后者把「安全税」从藏起来的口碑负债,变成用户自己捏在手里的选择。

厚道从来不是便宜,是用户把钱、时间和信任交给你之后**不后悔**。你现在最快能补回口碑的地方,就是让最会算账的那群人——开发者——算完账之后,依然觉得值。
