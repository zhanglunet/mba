## 五镜头逐条

### 1. Origin 起源真实性 — 7

角色内推理：从 OpenAI 出走、safety 和 capability 到底能不能兼得的分歧——这个故事我信，它有真人、真时间线、真公开记录，Dario+Daniela 技术+政策的搭档也是刻意设计的对称。更难得的是，这个"被嘲笑的理想主义"最后被企业侧数字反过来验证了，不是空叙事。**讲宏大叙事的是我最不喜欢的创业者** `[新]`——safety mission 天然带宏大叙事色彩，我要打个折；但它没停在故事里，落到了钱上，所以我给 7 不给 5。

你真会说的话：起源是真的，故事也讲得好，可它是个使命故事——我更认后面那串收入把这故事顶起来了这件事，故事本身我只信七成。

### 2. Category 品类定义 — 6

角色内推理：Safety-First 能变成医疗/金融/联邦政府的采购门槛，这是真付费场景，不是造词，这一分实打实。但你得看它站在哪儿——前沿大模型是全中国、全世界最共识、最挤、club deal 最密的一条道。**我们现在都是错开共识高度集中赛道，稍微偏开十五度** `[新]`，Anthropic 是往共识正中心扎。共识越集中，最后股份被摊薄、赢了也赚不到超额，这是结构问题。再加上 OpenAI、Google、Meta 现在都在发自己的 safety framework，连它自己都承认那个漏洞在 Opus/GPT/Kimi 上都能复现——品类独占性在稀释。Claude Code 那个 54% 的编码品类是它真正扎实的一块，把分从 5 拉到 6。

你真会说的话：站在最共识的正中心当第一名，听着性感，可那儿最没意思——真扎实的机会都在偏开 15 度、不那么性感的地方，Claude Code 才是它算得清账的那块。

### 3. Leverage 杠杆点 — 6

角色内推理：我看杠杆只看一件事——护城河在不在 AI 之外。**纯工具确实很难防守** `[新]`。它做对两件"AI 之外"的事：一是 Claude Code 那个 self-hosted gateway，第三方说"拔掉它要重构安全策略、不是换个 API key"，这是我认的真迁移成本；二是它自己去牵头写越狱严重度标准、拿政府快速放行——这是关系/监管资本，权重买不来。但问题也硬：它整个是架在 AWS/GCP/Azure 上的 API 组件，**移动互联网我们说离开大厂一条马路，AI 时代可能要离开大厂三条马路** `[新]`，而它就坐在三家大厂的正路上（Google 有 Gemini，Amazon 又投又竞）。监管护城河吊在华盛顿一根绳上，$100–200B 算力承诺又是"控进入成本"的反面。真杠杆，但脆。

你真会说的话：gateway 的迁移成本和它自己写标准这两下，是它离大厂马路最远的地方；可它整个人还坐在大厂的算力和监管善意上，护城河断在华盛顿一根绳——这不是永久壁垒，是抢来的时间窗。

### 4. Identity 身份系统 — 5

角色内推理：我看身份就一条——品牌叙事和商业化路径合不合得上，有没有为估值硬凹故事。Fable/Opus/Sonnet 三级命名是全市场最有纪律的，这个我夸。但两处拧着：第一，Anthropic 和 Claude 双品牌到 IPO 门口了还没收口，大众认 Claude、企业认 Anthropic，两套话互相不加强；第二，也是更要命的——"我是个负责任、该减速的 safety lab"和"我全速把前沿模型商业化跑到 $47B"是两个不相容的命题。**嘴上说一套，但做的是另外一套** `[新]`。这不是骗，是它自己内部还没想清楚要当哪个，估值叙事和经营现实是错位的。5 分，命名加的分被叙事错位吃回去。

你真会说的话：真减速的 safety lab 不会一边喊安全一边把收入 14 个月干到 47 倍——名字取得再好，这两张脸没对齐，我就得当它还没想清楚自己是谁。

### 5. Signal 真实信号 — 7

角色内推理：这是它最硬的一块，也是我唯一愿意真金白银认的地方——企业侧首超 OpenAI（~40% vs 27%）、Claude Code ~$8B run-rate、1000+ 家 $1M 客户。**到底有没有人在用，到底够不够用，这是核心问题** `[新]`，这几个数说明真有人用、真够用。但我不能装看不见那个 flag：$47B 里有一部分是 gross 转售总额，净口径重述最多能砍 20–40%，这正好是我最爱盯的地方——**它收入确实起得很快，但我觉得核心问题还是收入** `[新]`，Cursor 那种负毛利我见过，$100B 算力后面 Anthropic 的毛利简报里根本没给。带 flag 的头条数字按 21× ARR 定价，我不敢当审计过的事实认账。真信号，扣在口径不干净和毛利看不清上，7 分。

你真会说的话：有人用、够用，这两条我服；但你先把 net revenue 和毛利掏出来，别拿一个可能掺了转售总额的数去撑 21 倍 ARR——收入干不干净，是这家唯一还没证明的事。

## 一句话总评（角色内）

真收入是硬的，可这是一家最共识、最烧钱、单位经济看不清、宏大叙事最重的公司——它赢没赢我不怀疑，但站在共识正中心、账还没算干净的地方，超额回报早被摊平了；我十分钟能给它一个"好公司"，但十分钟也能说"不是我这套打法能赚到钱的标的"。

## vc-cn / 理性单位经济视角独有、SV VC 评委看不到的关键洞察

SV 评委把"最大 AI 品类里的第一名"当成最高级的强项来打——赢家通吃嘛。但在我这套错开共识 15 度、反内卷的框架里，**最大共识的正中心恰恰是超额回报被摊得最平、定价泡沫最厚的地方**：谁赢这个品类，股份都被 club deal 摊薄，21× ARR 的公开可比对一个可能掺 gross 的收入极度敏感。gross/net 简报里 SV 评委也 flag 了，但他们 flag 完还是给 Signal 打满分；我这套是反过来——**带 flag 的头条数字直接决定我认不认这个信号**，因为我框架的立身之本就是"有没有人用、够不够用、毛利算不算得清"，$100B 算力后面看不到毛利，我就把 Signal 从 10 压到 7。这一压，就是 vc-cn 和 vc-en 在 panel-merge 上最大的分叉点。

## 若判断成立，Anthropic 接下来该做的一个品牌动作

**少讲宏大叙事，把 Claude Code 拎出来做成一个单位经济透明、能独立算账的应用品牌，同时主动做一次 net-revenue + 毛利口径的披露，一把把 gross/net 的疑云打掉。** 它现在最大的品牌负债不是能力、是"账不干净 + 两张脸"：一边 AGI/safety 的宏大叙事，一边说不清的头条 ARR。**嘴上可能说的是 AGI，但事实上做的都是在应用端** `[新]`——那就大大方方把手上真做的那个应用（Claude Code，$8B、54% 编码份额）当品牌锚，用 net ARR 和毛利把"有人用、够不够用、赚不赚钱"讲成人话。对我这种理性钱，一份干净的净收入表，比十篇《Machines of Loving Grace》管用。
