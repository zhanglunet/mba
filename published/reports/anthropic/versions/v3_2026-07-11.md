## TL;DR — Panel Comparison Summary

- **两面板方向惊人一致,没有一个镜头分歧 ≥1.5。** 逐镜头均值差全部 ≤0.8(Origin −0.2 / Category −0.4 / Leverage −0.8 / Identity −0.8 / Signal 0.0)——Anthropic 的影响力构造**跨两种 VC 文化都读得出同一个形状**,这本身是最强的、与面板无关的信号。
- **vc-cn 系统性偏严 ~2.2 分(35.8 vs 38.0 /50),但只是"程度",不是"方向"。** 偏严集中在 **Leverage(−0.8)** 与 **Identity(−0.8)**:中国 VC 把三云依赖 + 华盛顿善意的单点故障、以及 Anthropic/Claude 双品牌分裂,读得比硅谷 VC 更重。
- **两面板的共识裂缝完全重合:** ① **Signal 全 10 位评委齐打 gross/net 折价**(没人给满分,都因 $47B ARR 的净额未审计);② **Identity 是两面板各自最弱的镜头**(双品牌 + "safety lab vs $47B 增长机器"的使命分裂)。
- **vc-cn 独有、SV VC 结构上产不出的批评:** 朱啸虎的"站在最共识赛道正中心=超额回报被摊平",徐新的"C 端心智空心化",雷军的"安全税=口碑税、开发者最会拆穿假厚道"——一套消费心智 + 单位经济的锐评。

### Where the panels agree(均值差 ≤1.0——全部 5 个镜头)

**这是本次最重要的结论:没有任何一个镜头的两面板均值差达到 1.5,最大差仅 0.8。** 换句话说,一支硅谷 VC 阵容和一支中国 VC 阵容,对同一家美国 AI 公司,给出了几乎同形的打分曲线。稳定的共识锚点有三处:

- **Origin(8.4 / 8.2)**:两面板都认这是真实、可消费十年的创始叙事。张磊"是我愿意'同行'的伟大格局观创业者",沈南鹏"赛手是真的、企图心是真的",与 vc-en 的 Andreessen/Graham 判断同向。
- **Signal(7.8 / 7.8,完全相等)**:**两面板 10 位评委无一给满分,全部因 $47B ARR 的 gross/net 未审计而折价。** 这是全审计里最强的跨面板共识——招牌数字的"干净度"是被两种投资文化共同标红的第一风险。
- **Identity 同为最弱镜头(6.4 / 5.6)**:双品牌分裂 + "safety lab vs 增长机器"的使命错位,是两面板各自打分最低的一维。

### Where the panels diverge(|Δ| ≥ 1.5)

**没有任何单一镜头达到 ≥1.5 的分歧阈值。** 真正的差异不在"方向"而在"程度":vc-cn 在 **Leverage(−0.8)** 与 **Identity(−0.8)** 两处系统性更严,拖出全场 −2.2 的总差。把它当"分歧"来读会失真;更准确的说法是**同一张脸,中国 VC 打了更重的阴影**。两处阴影的来源:

- **Leverage(8.4 → 7.6):** 两面板都认可 Claude Code gateway 的切换成本 + 自撰标准的关系资本;但 vc-cn 把"三云依赖 + 华盛顿善意单点故障"扣得更狠。朱啸虎:*"gateway 的迁移成本和它自己写标准这两下,是它离大厂马路最远的地方;可它整个人还坐在大厂的算力和监管善意上,护城河断在华盛顿一根绳——这不是永久壁垒,是抢来的时间窗。"* 对照 vc-en 的 Naval/Marc 把 gateway 读作"平台高一层的杠杆"给到 9,vc-cn 只给 8/6。
- **Identity(6.4 → 5.6):** vc-cn 的消费品牌视角对双品牌分裂的惩罚更硬。徐新:*"名字很美,可你有两个名字、两副面孔——消费者的心只给一个锚,你劈成两半,哪半都占不住第一。"* 这是"第一品牌=单一心智"铁律,vc-en 的判官不从这个角度扣分。

### Panel lens fingerprints

- **vc-en 指纹(硅谷 VC):** 相对更慷慨于 **Leverage(8.4)** 与 **Category(7.0)**——把分发/锁定读作杠杆、把"自撰监管标准"读作品类动作;整体对护城河的"迁移"更买账。隐含品牌优先级:**结构性护城河 + 时间窗内的分发速度**。
- **vc-cn 指纹(中国 VC):** 系统性偏严 ~0.5,最狠在 **Identity(5.6)** 与 **Leverage(7.6)**——用**消费心智一致性**(徐新)、**反内卷/错开共识**(朱啸虎)、**性价比与口碑**(雷军)、**赛道卡位确定性**(沈南鹏)、**长期护城河的"守正"**(张磊)五把不同的尺,不约而同地在"这护城河/身份到底稳不稳、账干不干净"上多扣。隐含品牌优先级:**单位经济的干净度 + 单一心智的确定性**。

### Cross-panel verdict

**Anthropic 在两种投资文化下都稳:** 影响力构造的形状(Origin 高、Signal 硬但口径存疑、Identity 最弱)跨面板高度一致——这是最难得、最不依赖单一面板的信号。**它的脆弱边缘也被两面板共同指认**:① Signal 的 gross/net 洁净度(10/10 评委齐扣),② Identity 的双品牌 + 使命分裂(两面板最弱镜头)。**唯有 vc-cn 单独浮现的脆弱点**:C 端心智空心 + "站在最共识赛道正中心、超额回报被摊平"——一套硅谷 VC 结构上不会产出的中国-消费/反内卷批评。稳的地方,两面板一起替它背书;虚的地方,换个文化只会看得更清、不会看反。

## Where the vc-cn judges agree and disagree with each other

**共识(vc-cn 内部):** 5 人一致把 **Identity(5-6)** 判为最弱、把 **Signal** 因 gross/net 折到 7-8、认可 **Origin(7-9)** 的真实。

**分歧:** 朱啸虎(**31**)是全场系统性低分离群点——单位经济/反内卷视角把"最共识赛道"本身当负项:*"站在最共识的正中心当第一名,听着性感,可那儿最没意思——真扎实的机会都在偏开 15 度、不那么性感的地方。"* 另一端张磊 / 沈南鹏(各 **38**)从长期主义 / 头部卡位给出最高分:张磊*"稀缺、值得取一瓢深研的标的;唯一系统性保留是护城河正从'持续创新'悄悄换成'制定规则'"*。中间雷军(**37**)独有的口碑视角:*"开发者是全世界最不好骗的一群用户……你挂牌降价、暗地里 token 涨回来,他们一算就知道——安全税就是口碑税。"* 组内极差 7 分(31↔38),核心分歧点=**"最共识赛道里的第一名"到底该加分还是减分**。

## Action recommendations (next 90 days)

**面板通用(两面板都指向):**

1. **抢在路演前做一次 net-revenue + 毛利口径披露**——Signal 折价是 10/10 评委的共同来源,把 $47B 的 gross/net 疑云一次打掉。(vc-en Marc"钱会说话,穿透只会走人";vc-cn 朱啸虎"收入干不干净,是这家唯一还没证明的事"。)
2. **收敛品牌架构**——Identity 是两面板最弱镜头;IPO 前把 Claude 立为唯一面向用户的心智锚,Anthropic 退到背书层(徐新"一个名字、一条心智";vc-en 亦长期悬而未决)。

**面板特定(仅 vc-cn 阵容浮现):**

3. **把安全从"产品税"重构成"信任红利"**(雷军 / 徐新)——安全税伤 C 端口碑;让用户感到"Claude 是唯一敢让它自己跑一整夜的 AI",而非"最强的车被限速"。
4. **把 Claude Code 拎成单位经济可独立算账的应用品牌**(朱啸虎)——少讲宏大叙事,用 net ARR + 毛利把"有人用、够不够用、赚不赚钱"讲成人话,对理性钱比十篇 essay 管用。
