## What changed since v1 (delta banner)

**4 dimensions re-researched · 5 judges re-scored · overall 191 → 190 (−1) — but the composition rotated hard.**

The 14-day window carried a genuinely major delta for Anthropic, and the panel's near-flat total hides the real story: **the score held, but the reasons changed.**

- **Claude Sonnet 5 shipped 2026-06-30** ("a cheaper way to run agents") — the model tier system hardened into a 3-rung ladder (Fable 5 premium / Opus 4.8 accuracy / Sonnet 5 cheap-agentic). Opus 4.8 itself (2026-05-28) *predates* v1 → not a delta.
- **The safety moat got its biggest real-world test — and relocated.** A June-12 US Commerce export-control shutdown of Fable 5 → a new cybersecurity classifier → CAISI (US-gov) validation → controls lifted, Fable 5 redeployed July 1. Anthropic also co-authored a cross-lab jailbreak-severity standard and published binding regulatory frameworks — but the same episode exposed a **safety product-tax** and Anthropic's *own* admission the vulnerability wasn't unique to its model.
- **The IPO became concrete but its framing corrected.** Confidential S-1 filed 2026-06-01; v1's "$965B IPO valuation" was **mis-framed** — that is the private May-28 Series H post-money, not an IPO price.
- **Headline ARR grew a quality flag.** The $47B run-rate is partly **gross reseller passthrough**; Claude Code specifically is now ~$8B run-rate (up from v1's $2.5B).
- **Distribution deepened** via a self-hosted Claude Code gateway (~July 1-2) that makes ripping Claude out "re-architecting security policy, not updating an API key."

**Net score rotation:** Leverage **↑0.6** (gateway lock-in), Signal **↓1.0** (unanimous, gross-vs-net ARR), Category **↑0.2** (net — but masking a real directional split: Hoffman/Thiel ↑ on the authored standard vs Paul Graham ↓ on the safety-tax). Origin and Identity flat.

## TL;DR

v1's headline was an unresolved question: is Anthropic a genuine 0→1 category creator, or a superbly-executed 1→n competitor whose "AI Safety" edge evaporates when everyone ships a Responsible-AI framework? **v2's answer: the moat didn't disappear — it relocated.** It moved from "we train a safer model" (which the panel now agrees is evaporating as a *technical* edge — Anthropic itself conceded the jailbreak vulnerability was replicable on rival models, and FLI's Summer-2026 index gives Anthropic only a C+) to "we author the rules everyone certifies against, and we own the workflow nobody wants to re-architect" (binding frameworks + the cross-lab CJS standard + a self-hosted Claude Code gateway).

That relocation split the panel productively. **Hoffman (+1) and Thiel (+1) both moved Category up** — an authored regulatory standard is a category move even a monopoly skeptic will score, because it forces rivals to compete on terms you hold the pen on. **Paul Graham moved Category down (−1)** on the opposite reading: the safety philosophy, via a classifier that reportedly refuses "Hello" and taxes debugging, is now *actively hurting* the make-something-people-want core. Both are right about different layers.

The one thing the panel unanimously agreed on is new: **all five judges docked Signal −1** — not because growth slowed (Anthropic now exceeds OpenAI's run-rate for the first time), but because the $47B headline picked up a **gross-vs-net cleanliness flag** right before a roadshow. As Marc put it: money talks, passthrough walks.

**Core tension for v3:** the moat became more interesting and less technical. It is now durable in the way regulatory positions are durable — which means its two failure modes are a **net-ARR restatement** and a **change in Washington's posture**, not a rival's better model. A narrower, more legible risk surface than v1's "can they stay ahead on capability forever" — but a Washington-dependent one.

**The new fault line — Category (σ 1.10, the panel's sharpest disagreement):**
- **Hoffman 8→9 / Thiel 5→6 (both ↑):** Anthropic didn't just sell into the safety category — it *authored the rulebook* (binding Advanced AI Framework + the cross-lab CJS "CVSS for jailbreaks" standard). Thiel, the monopoly skeptic: *"Owning a category means the market can't leave you; authoring one means the government can un-author it"* — a real last-mover move, worth exactly one point and not the 0→1 re-rating the bulls want.
- **Paul Graham 7→6 (↓):** the same safety philosophy is now *taxing the product people love* — *"a category you own is one people would be upset to lose; 'cheaper agents' is a price, not a thing people love — and a bot that won't say hello is a policy, not a product."*

**The new consensus — Signal (σ 0.40, all five ↓1):** the growth is real and still compounding, but the $47B run-rate's *cleanliness* is now the single biggest tail risk on the revenue line. ## 各维度评估 (v2 delta)

### Origin（起源叙事）— 8.4/10 ↔
**零移动,但获得最强旁证**。14 天不改创始事实。值得标注的是:Anthropic 自认越狱漏洞"非其模型独有"(可在 Opus 4.8/GPT-5.5/Kimi K2.7 复现),这恰好印证了 v1 里 Thiel/Naval 的判断——2021 年的"秘密"始终是**商业洞察**(安全在受监管行业可商业化),而非**技术范式**。Andreessen:*"你回过头去支持那个在市场之前就看对的创始人——这是关于 2021 年的事实,2026 年的任何新闻周期都改写不了。"*

### Category（品类定义）— 7.0/10 ↑0.2(**面板最大分歧 σ1.10,方向对立**)
v2 最有信息量的一维。净上升 0.2,但均值掩盖了一次**方向性撕裂**:
- **Hoffman 8→9、Thiel 5→6(结构派,↑)**:安全从"口号"硬化为**Anthropic 亲自执笔的采购/监管标准**(绑定框架 + CJS 跨厂商标准 + 政府 19 天快速通道)。Hoffman:*"给品类起名是标语;让所有竞争对手都必须对照认证你写的标准,才是你握着笔的那种耐久护城河。"* Thiel 只给 +1,因为"立法出来的品类只在华盛顿善意持续时才耐久"。
- **Paul Graham 7→6(产品爱派,↓)**:同一套安全哲学现在**主动妨碍**"做人们想要的东西"——分类器按词表匹配、据报拒答"Hello"、debugging 掉 70%。而本窗口真正的头条动作 Sonnet 5"更便宜的 agents"是**价格战,不是品类创造**(且"更便宜"存疑:Artificial Analysis 测得按任务贵约 15%)。
- **pmarca / naval 持平**:price war 是"好市场的证据"(Marc)、"执笔标准 vs 商品降价两向抵消"(Naval)。

### Leverage（杠杆点）— 8.4/10 ↑0.6(**本次最大正向移动,3 人↑1**)
v1 第二大分歧,v2 因**自托管 Claude Code 网关**收窄:pmarca 8→9、pthiel 6→7、rhoffman 8→9 全部上调。第三方评价该网关"不是功能,是护城河——拔掉它要重构安全策略而非换 API key",叠加 ~$8B ARR、~54% 编码份额,构成模型层之上一层的真实转换成本。但**分歧未闭合**:
- **Naval 持 9(但转为"contingent 9")**:*"锁定是你从客户转换成本里租来的杠杆;被爱是你自己拥有的杠杆——我当初给 9 是给'被爱'那种。"* 安全产品税若压弯 Claude Code 斜率,这个 9 是第一张倒下的多米诺。
- **Thiel +1 止步于 1**:因为 Sonnet 5 是**防御性降价**——*"资本主义与竞争是反义词;而降价就是竞争在自我招供。"*

### Identity（身份系统）— 6.4/10 ↔
**零移动**。命名阶梯干净延伸为三段式(Fable 5 / Opus 4.8 / Sonnet 5),市场上最有纪律的分层命名——加分;但 Anthropic/Claude 双品牌分裂仍未解(S-1 会逼出决定),且"给旗舰上笼子"是一致性问题。加分与减分相抵。Marc:*"给模型起好名字是餐桌礼仪;身份问题是 Dario 会不会在 S-1 逼他之前,成为这家公司需要的那张脸。"*

### Signal（真实信号）— 7.8/10 ↓1.0(**全员齐降,面板最紧共识 σ0.40**)
v1 面板第二高共识(8.8),v2 因一个共同的新理由集体 −1:
- **增长本身仍真实**:1000+ 客户 $1M+ 合同(2 月时 500)、~40% 企业 LLM 支出、首次超越 OpenAI 运行率。
- **但头部数字长出洁净度问号**:$47B 部分为 **gross 转售穿透**(Sacra),净额重述在熊派读法下可削 20–40%(FutureSearch 反驳为个位数)。此外 v1 引用的"$965B IPO 估值"实为私募 Series H 投后价,非 IPO 定价——本报告予以更正。
- 五位评委异口同声:增长可信,但"~21× ARR 的公开可比对'净额能否过审'极度敏感"(Thiel),"我只给能过审计的 ARR 打分"(PG/Naval)。

## 90 天品牌行动建议 (v2,依 delta 重排)

**立即(0-30 天)—— IPO 洁净度是头号**

1. **抢在路演前解决 gross/net ARR 叙事**(全员 Signal ↓1 的直接来源):主动、清楚地披露 $47B 里 net 口径的真实数字与转售穿透占比。五位评委一致把"数字洁净度"列为当前最大尾部风险——沉默只会让熊派的"20–40% 削减"叙事定价。
2. **给安全产品税止血**(Graham/Naval 的核心新扣分):分类器按词表误杀("Hello"/SSH/iptables)、debugging 掉 70%、"caged flagship"反弹——正在侵蚀让开发者选择 Claude Code 的**产品爱**。最安全的安全层是看不见的;把误报率和 debugging 回归当作 P0 产品缺陷修。

**中期(30-90 天)—— 把迁移后的护城河做实**

3. **把"执笔标准"从 PR 变成可验证的采购资产**(Hoffman/Thiel Category ↑ 的依据):CJS 跨厂商标准 + CAISI 背书是权重买不到的关系资本,但依赖华盛顿善意=单点故障。建议主动多元化(EU GPAI、跨政府 AISI 合作)降低单一政府依赖。
4. **网关锁定 = 最强新杠杆,但要保持"被爱"而非仅"被锁"**(Naval 的 contingent-9 警告):自托管网关是真实 SDLC 转换成本,但 lock-in 会在"更干净的竞品不带安全税发布同等能力"那天复利成怨气。守住 Claude Code 的**felt quality**,别让锁定替代吸引。
5. **S-1 逼出品牌架构决定**(Identity 停滞项):Anthropic/Claude 双品牌"managed ambiguity"到 IPO 就是被迫交账的时刻;同步推 Dario 的公众叙事(v1 遗留项,仍未解)。
