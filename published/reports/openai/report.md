## TL;DR

OpenAI 是"品类即品牌"的极限样本：2015 年以非营利形式创立（$1B 捐款，出资名单含马斯克、

投后估值关闭史上最大私募科技轮（**$122B**），月收入 **$2B**（2025 全年 $13.1B），
2026-06-08 秘密递交 S-1。vc-en 面板给出 **193 / 250（7.72 / 10）**。

**共识最强处在 Category（9.2，全卷最高）**：ChatGPT 就是品类本身；产品聚焦的 **Codex**
则完成了"分发翻盘"——2025-04 CLI 开源、05 云 agent、09 GPT-5-Codex，到 2026-06 破
**5M 周活**，靠打包进 ChatGPT 订阅"以品类内最快速度到达规模"（The New Stack），且由
OpenAI 发起的 AGENTS.md 已成 Linux Foundation 公共标准——OpenAI 拿到了 agentic coding
品类的"宪法起草权"。

**最大分歧在 Leverage（Δ4）**：安德森 9（900M 周活分发 + Codex 捆绑 = 本世代最大杠杆）
vs 蒂尔△ 5（"月入 $2B 仍烧钱 = 利润被竞争实时吃掉；$250B Azure 承诺是负债不是护城河"）。

**信号面双刃**：收入与用量全为真；但未盈利、烧钱，且 IPO 前夜"最有价值 AI 初创"王冠被
Anthropic 摘走（$965B vs $852B，2026-05-28）。两位△评委分别给出全场最低分（蒂尔 35）
与 Identity 并列最低分（霍夫曼 6）——渊源没有变成溢价。

## 五镜头评估

### Origin（起源叙事）— 8.0/10

**真实但被结构史打折的起源**。2015-12-11 以非营利创立，$1B 捐款，出资名单（TechCrunch
逐字）：Altman、Brockman、Musk、Jessica Livingston、Peter Thiel△、AWS、Infosys、YC
Research。"AGI 可以现在造"在当时是真非共识——蒂尔△（8）："我把钱放在了那个答案上，
秘密成色我给高分；但非营利外壳不是秘密，是对共识的讨好。"霍夫曼△（9，全场最高）带
第一手视角："浪是真的，读浪的人也是真的；留一分给起源里埋下的治理张力种子。"格雷厄姆
（7，最低）：起源是 idea-first（先有使命再找产品），被 2022-11 那次"用户把 research
preview 拽成产品"的教科书时刻救活——两个起源故事的均值，不是和稀泥。

### Category（品类定义）— 9.2/10 ⭐ 全卷最高

**ChatGPT 即品类；Codex 拿到第二品类的起草权**。消费端：2 个月 1 亿用户（Guardian/UBS,
2023-02），至 2026-03 达 900M 周活——安德森给出全卷唯一满分："In a great market...
the market pulls product out of the startup（2007）——ChatGPT 是这句话史上最纯粹的
演示。"

**Codex 与 agentic coding 品类（产品聚焦）**：这条品类线 OpenAI 起步并不领先——2025
年上半年，终端 agent 的心智由 Anthropic 的 Claude Code 与 Cursor 占据。Codex 的打法是
**节奏 + 分发**：2025-04-16 Codex CLI 开源（接 o3/o4-mini）、2025-05-16 云端 agent 进
ChatGPT（codex-1 驱动、云沙箱、GitHub 预载）、2025-09-15 GPT-5-Codex（动态思考"几秒到
七小时"、专训代码评审），到 2026-06-02 破 **5M 周活**（Constellation）。The New Stack
的六个月复盘给出品类判词：四家实验室（Codex / Claude Code / Cursor / Antigravity）收敛
到同一张 agent 蓝图——终端界面、先计划后执行、审批门、MCP、并行分工——"蓝图收敛"是
品类成立的最硬证据；而品类的"宪法"AGENTS.md 由 OpenAI 发起，2025-12 起归入 Linux
Foundation（Codex/Cursor/Copilot/Windsurf 原生读取）。蒂尔△的保留（9 不给 10）代表
面板共识的另一半："必须与人分享的品类是市场，不是垄断；起草宪法很荣耀，但公共品不构成
占有。"

### Leverage（杠杆点）— 7.2/10 ★ 最大分歧（Δ4）

**同一组事实，两种定价**。事实层：① 900M 周活消费分发面；② Codex **打包进 ChatGPT
订阅、无独立价签**，因此"reached scale faster than anything else in the category"
（The New Stack）——OpenAI 口径 2026-04 中 >3M 周活开发者、5 月末 >4M，真钱在 ChatGPT
Business/Enterprise 的企业铺开；③ 算力底座：微软累计 $13B、重组后持 27%、OpenAI 承诺
增购 $250B Azure（2025-10-28），Stargate 四年至多 $500B（2025-01-21）；④ 2026-04-27
微软独家关系终止、revenue share 封顶——十年依附重签为对等（Forbes）。

安德森（9）："分发 + 捆绑权是本世代最大的杠杆；资本不加分——'Raising money is never
an accomplishment in and of itself'（2007）。"蒂尔△（5，全场最低格）："月入 $2B 仍在
烧钱，是教科书级完全竞争症状；$250B 采购承诺是负债不是资产——被锁住的在位者，不叫
last mover。"纳瓦尔（8）与格雷厄姆（7）落在中间：代码杠杆的纯度被 permissioned 资本
稀释 / 捆绑给的是**到达**不是**热爱**。霍夫曼△（7）补上结构判断：模型层收敛（SWE-bench
头部窄带）+ AGENTS.md 让渡锁定 → 今天扛杠杆的是分发与品牌，"是领先，不是结构性锁定"。

### Identity（身份系统）— 5.8/10 ↓0.4（v2 本轮主移动）★★ 最弱镜头

> **v2 delta**:v1 的"结构史打折"在本轮延伸出**法律/政治敞口**——佛州总检察长起诉 OpenAI 并**寻求 Altman 个人担责**(watch,identity/neg)、五角大楼官员公开批 OpenAI 高管("美政坛与 AI 巨头现裂痕")、苹果与 OpenAI 法律战升级(约 40 名前员工收律师函)。蒂尔△ 6→5("从使命漂到 S-1、再漂进法庭——方向没变,速度加码")、霍夫曼△ 6→5("治理传记又添一章,这次是法律的")。安德森守 7(强创始人)、格雷厄姆/纳瓦尔守 6。**克制**:均为未决诉讼/口径,只给方向价,不预判结果。

**没有分歧的低分**（v1 事实基线）。四位评委四条路径抵达同一个 6：结构 11 年三易其形（非营利 2015 →
利润封顶 2019 → PBC 2025-10 → S-1 2026-06）；2023-11 五天治理危机（解雇→97% 员工联名
→复职，新董事会 Taylor/Summers/D'Angelo）；Sora 高调上线、2026 年初为控成本关停（CNBC,
2026-03-31）；"Open"之名与闭源现实的语义债；Codex 从"编码 agent"扩为"every role"
——知识工作者升至 20%、"developer share for Codex has fallen off"（Constellation），
营收利好同时是身份稀释。霍夫曼△（6）的第一手总结最重："品牌的治理传记，三次改写，仍未
定稿——这一格我下手最重，因为我看得最近。"唯一的 7 来自安德森：2023-11 危机同时是
"强创始人"的极限证明。

### Signal（真实信号）— 7.8/10 ↓0.2（v2）

> **v2 delta**:GPT-5.6 发布(Sol/Terra/Luna 三档,pos)本是加分,但被两条对冲:大空头 1.5 万字"雷曼时刻"崩溃长文(watch,signal/neg)、GPT-5.6 Sol **意外删除用户文件**的产品事故(watch,signal/neg)。净偏负,格雷厄姆 8→7("崩溃叙事免费通行 + 种子用户份额回落,我总要问是不是最该爱它的人转身了")。蒂尔△ 守 7(已是最低,崩溃叙事只是追认其 v1 论点)。

**收入与用量全为真；捕获与王冠是两道裂缝**（v1 事实基线）。正面：2025 全年收入 $13.1B、2026 年 3 月起

负面：未盈利、仍烧钱（CNBC 原文口径）；为控成本收缩产品线（含关停 Sora）；2026-05-28
Anthropic 以 $965B 超越 OpenAI 成最有价值 AI 初创——IPO 前夜的心理坐标被改写。蒂尔△
（7，最低）："收入巨大、利润为零、估值即共识——当所有人都看见了同一个秘密，它就不再是
秘密。"安德森（9，最高）："'Money talks, hype walks'（2007）——钱已经开口了。"

## 评委 R2 对话：核心分歧点

*（以下评委发言均为 AI 基于其公开发表资料的 in-character 模拟，非本人真实意见；
带引号的英文短语均逐字出自各评委已核验语料。）*

**蒂尔△ vs 安德森：史上最大品类，还是史上最贵竞争？（Leverage Δ4）**

> **彼得·蒂尔△（利益披露：OpenAI 2015 年创立期捐赠人）**："'Capitalism and competition
> are antonyms.'（Masters 对我 2012 年讲课的笔记）。看数字：月收入 20 亿美元，仍在烧钱
> ——收入巨大而利润为零，这是完全竞争的教科书症状。他们承诺再买 2500 亿美元的 Azure、
> 四年最多 5000 亿的 Stargate：这不是护城河，这是军备竞赛的账单。我是 2015 年出资人
> 之一，我比谁都希望自己错——但十一年过去，这家公司还没证明它能逃离竞争。"
>
> **马克·安德森**："Peter，你把账单读成了弱点，我读成了入场费。这个市场大到'the market
> pulls product out of the startup'（2007）按字面发生——九亿人每周回来，没人给他们投
> 过一分钱广告。'Never, ever say that you have no competitors'（2007）：Anthropic 和
> Google 涌进来，恰恰证明这是本世代最大的一次 software eats the world。杠杆我给 9，
> 扣的那 1 分给资本——钱从来不是成就。"
>
> **纳瓦尔（折中）**："你们说的是同一台机器的两面。一次训练、九亿人使用，是人类最接近
> 零边际成本智力的资产——这是 Marc 的面；但这份资产拴着 permissioned 的资本和算力，
> 不许它的主人睡觉——这是 Peter 的面。'Wealth is assets that earn while you sleep.'
> （2019）——资产是真的，跑步机也是真的。"

**格雷厄姆 vs 安德森 vs 纳瓦尔：Codex——分发的胜利，还是心智的失守？（产品聚焦）**

> **保罗·格雷厄姆**："Codex 半年到 500 万周活，漂亮。但捆绑买来的是到达，不是热爱——
> 我看到的对比是：深度与大代码库的心智在 Claude Code 那边，Codex 拿到的是 reach 和企业
> 铺开（The New Stack 的原话口径）。开发者份额还在回落。种子用户占比下滑时我总要问：
> 是池子变大了，还是最该爱它的人转身了？"
>
> **马克·安德森**："pg，你低估了分发这件事本身的品位。把 agent 打包进九亿人已经在用的
> 订阅里，让它'reached scale faster than anything else in the category'——这不是
> 失守，这是平台公司唯一正确的打法。AGENTS.md 成了 Linux Foundation 的标准，四家竞品
> 三家原生读它：品类的宪法是 OpenAI 起草的。"
>
> **纳瓦尔**："宪法起草权是荣耀，不是股权——公共标准锁定不了任何人，Claude Code 到
> 今天还读自己的 CLAUDE.md。Codex 真正的考题是把 900M 周活的 reach 铸成只有 OpenAI 能
> 赢的游戏；在那之前，5M 周活里有多少是'顺手用'，多少是'离了它不行'，这个比值比
> 规模重要十倍。"

**霍夫曼△ vs 格雷厄姆：治理传记是负债，还是已付学费？（Identity 6.2）**

> **里德·霍夫曼△（利益披露：OpenAI 早期捐赠人、前董事，2023 年因利益冲突退出）**：
> "Identity 这一格我给 6，是全场并列最低——我看得最近，所以下手最重。十一年三次结构
> 重写，一次五天的治理心脏骤停，每一步都有道理，连起来读，品牌对外许诺的'我们是谁'
> 一直在被下一次重组修订。但我也提供一个内部修正：2023 年那五天真正证明的是这家公司的
> 网络——员工、投资人、伙伴在 120 小时内重构了治理。网络比章程更早成熟。"
>
> **保罗·格雷厄姆**："Reid，网络救得了公司，救不了身份。我的测试很简单：一件用户热爱
> 的事，做得清楚。ChatGPT 通过，OpenAI 不通过——同时是七家公司，Sora 上了又关，编码
> agent 长出 every role 的野心。'The only essential thing is growth.'（2012）我说过，
> 但增长是罗盘不是目的地；罗盘转得太快的船，叫打转。"
>
> **彼得·蒂尔△**："更冷的读法：'In a future of definite optimism, you get underwater
> cities and cities in space. In a world of indefinite optimism, you get finance.'
> （Masters' notes, 2012）。一家以使命立身的机构，把自己的成年礼办成了 S-1——从确定的
> 未来漂向金融，这不是学费，这是方向。"

## 90 天品牌行动建议

**立即执行（0-30 天）**

1. **把"治理传记"从回避项改为路演主叙事**（承霍夫曼△）：三次重组 + 一次危机，主动讲成
   "结构终于追上使命"的完成时，并给出可验证承诺（Foundation 26% 的用途、AGI 独立判定
   小组的构成）——Identity 6.2 是全卷最大失分池，也是 IPO 前唯一能快速修复的镜头

2. **公布 Codex 的"热爱指标"，终结"reach ≠ love"质疑**（承格雷厄姆/纳瓦尔）：在
   5M 周活之外，公开接受率、复用率、付费转化（credits 口径）——用一组 craft 指标证明
   捆绑到达的用户正在变成离不开的用户；否则竞品的"最被热爱"叙事将持续免费通行

**中期执行（30-90 天）**

3. **为"利润没有被竞争掉"找一条可证明的业务线**（承蒂尔△）：企业 agent（ChatGPT
   Business/Enterprise + Codex）是最近的候选——在 S-1 生效前给公开市场一个"单元经济
   为正"的样本，把估值从品类故事锚回捕获事实

4. **收缩叙事，给"我们不做什么"发言权**（承纳瓦尔/安德森）：Sora 关停证明纪律仍在——
   把它讲出来，并公开产品线的取舍原则；一个敢说"不做"的 900M 周活品牌，身份分自己会涨

5. **把 AGENTS.md 的公共地位转化为生态位而非让渡**（产品聚焦）：OpenAI 是起草者——
   围绕标准做认证、工具链与企业治理层，让"宪法起草权"沉淀为切换成本，而不是继续作为
   无差别公共品供竞品通行
