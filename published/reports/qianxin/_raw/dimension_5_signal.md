# Dimension 5 — Signal / 真实信号
# 奇安信 Brand Audit · Phase 2F
# Audit Date: 2026-06-26

## Core Finding

奇安信的市场信号是真实且可核查的：政府客户覆盖率、营收规模和研究输出均有数据支撑。最大不确定变量：长期盈利能力，以及政策驱动型营收在宏观经济压缩政务支出时的韧性。

## 核心财务信号

| 指标 | 数据 | 说明 |
|------|------|------|
| FY2023 营收 | ~67 亿人民币（约 $9.3B) | 同比增长约 7%（增速放缓）|
| FY2022 营收 | ~62 亿人民币 | |
| 毛利率 | ~50-55%（软件+服务为主） | 安全行业正常水平 |
| 净利润 | 持续亏损（净亏损约 10-15 亿/年） | 高 R&D + 销售费用 |
| 市值（2024） | 约 200-250 亿人民币 | 股价较 2021 高点下跌 50%+ |
| 研发投入 | 营收约 20-25% | 高于行业均值 |

*注：净亏损是中国安全头部公司的普遍状态，深信服等同类公司也面临类似压力*

## 客户覆盖信号

- **政府客户**：80%+ 中央部委（公开年报披露）
- **央企/国企客户**：覆盖主要央企 IT 安全建设
- **总客户数**：约 3 万家企业级客户（据公司披露）
- **重点合同**：多个"国家级网络安全基础设施"建设项目（不可公开披露具体内容）

## 技术/研究信号

- CVE 年度发现量：持续进入全球 Top 10 漏洞发现机构（2023）
- APT 组织追踪：公开追踪 200+ APT 组织，是国内情报最完整厂商之一
- 国际学术发表：安全研究院在 USENIX Security、CCS、NDSS 等顶会有论文
- 安全报告质量：天眼年度报告在行业内具有权威参考价值

## 风险信号

| 风险 | 说明 |
|------|------|
| 政策依赖性 | 80%+ 营收来自政务/央企，受政府采购周期影响大 |
| 持续亏损 | 盈利拐点尚未到来，高 R&D+销售费用压缩净利 |
| 国际化滞后 | 几乎无海外营收，国际扩张路径不清晰 |
| 竞争加剧 | 华为云安全、深信服持续向上侵入政务市场 |

## Brand Signal Score

**7 / 10** — 营收规模和客户覆盖数据扎实可信，是上市公司审计数据；但盈利能力困境和政策依赖集中度，限制了信号的"可持续性"评分。
