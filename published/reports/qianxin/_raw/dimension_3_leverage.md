# Dimension 3 — Leverage / 杠杆点
# 奇安信 Brand Audit · Phase 2F
# Audit Date: 2026-06-26

## Core Finding

奇安信的核心杠杆是政府关系网络 + 合规资质壁垒——这是一种强护城河，但不是技术杠杆，而是监管制度性杠杆。AI+安全大模型是新兴的技术杠杆，处于早期验证阶段。

## 杠杆点清单

### 杠杆 A：政府关系与合规资质（最强护城河）
- 80%+ 中央政府部委是奇安信客户（2023 年报）
- 持有：商密认证、涉密资质、国家密码应用认证、等保三级认证能力
- 参与制定：中国网络安全国家标准（GB/T系列）
- 这些资质是强监管壁垒，10+ 年才能建立，竞争对手无法短期复制

### 杠杆 B：威胁情报体系（天眼）
- 天眼高级威胁检测系统：持续追踪 APT 攻击组织，情报库来自多年政务安全运营积累
- 威胁情报的护城河在于"数据飞轮"：越多政府客户→越多威胁数据→越好的检测能力→吸引更多政府客户
- 公开披露追踪 APT 组织数量超 200 个，是国内最完整的威胁情报库之一

### 杠杆 C：AI 安全大模型（AISOC）（新兴杠杆）
- 2023 年发布安全大模型，将 LLM 能力融入安全运营
- AISOC：自动化告警研判、威胁溯源、处置建议——减少一线分析师 60-70% 的重复工作
- 领先于国内同类厂商（安恒、深信服）18-24 个月
- 风险：CrowdStrike Charlotte AI、Microsoft Copilot for Security 已在全球版本快速迭代

### 杠杆 D：人才密度
- 全职员工超 15,000 人（2024）
- 安全研究院（奇安信研究院）持续输出 CVE、漏洞研究、APT 报告
- 安全行业公认的顶级漏洞研究团队，在 DEF CON / Black Hat 等国际舞台有露出

## 杠杆结构评估

| 杠杆类型 | 强度 | 可复制性 | 国际可移植性 |
|----------|------|----------|--------------|
| 政府关系/合规资质 | ★★★★★ | 极低 | 无 |
| 威胁情报体系 | ★★★★ | 低 | 中等 |
| AI安全大模型 | ★★★ | 中 | 高 |
| 人才密度/研究 | ★★★ | 中等 | 中等 |

## Brand Signal Score

**7 / 10** — 政府关系+合规资质构成的监管壁垒是市场中最强的护城河，但这种杠杆的本质是"制度性杠杆"而非"技术杠杆"或"品牌杠杆"；AI安全大模型是真正的技术杠杆，但国际竞争压力大。
